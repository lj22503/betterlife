# BetterLife Skills 🌟

**帮助普通人更好地处理与金钱的关系**

---

## 📖 简介

BetterLife 是一个完整的财商教育与理财实践 Skill 套件，包含 **17 套主题、79 个技能、99 个模板**，覆盖从财商启蒙到人生意义的全生命周期。

**最后更新：** 2026-03-19

**核心理念：**
- 💰 **金钱是工具，人生是目的**
- 🧠 **思维转变先于技能学习**
- 📊 **规划→保障→投资→跃迁→意义**
- 🎯 **让金钱为你服务，而非你为金钱服务**

## 🎯 按人生阶段的使用指南

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/lj22503/betterlife.git

# 进入目录
cd betterlife
```

### 使用示例

#### 示例 1：财商启蒙

```
@ant 我想转变理财思维，从雇员思维到投资者思维
```

#### 示例 2：财务诊断

```
@ant 帮我诊断财务状况

信息：
- 年龄：35 岁
- 月收入：3 万
- 月支出：1 万
- 资产：150 万
- 负债：20 万
```

#### 示例 3：保险配置

```
@ant 帮我规划家庭保险

信息：
- 丈夫：35 岁，年收入 50 万
- 妻子：33 岁，年收入 20 万
- 孩子：5 岁
- 房贷：800 万
```

#### 示例 4：投资配置

```
@ant 帮我构建投资组合

信息：
- 年龄：35 岁
- 可投资资金：100 万
- 风险承受：中等
- 投资期限：10 年
```

#### 示例 5：财务自由规划

```
@ant 我需要多少钱才能财务自由？

信息：
- 月支出：1 万
- 当前储蓄：50 万
- 年储蓄：20 万
```
