# 每周 150 分钟方案 ⏱️

**基于丹尼尔·利伯曼《锻炼》**

---

## 📋 功能描述

指导用户科学完成每周 150 分钟中等强度锻炼，获得最大健康收益。

**适用场景：**
- 锻炼新手
- 时间管理
- 健康维护
- 慢性病预防

**边界条件：**
- 不替代医疗建议
- 循序渐进
- 个体差异需调整

---

## 🎯 核心功能

### 1. 150 分钟标准

**来源：**
```
• 世界卫生组织推荐
• 美国身体活动指南
• 中国全民健身指南
• 基于大量流行病学研究
```

**定义：**
```
中等强度有氧活动：
• 每周 150-300 分钟
  或
• 每周 75-150 分钟高强度
  或
• 等效组合
```

**额外收益：**
```
• 超过 300 分钟可获得额外健康益处
• 最大受益来自每周 90 分钟
• 锻炼越多，寿命越长
```

### 2. 强度判断

**中等强度：**
```
• 心率提升到最大心率的 50-70%
• 最大心率 ≈ 220 - 年龄
• 能说话但不能唱歌
• 微微出汗
• 呼吸加快但不过度

例子：
✓ 快走（5-6 km/h）
✓ 轻松骑行
✓ 休闲游泳
✓ 园艺劳动
✓ 家务劳动（擦地、拖地）
```

**高强度：**
```
• 心率提升到最大心率的 70-85%
• 不能完整说句子
• 大量出汗
• 呼吸急促

例子：
✓ 跑步
✓ 快速骑行
✓ 快速游泳
✓ 高强度间歇训练（HIIT）
✓ 竞技运动
```

### 3. 150 分钟分配方案

**方案 A：每日型**
```
周一至周五：
• 每天 30 分钟中等强度
• 周末休息或活动

优点：
✓ 形成习惯
✓ 容易坚持
✓ 不影响周末
```

**方案 B：隔日型**
```
周一、三、五、日：
• 每次 35-40 分钟
• 间隔休息日

优点：
✓ 恢复时间充足
✓ 每次时间充裕
```

**方案 C：周末集中型**
```
周六、周日：
• 每天 75 分钟
• 平日活动身体

优点：
✓ 适合平日忙碌者
✓ 可进行长时间活动

注意：
× 不建议全部集中在 1-2 天
× 最好分散在整个星期
```

**方案 D：碎片累积型**
```
每日多次：
• 每次 10-15 分钟
• 累积到 30 分钟/天

例子：
✓ 早中晚各 10 分钟快走
✓ 通勤步行 + 午休散步

优点：
✓ 时间灵活
✓ 容易开始
✓ 一切都会累积
```

### 4. 力量训练配合

**建议：**
```
• 每周至少 2 天
• 涉及所有主要肌肉群
• 中等或更大强度
• 提供额外健康益处
```

**主要肌肉群：**
```
• 腿部（股四头肌、腘绳肌、小腿）
• 臀部
• 背部
• 胸部
• 腹部
• 肩部
• 手臂
```

**简单方案：**
```
基础动作（每个动作 2-3 组，每组 10-15 次）：
1. 深蹲（腿部、臀部）
2. 俯卧撑或跪姿俯卧撑（胸部、手臂）
3. 划船或弹力带划船（背部）
4. 平板支撑（核心）
5. 弓步蹲（腿部）
```

### 5. 活动量追踪

**追踪方法：**
```
✓ 运动手环/手表
✓ 手机计步器
✓ 运动 APP
✓ 纸质记录表
```

**目标转换：**
```
150 分钟中等强度 ≈
• 10,000 步/天 × 7 天
• 或 7,000-8,000 步/天 + 专门运动
• 或 15,000-20,000 步/天 × 3-4 天
```

**每周检查：**
```
• 总活动分钟数
• 强度分布
• 力量训练次数
• 休息日安排
```

### 6. 进阶方案

**超过 150 分钟：**
```
• 200-300 分钟/周：额外健康益处
• 300+ 分钟/周：最大益处
• 适合有锻炼基础者
```

**高强度间歇训练（HIIT）：**
```
• 时间效率高
• 75 分钟高强度 ≈ 150 分钟中等强度
• 适合时间紧张者

示例：
• 30 秒冲刺 + 60 秒放松
• 重复 6-10 次
• 每周 2-3 次
```

## 📚 核心理念

**关键洞察：**
1. 150 分钟是科学推荐的最低标准
2. 最大受益来自每周 90 分钟
3. 分散进行优于集中
4. 力量训练提供额外益处
5. 一切活动都会累积

**健康公式：**
```
健康收益 = 150 分钟有氧 + 2 天力量 + 分散进行
```
