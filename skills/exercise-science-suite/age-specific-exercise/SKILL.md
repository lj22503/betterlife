# 分年龄段锻炼指南 👶👴

**基于丹尼尔·利伯曼《锻炼》+ 美国身体活动指南第二版**

---

## 📋 功能描述

根据不同年龄段提供针对性的锻炼指南，实现全生命周期健康管理。

**适用场景：**
- 儿童青少年锻炼
- 成人健康维护
- 老年人防跌倒
- 孕期产后运动

**边界条件：**
- 不替代医疗建议
- 慢性病需医师指导
- 个体差异需调整

---

## 🎯 核心功能

### 1. 学龄前儿童（3-5 岁）

**发展目标：**
```
• 促进生长发育
• 建立活动习惯
• 发展基本运动技能
```

**活动建议：**
```
• 全天保持活跃
• 多种活动类型的积极游戏
• 由成人护理者鼓励
```

**活动类型：**
```
✓ 跑跳游戏
✓ 攀爬
✓ 投接球
✓ 跳舞
✓ 游泳（有监督）
✓ 三轮车/平衡车

避免：
× 结构化训练
× 长时间静坐
× 屏幕时间过多
```

### 2. 儿童青少年（6-17 岁）

**发展目标：**
```
• 提高认知能力
• 骨骼健康
• 身体健康和心脏健康
• 降低抑郁症风险
```

**活动建议：**
```
每日 60 分钟（1 小时）或更长
中等到剧烈体力活动

每周至少 3 天包括：
• 剧烈强度有氧活动
• 肌肉强化活动
• 骨骼强化活动
```

**活动类型：**
```
有氧（大部分时间）：
✓ 跑步、快走
✓ 骑行
✓ 游泳
✓ 球类运动（足球、篮球、网球）
✓ 舞蹈

肌肉强化（每周 3 天）：
✓ 攀爬
✓ 俯卧撑
✓ 仰卧起坐
✓ 拔河
✓ 体操

骨骼强化（每周 3 天）：
✓ 跑步
✓ 跳跃
✓ 跳绳
✓ 球类运动
✓ 体操
```

### 3. 成人（18-64 岁）

**健康益处：**
```
• 预防 8 种癌症
  （膀胱、乳腺、结肠、子宫内膜、食道、肾、胃、肺）
• 降低全因死亡率
• 降低阿尔茨海默症风险
• 降低心脏病、卒中、高血压、糖尿病风险
• 改善骨骼健康、身体功能
• 提高生活质量
```

**活动建议：**
```
有氧运动：
• 每周 150-300 分钟中等强度
  或
• 每周 75-150 分钟高强度
  或
• 等效组合
• 最好分散在整个星期

力量训练：
• 每周至少 2 天
• 中等或更大强度
• 涉及所有主要肌肉群
• 提供额外健康益处

核心原则：多动少坐
```

**额外益处：**
```
• 超过 300 分钟中等强度可获得额外益处
• 一些体力活动总比没有好
• 减少久坐并进行任何程度的活动都能获益
```

### 4. 老年人（65 岁+）

**健康益处：**
```
• 降低跌倒和摔伤风险
• 保持健康和独立
• 更轻松完成日常任务
• 缓解疼痛和管理健康问题
```

**活动建议：**
```
基础要求（同成人）：
• 每周 150-300 分钟中等强度有氧
• 每周 2 天力量训练

额外要求：
• 包括平衡训练
• 多组分体力活动
  （有氧 + 肌肉强化 + 平衡）
```

**活动类型：**
```
有氧：
✓ 快走
✓ 游泳
✓ 骑行（固定自行车）
✓ 太极拳

力量：
✓ 弹力带训练
✓ 自重训练
✓ 轻重量训练

平衡：
✓ 单脚站立
✓ 脚跟脚尖行走
✓ 太极拳
✓ 瑜伽

多功能活动：
✓ 跳舞
✓ 太极
✓ 园艺
```

**注意事项：**
```
• 根据健康水平确定努力水平
• 患有慢性条件应了解如何影响安全活动
• 不能进行 150 分钟时，根据能力尽可能活动
• 避免不活动
```

### 5. 孕期和产后妇女

**健康益处：**
```
• 降低产后抑郁症风险
• 更多能量
• 更好心情
• 减轻压力
• 可能缩短分娩时间
• 加快恢复速度
```

**活动建议：**
```
• 每周至少 150 分钟中等强度有氧
• 最好分散在整个星期
• 每周 2 天肌肉强化活动

习惯剧烈强度者：
• 可继续原有活动
• 需在医疗保健提供者照顾下
```

**安全活动：**
```
✓ 快走
✓ 游泳
✓ 固定自行车
✓ 孕妇瑜伽
✓ 低冲击有氧操

避免：
× 接触性运动
× 可能摔倒或被撞击腹部的活动
× 怀孕第一孕期后平躺活动（导致血流问题）
× 潜水
× 高海拔运动（未适应者）
```

### 6. 慢性条件和残疾人士

**原则：**
```
• 能够进行时应达到标准建议
• 不能达到时，根据能力进行常规活动
• 避免不活动
• 在医疗保健提供者照顾下
```

**活动建议：**
```
有氧：
• 每周 150-300 分钟中等强度
  或
• 每周 75-150 分钟高强度

力量：
• 每周至少 2 天
• 中等或更大强度
• 涉及所有主要肌肉群
```

**注意事项：**
```
• 咨询医疗保健专业人员
• 了解适合能力和条件的活动类型和数量
• 可能需要特殊设备或辅助
• 可能需要专业指导
```


## 📚 核心理念

**关键洞察：**
1. 各年龄段都有不同的活动需求
2. 儿童青少年每日 60 分钟
3. 成人每周 150 分钟 + 力量训练
4. 老年人需加平衡训练
5. 多动少坐是核心原则

**健康公式：**
```
全生命周期健康 = 适龄活动 + 长期坚持 + 多动少坐
```
