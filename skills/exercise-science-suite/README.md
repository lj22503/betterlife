# 锻炼科学套件 🏃

**基于丹尼尔·利伯曼《锻炼》**

---

## 📖 简介

锻炼科学套件包含 5 个技能，从进化视角理解锻炼，科学制定锻炼计划，获得最大健康收益。

**核心理念：**
- 🏃 人类从未进化出锻炼本能
- ⏱️ 每周 150 分钟是科学标准
- 🏥 不同疾病有不同运动处方
- 👶👴 各年龄段有不同需求
- 💪 让锻炼必要而有趣

---

## 🎯 技能列表

| 技能 | 功能 | 适用场景 |

| [exercise-evolution](exercise-evolution/) | 锻炼进化视角 | 理解锻炼意义、建立动机 |

## 📊 使用流程

```
1. exercise-evolution → 理解锻炼本质
2. weekly-150min → 制定基础计划
3. disease-exercise-prescription → 针对性调整（如有健康问题）
4. age-specific-exercise → 年龄段适配
5. exercise-motivation → 克服障碍、坚持锻炼
```

## 🔑 关键概念

### 锻炼悖论
- 有益健康却畸形扭曲
- 本该免费却高度商品化
- 人类从未进化出锻炼本能

### 150 分钟标准
- 每周 150 分钟中等强度
- 或 75 分钟高强度
- 最大受益来自每周 90 分钟

### 疾病运动处方
- 肥胖：有氧为主，每周 200-300 分钟
- 糖尿病：HIIT 尤其有效
- 心血管：提高心肺功能
- 癌症：高强度增强免疫

### 分年龄段
- 儿童青少年：每日 60 分钟
- 成人：每周 150 分钟 + 力量
- 老年人：加平衡训练
