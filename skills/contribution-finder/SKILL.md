# 他者贡献发现器 🎁

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户发现他者贡献，感受"对别人有用"，获得自我价值感。

**适用场景：**
- 感到没有价值
- 觉得生活没意义
- 想为别人做贡献但不知道如何做

**边界条件：**
- 不追求别人的认可
- 主观贡献感最重要
- 小贡献也是贡献

---

## 🎯 核心功能

### 1. 他者贡献三层次

**层次 1：行为贡献**
```
• 帮助他人
• 提供服务
• 创造价值
```

**层次 2：存在贡献**
```
• 仅仅是存在就有价值
• 被需要、被认可
```

**层次 3：主观贡献感**
```
• "我对别人有用"的感觉
• 不需要客观证明
```

---

## ⚠️ 常见错误

**错误 1：追求可见的贡献**
```
问题：
• 只有大贡献才算贡献
• 需要别人认可

解决：
✓ 主观贡献感最重要
✓ 小贡献也是贡献
✓ 不需要客观证明
```

**错误 2：把贡献当交易**
```
问题：
• "我帮助了别人，别人应该感谢我"
• 期待回报

解决：
✓ 贡献本身就是价值
✓ 不期待回报
✓ 为贡献而贡献
```

## 🧪 使用示例

**输入：**
```
@ant 我觉得自己没用，对别人没贡献

【贡献的三个层次】
1. 行为贡献：帮助他人

3. 主观贡献感："我有用"

【建议行动】
1. 每天做一件小贡献

【关键洞察】
人只有在能够感觉到"我对别人有用"的时候

幸福就是"贡献感"。
```
