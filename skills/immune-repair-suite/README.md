# 免疫功能修复套件 🛡️

**基于《免疫功能 90 天复原方案》**

---

## 📖 简介

免疫功能修复套件包含 5 个技能，帮助用户从饮食、压力、肠道、毒素四大根源修复免疫系统，构筑免疫防线。

**核心理念：**
- 🛡️ 免疫系统健康有三大表现
- 🥗 从四大根源解决问题
- 🦠 70% 免疫系统在肠道
- 🧘 压力管理修复肾上腺
- 🌿 肝脏排毒减少毒素负荷

## 📊 使用流程

```
1. immune-balance → 评估免疫健康状况
2. anti-inflammatory-diet → 开始抗炎饮食
3. stress-adrenal → 管理压力、修复肾上腺
4. gut-immune → 三阶段肠道修复
5. liver-detox → 减少毒素、支持肝脏
```

## 🔑 关键概念

### 免疫系统健康
- 杀伤性 T 细胞与 B 细胞平衡
- 辅助性 T 细胞与调节性 T 细胞平衡
- 能区分异物和自身组织

### 四大根源
- 饮食：抗炎食物 vs 增炎食物
- 压力：慢性压力、肾上腺疲劳
- 肠道：生态失调、消化问题、肠漏
- 毒素：环境毒素、肝脏负担

### 90 天复原方案
- 第 1-30 天：以食代疗
- 第 31-60 天：调控压力
- 第 61-90 天：修复肠道
- 持续：肝脏维护
