# 压力与肾上腺修复 🧘

**基于《免疫功能 90 天复原方案》**

---

## 📋 功能描述

指导用户管理压力，修复肾上腺功能，降低皮质醇水平，改善免疫健康。

**适用场景：**
- 慢性压力
- 肾上腺疲劳
- 皮质醇失衡
- 腹部肥胖
- 情绪波动

**边界条件：**
- 不替代医疗建议
- 严重抑郁需就医
- 个体差异需调整

---

## 🎯 核心功能

### 1. 压力反应机制

**两种压力反应：**
```
神经系统反应：
• 交感神经兴奋
• "战斗或逃跑"反应
• 心跳加速、血压升高

激素分泌反应：
• 肾上腺分泌皮质醇
• 肾上腺分泌肾上腺素
• 血糖升高，提供能量
```

**皮质醇的作用：**
```
正面作用（短期）：
✓ 升高血糖，提供能量
✓ 抗炎作用
✓ 抑制过度免疫反应
✓ 为应对潜在疾病做准备

负面作用（长期过高）：
× 食欲增强，胃口变好
× 体内脂肪增加（尤其腹部）
× 肌肉量减少
× 骨密度减小
× 焦虑加重
× 抑郁加重
× 情绪波动大（易怒）
× 性欲减退
× 免疫系统受损
× 记忆力和学习能力下降
× 经前综合征加重
× 月经周期改变
× 更年期症状加重
```

### 2. 肾上腺疲劳评估

**评估维度：**
```
自我评估 1：既往压力源
• 重大创伤经历
• 严重困难
• 是否仍有影响

自我评估 2：当前压力源
• 工作压力
• 人际关系压力
• 身边人生病
• 失去亲人

自我评估 3：自我照顾情况
• 是否好好照顾自己
• 休闲时间
• 社交活动
• 爱好培养

自我评估 4：肾上腺疲劳程度
• 疲劳程度
• 睡眠质量
• 情绪状态
• 体重变化
```

**高皮质醇信号：**
```
✓ 腰围变大（最常见）
✓ 腹部脂肪难消除
✓ 下午疲劳
✓ 渴望咸食
✓ 依赖咖啡因
✓ 睡眠问题
✓ 免疫力下降
```

### 3. 压力管理策略

**每日放松时间：**
```
每天给自己留时间：
✓ 泡澡
✓ 见朋友
✓ 园艺
✓ 散步
✓ 培养新爱好（编织、读书俱乐部）
✓ 放空自己
✓ 身心放松

关键：每天都要做
```

**睡眠优化：**
```
最佳睡眠时间：晚上 10-11 点
• 肾上腺补充能量的关键节点
• 身体最佳休息

睡前 4 小时：
× 不要剧烈运动
• 体温升高、难以入睡

睡前 1 小时：
× 关闭所有电子设备
• 电视、电脑、手机、平板
• 屏幕光刺激大脑
```

### 4. 减压技巧

**深呼吸练习：**
```
腹式呼吸：
• 吸气 4 秒
• 屏息 4 秒
• 呼气 6-8 秒
• 重复 5-10 次

效果：
✓ 激活副交感神经
✓ 降低皮质醇
✓ 促进放松
```

**渐进式肌肉放松：**
```
• 依次紧张 - 放松各肌群
• 从脚到头
• 每晚睡前

效果：
✓ 身体放松
✓ 心理放松
✓ 促进睡眠
```

**正念冥想：**
```
• 每日 10-20 分钟
• 专注呼吸
• 接纳不评判

效果：
✓ 降低压力
✓ 改善情绪
✓ 提升免疫力
```

### 5. 肾上腺支持营养

**关键营养素：**
```
维生素 C：
• 肾上腺合成皮质醇必需
• 来源：柑橘类、青椒、西兰花

维生素 B5（泛酸）：
• 肾上腺功能支持
• 来源：肝脏、蛋黄、全谷物

镁：
• 放松神经肌肉
• 来源：绿叶蔬菜、坚果、种子

锌：
• 免疫调节
• 来源：贝类、红肉、南瓜子

Omega-3 脂肪酸：
• 抗炎
• 来源：鱼类、亚麻籽
```

### 6. 90 天压力修复计划

**第 1-30 天：压力评估与意识**
```
• 完成压力评估
• 识别压力源
• 开始每日放松时间
• 优化睡眠时间
```

**第 31-60 天：减压技巧实践**
```
• 学习深呼吸
• 开始冥想练习
• 渐进式肌肉放松
• 培养放松爱好
```

**第 61-90 天：生活方式整合**
```
• 固化放松习惯
• 建立健康边界
• 学会说不
• 持续自我照顾
```

## 📚 核心理念

**关键洞察：**
1. 皮质醇长期过高损害健康
2. 腹部脂肪是高皮质醇信号
3. 晚上 10-11 点是肾上腺修复关键
4. 每天放松时间必不可少
5. 压力管理是免疫修复核心

**健康公式：**
```
肾上腺健康 = 压力管理 + 充足睡眠 + 营养支持 + 每日放松
```
