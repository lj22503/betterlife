# 抗炎饮食方案 🥗

**基于《免疫功能 90 天复原方案》**

---

## 📋 功能描述

指导用户采用抗炎饮食，戒除增炎食物，增加抗炎食物，修复免疫系统。

**适用场景：**
- 自身免疫性疾病
- 慢性炎症
- 食物敏感
- 免疫失衡

**边界条件：**
- 不替代医疗建议
- 需营养师指导（如有特殊需求）
- 个体差异需调整

---

## 🎯 核心功能

### 1. 食物戒除计划

**戒除食物（3 周）：**
```
麸质：
• 小麦、大麦、黑麦
• 卡姆小麦、斯佩尔特小麦
• 普通燕麦（可能交叉污染）
• 替代品：大米、藜麦、荞麦、小米

动物奶：
• 牛奶、羊奶
• 奶酪、酸奶
• 奶油、黄油
• 替代品：杏仁奶、椰奶、米奶

玉米：
• 玉米及玉米制品
• 玉米油、玉米糖浆
• 替代品：其他谷物

大豆：
• 豆腐、豆浆、豆豉
• 大豆油、大豆蛋白
• 替代品：其他豆类
```

**戒除原因：**
```
• 最常见食物敏感源
• 增加肠道通透性（肠漏）
• 促进炎症反应
• 触发免疫反应
```

### 2. 抗炎食物选择

**优质脂肪：**
```
✓ 鱼类（野生鲑鱼、沙丁鱼）
✓ 坚果（杏仁、核桃）
✓ 种子（葵花子、南瓜子）
✓ 绿叶蔬菜（羽衣甘蓝、瑞士甜菜）
✓ 椰子油（中链甘油三酯）
✓ 橄榄油（特级初榨）
✓ 牛油果

鱼油效果：
• 类风湿性关节炎
• 系统性红斑狼疮
• 抗炎效果显著
```

**抗氧化剂食物：**
```
✓ 浆果类（蓝莓、草莓、树莓）
✓ 深绿蔬菜（菠菜、羽衣甘蓝）
✓ 十字花科（西兰花、花椰菜）
✓ 彩色蔬菜（红、黄、橙色）
✓ 香料（姜黄、生姜、大蒜）
```

**有益于免疫的营养素：**
```
维生素 D：
• 阳光照射
• 鱼类、蛋黄
• 补充剂（如缺乏）

维生素 A：
• 胡萝卜、南瓜
• 深绿蔬菜
• 动物肝脏

锌：
• 贝类、红肉
• 南瓜子、芝麻
• 豆类

硒：
• 巴西坚果（1-2 个/天）
• 鱼类、肉类
• 全谷物

绿茶：
• 抗氧化
• 抗炎
• 免疫调节
```

### 3. 饮食配比

**理想配比：**
```
蔬菜：50%
• 各种颜色
• 生熟搭配
• 有机优先

优质蛋白：25%
• 草饲肉类
• 野生鱼类
• 有机鸡蛋

健康脂肪：25%
• 鱼类
• 坚果种子
• 优质油脂

全谷物（可选）：
• 藜麦
• 糙米
• 荞麦
```

### 4. 血糖控制

**血糖与炎症：**
```
高血糖 → 炎症增加 → 免疫失衡

控制方法：
✓ 每餐包含蛋白质
✓ 每餐包含脂肪
✓ 选择低 GI 碳水
✓ 避免精制糖
✓ 少量多餐
```

**食物选择：**
```
低 GI 碳水：
• 藜麦
• 荞麦
• 糙米
• 红薯

避免：
× 白面包
× 白米饭
× 甜点
× 含糖饮料
```

### 5. 食物敏感测试

**自我测试方法：**
```
步骤 1：戒除某类食物 3 周
• 完全避免
• 仔细阅读标签
• 记录症状

步骤 2：重新引入
• 一次一类食物
• 连续吃 2-3 天
• 观察反应

步骤 3：记录反应
• 消化症状
• 皮肤反应
• 疲劳程度
• 关节疼痛
• 情绪变化

步骤 4：决定去留
• 有不良反应→永久戒除
• 无反应→适量食用
```

### 6. 90 天饮食计划

**第 1-30 天：严格戒除期**
```
• 戒除麸质、动物奶、玉米、大豆
• 100% 抗炎饮食
• 记录症状变化
```

**第 31-60 天：重新引入期**
```
• 逐一重新引入食物
• 每次一类，观察 3 天
• 记录不良反应
```

**第 61-90 天：个性化饮食**
```
• 根据测试结果调整
• 永久戒除敏感食物
• 建立长期饮食习惯
```

## 📚 核心理念

**关键洞察：**
1. 食物可以成为良药
2. 抗炎食物舒缓免疫系统
3. 增炎食物加重免疫反应
4. 食物敏感需个体化测试
5. 3 周戒除 + 重新引入

**健康公式：**
```
抗炎饮食 = 戒除增炎食物 + 增加抗炎食物 + 血糖控制
```
