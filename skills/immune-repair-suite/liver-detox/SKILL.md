# 肝脏排毒方案 🌿

**基于《免疫功能 90 天复原方案》**

---

## 📋 功能描述

指导用户减少毒素暴露，支持肝脏排毒功能，改善免疫健康。

**适用场景：**
- 慢性疲劳
- 脑雾
- 不明原因体重增加
- 化学物质敏感
- 自身免疫性疾病

**边界条件：**
- 不替代医疗建议
- 严重症状需就医
- 循序渐进排毒

---

## 🎯 核心功能

### 1. 毒素来源识别

**环境毒素：**
```
家居毒素：
• 干洗溶剂
• 去污剂
• 油漆
• 树脂胶
• 塑料（双酚 A）

交通毒素：
• 汽油
• 汽车尾气

个人护理：
• 化妆品化学物质
• 洗发水化学物质
• 香水

食物毒素：
• 农药残留
• 重金属（汞、铅、镉、砷）
• 多氯联苯
• 有机磷和有机氯杀虫剂
• 被污染的饮用水

其他：
• 烟草烟雾
• 重金属
• 外源性雌激素
```

### 2. 毒素负荷症状

**常见症状：**
```
✓ 总是感觉疲倦
✓ 思维不清晰（脑雾）
✓ 全身浮肿
✓ 头痛
✓ 肌肉疼痛
✓ 手指或脚发麻
✓ 不明原因的体重增加
```

**症状评估：**
```
• 症状较轻：<35 分
• 症状中等：35-69 分
• 症状较重：70-99 分
• 症状严重：>99 分
```

### 3. 减少毒素暴露

**家居环境：**
```
清洁产品：
✓ 选择天然清洁剂
✓ 白醋、小苏打
✓ 柠檬、精油
× 避免化学清洁剂

空气净化：
✓ 室内植物
✓ 空气净化器
✓ 定期通风
× 避免空气清新剂

水质：
✓ 过滤饮用水
✓ 淋浴过滤器
× 避免直接自来水
```

**个人护理：**
```
化妆品：
✓ 天然有机产品
✓ 简单成分
✓ 无香料
× 避免合成香料

洗发护发：
✓ 无硫酸盐
✓ 无对羟基苯甲酸酯
✓ 天然成分

身体护理：
✓ 天然除臭剂
✓ 有机身体乳
✓ 简单配方
```

**食物选择：**
```
有机优先：
✓ 有机蔬菜水果
✓ 草饲肉类
✓ 野生鱼类
✓ 有机鸡蛋

避免：
× 农药残留食物
× 加工食品
× 塑料包装食品
× 罐头食品（双酚 A）

清洗方法：
✓ 流水冲洗
✓ 小苏打浸泡
✓ 醋水浸泡
✓ 去皮（如适用）
```

### 4. 支持肝脏排毒

**肝脏排毒两阶段：**
```
第一阶段（转化）：
• 细胞色素 P450 酶系统
• 将毒素转化为中间产物
• 需要：维生素 B 族、抗氧化剂

第二阶段（结合）：
• 结合反应
• 将中间产物转化为水溶性
• 需要：氨基酸、硫、甘氨酸
```

**排毒食物：**
```
十字花科蔬菜：
✓ 西兰花
✓ 花椰菜
✓ 卷心菜
✓ 羽衣甘蓝
✓ 含萝卜硫素，支持排毒

绿叶蔬菜：
✓ 菠菜
✓ 瑞士甜菜
✓ 蒲公英叶
✓ 苦味蔬菜

其他排毒食物：
✓ 大蒜、洋葱（含硫）
✓ 柠檬、酸橙
✓ 甜菜根
✓ 胡萝卜
✓ 绿茶
✓ 姜黄
```

**排毒饮品：**
```
水：
• 每天 6-8 杯
• 帮助排出毒素
• 温水最佳

柠檬水：
• 早晨空腹
• 支持肝脏
• 碱性体质

绿茶：
• 抗氧化
• 支持排毒
• 适量饮用

排毒汤：
• 绿蔬排毒汤
• 多种蔬菜
• 营养支持
```

### 5. 肝脏支持营养

**关键营养素：**
```
谷胱甘肽：
• 体内主要抗氧化剂
• 来源：芦笋、牛油果、菠菜

奶蓟草：
• 肝脏保护
• 促进肝细胞再生
• 补充剂形式

N-乙酰半胱氨酸（NAC）：
• 谷胱甘肽前体
• 支持排毒
• 补充剂形式

维生素 B 族：
• 支持第一阶段排毒
• 来源：全谷物、绿叶蔬菜

硫：
• 支持第二阶段排毒
• 来源：大蒜、洋葱、十字花科

氨基酸：
• 结合反应必需
• 来源：优质蛋白质
```

### 6. 90 天肝脏排毒计划

**第 1-30 天：减少暴露**
```
• 识别毒素来源
• 更换清洁产品
• 更换个人护理
• 选择有机食物
• 开始过滤水
```

**第 31-60 天：饮食支持**
```
• 增加排毒食物
• 每天 6-8 杯水
• 支持肝脏营养
• 戒除增炎食物
• 记录症状变化
```

**第 61-90 天：深度排毒**
```
• 考虑排毒补充剂
• 桑拿/出汗疗法
• 干刷皮肤
• 持续健康习惯
• 评估改善情况
```

## 📚 核心理念

**关键洞察：**
1. 毒素来自多种来源
2. 减少暴露是第一步
3. 肝脏需要营养支持
4. 十字花科蔬菜是排毒明星
5. 每天 6-8 杯水帮助排毒

**健康公式：**
```
肝脏健康 = 减少毒素暴露 + 支持排毒功能 + 充足水分
```
