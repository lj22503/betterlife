# 免疫系统平衡评估 🛡️

**基于《免疫功能 90 天复原方案》**

---

## 📋 功能描述

帮助用户评估免疫系统健康状况，筛查自身免疫性疾病风险。

**适用场景：**
- 慢性疲劳
- 不明原因疼痛
- 皮肤问题
- 消化问题
- 自身免疫性疾病家族史

**边界条件：**
- 不替代医疗诊断
- 需医师确认诊断
- 个体差异需考虑

---

## 🎯 核心功能

### 1. 免疫系统健康三大表现

**健康免疫系统：**
```
1. 杀伤性 T 细胞与 B 细胞保持平衡
   • 杀伤性 T 细胞：消灭感染细胞
   • B 细胞：生成抗体

2. 辅助性 T 细胞与调节性 T 细胞保持平衡
   • 辅助性 T 细胞：启动免疫反应
   • 调节性 T 细胞：抑制免疫反应

3. 免疫系统能区分异物和自身组织
   • 识别病毒、细菌等异物
   • 不攻击自身细胞和组织
```

**自身免疫性疾病：**
```
• 免疫系统攻击自身组织
• 多种疾病本质相同，表现不同
• 因年龄、性别、基因等不同而表现不同
```

### 2. 常见自身免疫性疾病

**疾病列表：**
```
• 乳糜泻（麸质敏感）
• Graves 病（甲亢）
• 桥本甲状腺炎（甲减）
• 系统性红斑狼疮
• 多发性硬化症
• 类风湿性关节炎
• 干燥综合征
• 肾小球肾炎
• 1 型糖尿病
• 恶性贫血
• 白癜风
• 银屑病
• 溃疡性结肠炎
• 硬皮病
```

### 3. 自身免疫信号识别

**全身症状：**
```
✓ 疲劳（所有自身免疫性疾病）
✓ 浑身不舒服、不自在或难受
✓ 失眠（Graves 病）
```

**皮肤症状：**
```
✓ 皮肤干燥（桥本甲状腺炎）
✓ 容易淤青（乳糜泻）
✓ 皮肤瘙痒（乳糜泻）
✓ 脸颊或鼻梁蝴蝶斑（系统性红斑狼疮）
✓ 皮肤对阳光敏感（系统性红斑狼疮）
✓ 全身起疹（系统性红斑狼疮）
✓ 天冷时手指变色（雷诺现象）
✓ 皮下结节或肿块（类风湿性关节炎）
✓ 皮肤变厚（硬皮病）
✓ 斑点状色素脱失（白癜风）
```

### 4. 免疫失衡四根源

**四大根源：**
```
1. 饮食
   • 增炎食物过多
   • 抗炎食物不足
   • 食物敏感/过敏

2. 压力
   • 慢性压力
   • 肾上腺疲劳
   • 皮质醇失衡

3. 消化道健康
   • 肠道生态失调
   • 消化问题
   • 肠漏症

4. 毒素暴露
   • 环境毒素
   • 重金属
   • 肝脏负担过重
```

### 5. 免疫健康评估

**评估维度：**
```
饮食评估：
• 抗炎食物摄入
• 增炎食物摄入
• 食物敏感情况

压力评估：
• 既往压力源
• 当前压力源
• 自我照顾情况
• 肾上腺疲劳程度

肠道评估：
• 肠道生态失调
• 消化问题
• 肠漏症风险

毒素评估：
• 毒素负荷
• 症状评估
• 肝脏功能
```

### 6. 90 天复原方案框架

**四阶段修复：**
```
第一阶段（1-30 天）：以食代疗
• 戒除增炎食物
• 增加抗炎食物
• 食物敏感测试

第二阶段（31-60 天）：调控压力
• 压力评估
• 减压技巧
• 肾上腺修复

第三阶段（61-90 天）：修复肠道
• 三阶段修复计划
• 益生菌补充
• 肠漏修复

第四阶段（持续）：肝脏维护
• 减少毒素暴露
• 支持肝脏排毒
• 持续健康饮食
```

## 🧪 使用示例

```
@ant 我经常疲劳，关节痛，帮我评估

信息：
- 症状：慢性疲劳、关节痛、皮肤干燥
- 家族史：母亲有甲状腺问题
- 目前饮食：标准西式饮食
- 压力水平：高
- 消化问题：饭后胀气、便秘
```

**预期输出：**
- 风险评估：中高（自身免疫风险）
- 根源分析：饮食、压力、肠道
- 建议行动：抗炎饮食、压力管理
- 90 天计划框架
- 就医建议：甲状腺功能检查

## 📚 核心理念

**关键洞察：**
1. 多种自身免疫性疾病本质相同
2. 从四大根源解决问题
3. 饮食、压力、肠道、毒素
4. 免疫系统可以修复
5. 90 天复原方案

**健康公式：**
```
免疫健康 = 抗炎饮食 + 压力管理 + 肠道修复 + 肝脏排毒
```
