# 肠道免疫修复 🦠

**基于《免疫功能 90 天复原方案》**

---

## 📋 功能描述

指导用户修复肠道健康，改善肠道菌群，修复肠漏，增强免疫功能。

**适用场景：**
- 消化问题
- 肠道生态失调
- 肠漏症
- 自身免疫性疾病

**边界条件：**
- 不替代医疗诊断
- 严重症状需就医
- 个体差异需调整

---

## 🎯 核心功能

### 1. 肠道与免疫关系

**关键事实：**
```
• 人体 70% 的免疫系统在肠道中
• 肠道健康决定免疫健康
• 功能医学最先关注肠道
• 肠道问题→全身问题
```

**肠道问题的全身影响：**
```
• 自身免疫性疾病
• 慢性炎症
• 营养吸收不良
• 毒素积累
• 情绪问题（肠脑轴）
```

### 2. 肠道健康三问题

**问题 1：肠道生态失调**
```
症状：
• 便秘
• 腹泻
• 排气
• 进食后腹胀
• 腹部绞痛或不适
• 上腹部问题（反流、消化不良）

评估：
• 0-7 分：好
• 8-14 分：需要优化
• 15-20 分：差
```

**问题 2：消化问题**
```
胃酸不足：
• 消化不良
• 营养吸收不良
• 反流、排气、腹胀

胰酶不足：
• 脂肪消化问题
• 蛋白质消化问题

胆汁酸不足：
• 脂肪消化问题
• 脂溶性维生素吸收问题
```

**问题 3：肠漏症**
```
什么是肠漏：
• 肠道通透性增加
• 未消化食物进入血液
• 触发免疫反应
• 导致全身炎症

症状：
• 食物敏感
• 慢性炎症
• 自身免疫问题
• 皮肤问题
• 疲劳
```

### 3. 三阶段修复计划

**第一阶段：消化支持（1-4 周）**
```
目标：改善消化功能

食物支持：
✓ 发酵食物（泡菜、酸菜、酸黄瓜）
  • 促进消化的酶
  • 富含益生菌

✓ 促进胆汁分泌食物
  • 萝卜（马萝卜、红萝卜、白萝卜）
  • 蒲公英
  • 菊苣
  • 苦味绿色蔬菜
  • 洋蓟

✓ 维持胃酸食物
  • 苹果醋（餐前）
  • 柠檬汁
  • 发酵食物

缓和剂食物：
✓ 琼脂、杏仁、大麦
✓ 椰子油、无花果、亚麻籽
✓ 燕麦、秋葵、欧芹
✓ 车前草、石榴籽、李子
✓ 南瓜瓤、米汤
```

**第二阶段：菌群重建（5-8 周）**
```
目标：恢复肠道菌群平衡

益生菌食物：
✓ 发酵蔬菜（泡菜、酸菜）
✓ 发酵乳制品（如无奶敏感）
✓ 康普茶
✓ 味噌、纳豆

益生元食物：
✓ 洋葱、大蒜
✓ 韭菜、芦笋
✓ 菊芋、菊苣
✓ 香蕉（青）

补充剂（可选）：
• 益生菌补充剂
• 益生元补充剂
```

**第三阶段：肠漏修复（9-12 周）**
```
目标：修复肠道屏障

关键营养素：
✓ 谷氨酰胺
  • 肠道细胞主要燃料
  • 来源：骨头汤、肉类

✓ 锌
  • 肠道屏障修复
  • 来源：贝类、红肉、南瓜子

✓ 维生素 D
  • 肠道免疫调节
  • 来源：阳光、鱼类

✓ Omega-3
  • 抗炎
  • 来源：鱼类、亚麻籽

避免食物：
× 白糖、白面粉
× 白面包、曲奇、蛋糕
× 冰激凌、糖果
× 苏打水、薯条
× 加工食品
```

### 4. 肠道友好饮食

**推荐食物：**
```
主食：
✓ 全谷物面包和饼干
  • 每份至少 3g 膳食纤维
✓ 藜麦、糙米、荞麦

蔬菜：
✓ 各种颜色蔬菜
✓ 发酵蔬菜
✓ 苦味蔬菜

蛋白质：
✓ 草饲肉类
✓ 野生鱼类
✓ 有机鸡蛋

脂肪：
✓ 椰子油
✓ 橄榄油
✓ 牛油果

饮品：
✓ 水（每天 6-8 杯）
✓ 草本茶
✓ 骨头汤
```

**避免食物：**
```
× 白糖
× 白面粉
× 白面包
× 曲奇、蛋糕
× 冰激凌、糖果
× 苏打水、薯条
× 椒盐卷饼
× 苏打饼
× 加工食品
```

### 5. 肠道健康评估

**自我评估工具：**
```
肠道生态评估表：
• 0-7 分：好
• 8-14 分：需要优化
• 15-20 分：差

胰酶评估表：
• 0-3 分：不需要补充

胆汁酸评估表：
• 0-1 分：无需补充

胃酸评估表：
• 0-4 分：无需补充

肠漏症评估表：
• 0-1 分：可能未患上肠漏症
```

### 6. 90 天肠道修复计划

**第 1-30 天：戒除与消化支持**
```
• 戒除麸质、动物奶、大豆、玉米
• 食用发酵食物
• 支持消化功能
• 记录症状变化
```

**第 31-60 天：菌群重建**
```
• 增加益生菌食物
• 增加益生元食物
• 考虑补充剂
• 评估改善情况
```

**第 61-90 天：肠漏修复**
```
• 增加修复营养素
• 继续健康饮食
• 重新引入测试
• 建立长期习惯
```
