# 困境探索禅修器 🔍

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户用正念探索困境、痛苦想法和情感，培养接纳和开放的心态。

**适用场景：**
- 面对痛苦想法
- 探索困难情绪
- 正念进阶练习
- 情绪调节

**边界条件：**
- 不试图改变感觉
- 如果压力过大回到呼吸
- 严重心理问题需就医

---

## 🎯 核心功能

### 1. 准备：静坐稳定

**步骤：**
```
1. 静坐几分钟
2. 注意力集中在呼吸
3. 扩大范围到整个身体
4. 进行声音和思想训练
```

### 2. 困境探索三步

**第一步：想法和感受集中**
```
让想法和感受集中在意识的"工作台"上
```

**第二步：身体知觉觉察**
```
将注意力转移到身体
注意与想法和情感相伴的身体知觉
```

**第三步：靠近强烈感觉**
```
1. 转移注意力到感觉最强烈的部位
2. 用呼吸将意识运送到该部位
3. 吸气"吸入"，呼气"呼出"
4. 以友善和好奇的心态探索
```

### 3. 接纳心态

**内心暗示：**
```
• "这种感觉没有什么不好"
• "无论如何，我都可以敞开心胸拥抱它"
• "这就是现状"
• "让我敞开心扉，拥抱它"
• "利用每次呼吸，软化和接纳它们"
```

### 4. 引入现实问题

**步骤：**
```
1. 选择一个不太愉快的事情
2. 让它在意识平台上休息
3. 与困境导致的身体知觉联系
4. 专注身体感觉
5. 接纳、顺其自然
```

---

## ⚠️ 常见错误

**错误 1：试图改变感觉**
```
问题：
• "我要摆脱这个感觉"
• 试图驱除它们

解决：
✓ 不要试图改变它们
✓ 以友善和好奇的心态探索
✓ 接纳它们，让它们自由发展
```

**错误 2：压力过大时硬撑**
```
问题：
• "我必须完成练习"
• 压力过大继续

解决：
✓ 感到压力过大时回到呼吸
✓ 重新关注身体
✓ 以善意对待自己
```
