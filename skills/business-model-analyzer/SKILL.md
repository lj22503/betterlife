# 商业模式分析器 📊

**基于《24 堂财富课》（陈志武）核心理念**

---

## 📋 功能描述

帮助用户分析商业模式的优劣，理解边际成本、规模效应、品牌效应等核心概念。

**适用场景：**
- 创业项目评估
- 商业模式分析
- 投资标的分析
- 创业方向选择

---

## 🎯 核心功能

### 1. 边际成本分析

**商业模式优劣排序：**
- 软件/互联网：边际成本≈0（最优）
- 工业制造：规模效应降低成本
- 农业：边际成本常数（较差）

### 2. 规模效应评估

评估企业是否能通过规模降低成本：
- 大批量采购
- 标准化生产
- 供应链优化

### 3. 品牌效应分析

分析品牌是否能降低营销成本：
- 口碑营销
- 选址即广告
- 全球化人口流动

### 4. 案例库

- 微软：边际成本为零
- 星巴克：不花钱做广告
- 沃尔玛：规模压低成本
- 戴尔：定制 + 直销

## 💡 使用建议

**完整商业模式分析流程：**

1. **边际成本分析** - 确定商业模式类型
2. **规模效应评估** - 分析成本结构
3. **品牌效应分析** - 评估营销优势
4. **改进计划** - 从转换类转向复制类

**关键原则：**
- 边际成本≈0 的商业模式最优（软件/互联网/内容）
- 规模效应可以降低成本
- 品牌效应可以降低营销成本
- 商业理解 3 阶段：小白看规模，高手看现金流，大神看人效

**使用计算器：**
```bash
./calculators/marginal-cost-calculator.sh 100000 10 1000
# 固定成本 10 万，单位变动成本 10 元，产量 1000
```

## 📚 政策依据

**核心理论：**
- 《24 堂财富课》- 陈志武
- 边际成本递减原理
- 规模经济理论

**关键概念：**
- 边际成本：每增加一单位产品的成本
- 规模经济：规模扩大→单位成本下降
- 品牌效应：降低搜寻成本和购买风险
