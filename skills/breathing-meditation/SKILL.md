# 呼吸禅修引导器 🌬️

**基于《5-8 周正念禅修练习》+《十分钟冥想》**

---

## 📋 功能描述

引导用户进行呼吸禅修练习，培养专注力和觉察力，驯服心灵野马。

**适用场景：**
- 提升专注力
- 减压放松
- 情绪调节
- 正念基础练习
- 驯服心灵野马

**边界条件：**
- 不控制呼吸
- 思绪游离是正常的
- 每天至少练习 2 次

---

## 🎯 核心功能

### 1. 准备姿势

**坐姿要求：**
```
• 坐在椅子上，后背挺直但不僵硬
• 脊柱自行支撑，不依靠靠背
• 双脚平放地板，双腿不交叉
• 双手放在大腿上
• 闭眼或视线放低
```

### 2. 呼吸觉察

**觉察要点：**
```
1. 注意力放在腹部
2. 感受气息进出时腹部的起伏
3. 注意充胀感和下降感
4. 注意吸气和呼气之间的轻微停顿
5. 不要控制呼吸，让呼吸自然进行
```

### 3. 驯服野马（来自《十分钟冥想》）

**方法：**
```
1. 把心灵带到自然而舒服的安定之地
2. 将注意力集中在呼吸的起伏上
3. 不要试图阻止或改变想法
4. 当心灵游离时，温和地带回呼吸
5. 像看云卷云舒一样观察想法
```

### 4. 处理思绪游离

**步骤：**
```
1. 发现注意力游离时，恭贺自己
2. 注意注意力转移到了什么地方
3. 以温和的方式将注意力带回腹部
4. 无论游离多少次，都要宽容
```

### 5. 练习时长

**建议：**
```
• 第一周：每天至少 2 次，每次 8 分钟
• 持续练习：可延长至 10-20 分钟
• 3 分钟呼吸空间：随时可进行
• 十分钟冥想：日常练习
```

---

## ⚠️ 常见错误

**错误 1：控制呼吸**
```
问题：
• "我应该这样呼吸"
• 刻意调节呼吸节奏

解决：
✓ 完全没有必要控制呼吸
✓ 让气息自然呼吸
✓ 只是观察，不改变
```

**错误 2：对游离沮丧**
```
问题：
• "我怎么又走神了"
• 责备自己无法专注

解决：
✓ 思绪游离是大脑的正常活动
✓ 每次发现都是警醒的表现
✓ 温和带回，培养耐心
```
