# 理财目标设定器 🎯

**基于《理财就是理生活》核心理念**

---

## 📋 功能描述

帮助用户用 SMART 原则设定清晰的 5 年理财目标，并拆解为可执行的年度/月度计划。

**适用场景：**
- 理财目标设定
- 5 年规划制定
- 目标拆解与追踪
- 财务双轨制管理

---

## 🎯 核心功能

### 1. SMART 目标设定

用 SMART 原则设定理财目标。

**SMART 原则：**
- Specific（具体）
- Measurable（可衡量）
- Achievable（可实现）
- Relevant（相关）
- Time-bound（有时限）

### 2. 财务双轨制

设立共同账户 + 个人账户双轨制。

### 3. 消费警戒线

设定个人/家庭消费警戒线。

### 4. 目标追踪

定期追踪目标进度。
