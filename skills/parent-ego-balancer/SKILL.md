# 父母自我平衡器 ⚖️

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户平衡父母自我状态，减少自我批判，建立健康的内在标准。

**适用场景：**
- 过度自我批判
- 完美主义
- "应该""必须"思维
- 内在批评声音强

---

## 🎯 核心功能

### 1. 父母自我特征

**挑剔型父母自我：**
```
• 挑剔别人的缺点
• 批评、指责
• 说"应该""必须"
• 自我批判
```

**健康父母自我：**
```
• 有合理的是非观
• 关心但不控制
• 引导但不批评
```

---

## ⚠️ 常见错误

**错误 1：过度自我批判**
```
问题：
• "我不够好"
• "我应该做得更好"

解决：
✓ 没有一种批判比自我批判更强烈
✓ 没有一个法官比我们自己更严苛
✓ 学习自我接纳
```

---

## 🧪 使用示例

**输入：**
```
@ant 我总是觉得自己不够好
```

**输出：**
```
⚖️ 父母自我分析

【状态识别】
✓ 你在父母自我状态
  • 自我批判
  • "我不够好"

【建议行动】
1. 觉察内在批评声音
2. 问自己：这是谁的声音？
3. 学习自我接纳

【关键洞察】
没有一种批判比自我批判更强烈，
也没有一个法官比我们自己更严苛。
```

## 📚 核心理念

**关键洞察：**
1. 自我批判最强烈
2. 自己是自己最严苛的法官
3. 学习自我接纳
