# 要钱还是要生活 Skill 套件 🔥

**基于维姬·罗宾《要钱还是要生活》的 FIRE 运动技能**

---

## 📖 简介

这套技能将 FIRE 运动经典《要钱还是要生活》的九步骤计划转化为实用的 AI 工具，帮助用户实现财务独立，提前退休。

**核心理念：**
- ⏰ **生命能量** —— 金钱是用生命能量换来的
- 🎯 **足够线** —— 知道多少钱就足够了
- 🌱 **节俭生活** —— 10 个省钱妙招
- 📈 **交叉点** —— 投资收入超过支出的时刻
- 🚀 **九步法** —— FIRE 运动的完整路径

## 🚀 快速开始

### 使用示例

#### 1. 生命能量计算

```
@ant 帮我计算实际时薪

信息：
- 年薪：30 万
- 工作相关支出：5 万/年
- 工作小时：2000 小时/年
- 通勤等时间：500 小时/年
```

**预期输出：**
- 实际时薪 = (30 万 -5 万)÷(2000+500) = 100 元/小时
- 名义时薪 vs 实际时薪对比

#### 2. 足够线计算

```
@ant 我需要多少钱才能财务自由？
```

#### 3. 节俭生活

```
@ant 如何减少我的支出？
```

#### 4. 交叉点追踪

```
@ant 帮我追踪财务自由进度
```

#### 5. FIRE 九步法

```
@ant 指导我完成 FIRE 九步骤
```

## 📚 九步骤计划

| 步骤 | 内容 | 核心问题 |
|------|------|----------|
| 1 | 坦然接受过去 | 一生赚了多少钱？ |
| 2 | 活在当下 | 实际时薪是多少？ |
| 3 | 月度表 | 钱花到哪里去了？ |
| 4 | 三个问题 | 花得值吗？符合价值观吗？ |
| 5 | 挂图 | 可视化收支趋势 |
| 6 | 减少支出 | 珍惜生命能量 |
| 7 | 增加收入 | 珍惜工作能量 |
| 8 | 交叉点 | 何时投资收入>支出？ |
| 9 | 投资理财 | 以钱养钱 |
