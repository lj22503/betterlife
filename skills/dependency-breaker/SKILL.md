# 依赖关系打破器 🔓

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户打破依赖关系，成为独立自主的人，这是成长的本质。

**适用场景：**
- 依赖型人格
- 需要别人肯定
- 想独立但做不到
- 成长困扰

**边界条件：**
- 成长的本质是打破依赖关系
- 很少有人能完全达成
- 但要努力成为独立自主的人

---

## 🎯 核心功能

### 1. 依赖类型

**依赖父母：**
```
• 需要父母认可
• 按父母期望生活
• 无法做自己
```

**依赖伴侣：**
```
• 需要伴侣肯定
• 害怕被抛弃
• 失去自我
```

**依赖外界：**
```
• 需要别人认可
• 迎合别人期望
• 不敢表达真实想法
```

### 2. 打破步骤

**步骤 1：认识依赖**
```
我依赖什么？谁？
```

**步骤 2：理解来源**
```
这个依赖从哪里来？
```

**步骤 3：逐步打破**
```
从小事开始自己做决定
```

**步骤 4：承担责任**
```
为自己的选择负责
```

---

## ⚠️ 常见错误

**错误 1：依赖一辈子**
```
问题：
• "我就是这样的人"
• 无法独立

解决：
✓ 成长的本质就是要减少并最终打破依赖关系
✓ 很少有人能完全达成
✓ 但要努力成为独立自主的人
```

**错误 2：突然断绝关系**
```
问题：
• "我要和父母断绝关系"
• 极端独立

解决：
✓ 打破依赖不是断绝关系
✓ 是心理上的独立
✓ 逐步减少依赖
```
