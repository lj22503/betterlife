# 声音和思想禅修器 🔊

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行声音和思想禅修，培养对声音和想法的觉察力。

**适用场景：**
- 提升听觉觉察
- 观察想法而不卷入
- 情绪调节
- 正念进阶练习

**边界条件：**
- 先稳定呼吸与身体
- 不试图促使想法来或去
- 让想法自然出现与消失

---

## 🎯 核心功能

### 1. 准备：稳定呼吸与身体

**步骤：**
```
1. 合适坐姿，脊柱自行支撑
2. 双肩放松，头部端正
3. 关注呼吸几分钟
4. 扩大注意力到全身
5. 感觉身体内部的所有知觉
```

### 2. 声音禅修

**练习要点：**
```
1. 注意力从身体转移到听觉
2. 接收来自各个方向的声音
3. 注意声音的出现、持续、消失
4. 将声音当成单纯的声音
5. 注意贴标签的倾向
6. 关注意识游离时温和带回
```

**声音背景（Soundscape）：**
```
• 近处的、远处的
• 前方的、后方的
• 明显的、细微的
• 声音之间的间隔
```

### 3. 思想禅修

**练习要点：**
```
1. 将想法视作意识中的事件
2. 注意想法的出现、逗留、消失
3. 像云彩在意识天空飘过
4. 不试图促使想法来或去
5. 注意被卷入时庆祝发现
6. 重新返回座位等待下一个想法
```

**比喻：**
```
• 云彩在天空飘过
• 电影在银幕上播放
• 你坐着静静观察
```

---

## ⚠️ 常见错误

**错误 1：寻找特定声音**
```
问题：
• "我要听某种声音"
• 刻意寻找

解决：
✓ 不必特意寻找声音
✓ 保持接纳心态
✓ 接收来自各个方向的声音
```

**错误 2：被想法卷入**
```
问题：
• 跟随想法编故事
• 忘记观察

解决：
✓ 将想法视作事件
✓ 注意被卷入时庆祝
✓ 重新返回座位
```
