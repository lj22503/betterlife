# 共情能力构建器 💗

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户建立共情能力，从独立个体走向共生关系，实现"你好我好"。

**适用场景：**
- 想提升情商
- 人际关系困扰
- 想建立深度连接
- 从独立走向共生

---

## 🎯 核心功能

### 1. 共情能力

**共情的作用：**
```
✓ 理解自己内心的情感世界
✓ 理解别人的感受
✓ 从独立个体走向共生关系
✓ 协同合作
```

### 2. 高情商特征

**高情商的人：**
```
• 有强大的自我意识
• 了解自己的情感
• 能管理情绪
• 能从悲伤中重新振作
• 能控制冲动
• 懂得延迟满足
```

---

## ⚠️ 常见错误

**错误 1：共情就是讨好别人**
```
问题：
• "共情就是迎合别人"
• 失去自我

解决：
✓ 共情是理解彼此
✓ 不是讨好
✓ 在自我成长基础上建立
```
