# 自卑感转换器 🔄

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户转化自卑感，从"与他人比较"转向"与理想的自己比较"。

**适用场景：**
- 与他人比较感到自卑
- 无法祝福别人的幸福
- 站在竞争角度看人际关系

**边界条件：**
- 不否定健康的自卑感
- 转化为成长动力
- 不与他人竞争

---

## 🎯 核心功能

### 1. 两种自卑感

**不健康的自卑感：**
```
• 来自与别人的比较
• 把别人的幸福看作"我的失败"
• 无法真心祝福别人
```

**健康的自卑感：**
```
• 来自与"理想的自己"的比较
• 促进自我成长
• 不与他人竞争
```

### 2. 转化方法

```
✓ 从竞争转向合作
✓ 从比较转向成长
✓ 祝福别人的幸福
```

---

## ⚠️ 常见错误

**错误 1：把自卑当动力**
```
问题：
• "我要证明给别人看"
• 用自卑驱动自己

解决：
✓ 健康的自卑感来自与理想的自己比较
✓ 不是为了证明给别人
✓ 是为了自己的成长
```

**错误 2：无法祝福别人**
```
问题：
• 看到别人成功就难受
• 把别人的幸福看作自己的失败

解决：
✓ 从竞争转向合作
✓ 真心祝福别人的幸福
✓ 别人的幸福不是你的失败
```
