# 自卑感转换器 🔄

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户转化自卑感，从"与他人比较"转向"与理想的自己比较"。

**适用场景：**
- 与他人比较感到自卑
- 无法祝福别人的幸福
- 站在竞争角度看人际关系

---

## 🎯 核心功能

### 1. 两种自卑感

**不健康的自卑感：**
```
• 来自与别人的比较
• 把别人的幸福看作"我的失败"
• 无法真心祝福别人
```

**健康的自卑感：**
```
• 来自与"理想的自己"的比较
• 促进自我成长
• 不与他人竞争
```

### 2. 转化方法

```
✓ 从竞争转向合作
✓ 从比较转向成长
✓ 祝福别人的幸福
```

---

```
@ant 看到别人成功我就难受
```

```
【问题诊断】

【转化建议】
1. 从竞争转向合作
2. 与理想的自己比较
3. 真心祝福别人的幸福

【关键洞察】
无法真心祝福过得幸福的他人，
是因为站在竞争的角度来考虑人际关系。
健全的自卑感来自与"理想的自己"的比较。
```

---

## 📚 核心理念

**关键洞察：**
1. 健全的自卑感来自与"理想的自己"的比较
2. 从竞争转向合作
3. 真心祝福别人的幸福

---

*健全的自卑感不是来自与别人的比较，
而是来自与"理想的自己"的比较。* 🔄
