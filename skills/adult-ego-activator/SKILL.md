# 成人自我激活器 🧠

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户激活成人自我状态，理性决策，用知识和技能解决问题。

**适用场景：**
- 想理性决策
- 不想情绪化
- 面临重要选择
- 想学习新东西

---

## 🎯 核心功能

### 1. 成人状态特征

**成人自我状态：**
```
✓ 摆脱情绪化控制
✓ 合理计划、考虑、决定、行动
✓ 用知识和技能解决当下问题
✓ 唯一能学到新知识的状态
```

### 2. 激活方法

**步骤 1：觉察当前状态**
```
我现在在儿童/父母/成人状态？
```

**步骤 2：选择成人状态**
```
我决定进入成人状态
```

**步骤 3：理性分析**
```
用知识和技能解决当下问题
```
