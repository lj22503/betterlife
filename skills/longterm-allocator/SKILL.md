# 长期投资配置器 🚀

**基于《管好四笔钱》（且慢）核心理念**

---

## 📋 功能描述

帮助用户进行长期投资配置（指数基金定投），实现财富长期增值，跨越牛熊周期。

**适用场景：**
- 指数基金筛选
- 定投策略制定
- 资产配置规划
- 3 年以上资金投资

---

## 🎯 核心功能

### 1. 指数基金筛选

**三大维度：**
- 投资地域：A 股/港股/美股等
- 投资风格：超大盘/大盘/中盘/小盘
- 行业指数：朝阳行业（可选）

### 2. 基金定量分析

**指标：**
- 基金规模
- 最大回撤
- 夏普比率
- 波动率

### 3. 估值与定投

**估值指标：**
- 市盈率（P/E）
- 市净率（P/B）
- 百分位：<30% 低估，30-80% 正常，>80% 高估

**定投策略：**
- 低估时多投
- 正常时定投
- 高估时少投或停止

### 4. 风险分散

**三维度分散：**
- 跨市场投资
- 跨品种投资
- 同一市场分散投资
