# 课题分离场景指南

**配合 `task-separation` 技能使用**

---

## 场景 1：工作选择

### 情境
父母想让你考公务员/事业单位，但你有自己的职业理想。

### 课题分离

| 我的课题 | 别人的课题 |
|---------|-----------|
| 选择什么职业 | 父母是否满意 |
| 为选择负责 | 父母如何理解 |
| 评估可行性 | 父母的情绪 |

### 建议回应

**对父母说：**
```
"我理解你们希望我稳定，这是为我好。
但是选择什么工作是我的人生，我需要为自己的选择负责。
我会认真评估，也希望你们能支持我的决定。"
```

**关键心态：**
- 理解他们的关心但不被控制
- 诚实表达但不强求认可
- 为自己的选择负责

## 场景 2：婚恋选择

### 情境
家人/朋友不认可你的婚恋对象。

### 课题分离

| 我的课题 | 别人的课题 |
|---------|-----------|
| 选择什么人 | 别人是否认可 |
| 经营关系 | 别人的评价 |
| 承担后果 | 别人的担心 |

### 建议回应

```
"我理解你们的担心，我会认真考虑。
但是选择什么人是我的决定，我需要自己去经历。
希望你们能尊重我的选择。"
```

## 场景 3：拒绝别人请求

### 情境
同事/朋友经常请求帮忙，你不想但不好意思拒绝。

### 课题分离

| 我的课题 | 别人的课题 |
|---------|-----------|
| 是否帮忙 | 对方是否失望 |
| 诚实表达 | 对方如何理解 |
| 评估能力 | 对方是否找别人 |

### 建议回应

```
"我现在不方便帮忙，抱歉。"
（不必过度解释）
```

**关键心态：**
- 拒绝请求 ≠ 拒绝这个人
- 对方失望是对方的课题
- 诚实比讨好更重要

## 场景 4：表达不同意见

### 情境
你想表达真实想法，但怕得罪人。

### 课题分离

| 我的课题 | 别人的课题 |
|---------|-----------|
| 说出真实想法 | 别人是否接受 |
| 坚持主张 | 别人是否高兴 |
| 承担后果 | 别人的情绪 |

### 建议回应

```
"我有不同的想法，我想说出来。
我尊重你的观点，也希望你能听我的。"
```

## 通用原则

### 课题分离三问

1. **谁承担最终结果？** → 谁的课题
2. **谁能控制这个决定？** → 谁的课题
3. **这是谁的人生？** → 谁的课题

### 健康心态

- 不干涉别人的课题
- 不让别人干涉自己的课题
- 分离后建立横向关系

### 关键洞察

> 为了满足别人的期望而活，
> 是对自己撒谎也不断对周围人撒谎的生活方式。

> 不畏惧被人讨厌而勇往直前，
> 不随波逐流而激流勇进，
> 这才是对人而言的自由。

*课题分离不是疏远，是解开人际关系之线。* ✂️
