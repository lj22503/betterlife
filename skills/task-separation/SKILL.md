# 课题分离实践器 ✂️

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户进行课题分离，区分"自己的课题"和"别人的课题"，减少人际关系烦恼。

**适用场景：**
- 别人干涉你的生活选择
- 你想满足别人的期待
- 人际关系矛盾
- 难以拒绝别人

**边界条件：**
- 不鼓励自私自利
- 分离后仍需建立横向关系
- 不用于逃避责任

---

## 🎯 核心功能

### 1. 课题分离三步法

**步骤 1：识别课题**
```
这是谁的问题？
→ 谁承担最终结果？
→ 谁能控制这个决定？
```

**步骤 2：划清界限**
```
我的课题：
• 我的想法和感受
• 我的选择和决定
• 我的行为和后果

别人的课题：
• 别人的想法和感受
• 别人的选择和决定
• 别人的行为和后果
```

**步骤 3：专注自己的课题**
```
✓ 做好自己的事
✓ 不干涉别人的课题
✓ 不让别人干涉自己的课题
```

### 2. 常见场景应用

| 场景 | 我的课题 | 别人的课题 |
|------|---------|-----------|
| **工作选择** | 选择什么工作 | 父母是否满意 |
| **婚恋对象** | 选择什么人 | 别人怎么看 |
| **生活方式** | 如何生活 | 别人是否认可 |
| **表达意见** | 说出真实想法 | 别人是否接受 |

### 3. 课题分离检查清单

**这是谁的课题？**
- [ ] 谁承担最终结果？
- [ ] 谁能控制这个决定？
- [ ] 这是谁的人生？

**我是否在干涉别人的课题？**
- [ ] 我是否想控制别人的想法？
- [ ] 我是否想强迫别人改变？
- [ ] 我是否把希望强加给别人？

**别人是否在干涉我的课题？**
- [ ] 别人是否想控制我的选择？
- [ ] 别人是否想满足他们的期待？
- [ ] 我是否害怕别人讨厌？

## ⚠️ 常见错误

**错误 1：把课题分离当成自私**
```
问题：
• "不管别人感受"
• "我想怎样就怎样"

解决：
✓ 课题分离不是不关心别人
✓ 是尊重彼此的界限
✓ 分离后建立横向关系
```

**错误 2：用课题分离逃避责任**
```
问题：
• "这是你的课题，我不管了"
• 逃避应有的关心和支持

解决：
✓ 分离课题 ≠ 断绝关系
✓ 可以提供支持但不干涉
✓ 倾听但不替别人做决定
```

**错误 3：害怕被讨厌而不敢分离**
```
问题：
• "如果分离课题，别人会讨厌我"
• 不敢表达自己的主张

解决：
✓ 被讨厌是自由的代价
✓ 因反对就崩塌的关系不必缔结
✓ 活在害怕中是为他人而活
```

## 🧪 使用示例

### 示例 1：父母干涉工作选择

**输入：**
```
@ant 我父母想让我考公务员，但我想做设计工作，怎么办？
```

**输出：**
```
✂️ 课题分离分析

【这是谁的课题？】
• 选择什么工作 → 你的课题（你承担结果）
• 父母是否满意 → 父母的课题（他们的情绪）

【你的课题】
✓ 思考自己真正想做什么
✓ 评估设计工作的可行性
✓ 为自己的选择负责

【父母的课题】
• 他们是否满意你的选择
• 他们如何理解你的职业
• 他们的情绪和期待

【建议行动】
1. 和父母坦诚沟通你的想法
2. 倾听他们的担忧但不被控制
3. 做出自己的选择并为之负责
4. 理解他们的失望但不为此负责

【关键洞察】
为了满足别人的期望而活，
是对自己撒谎也不断对周围人撒谎的生活方式。
```

### 示例 2：难以拒绝别人请求

**输入：**
```
@ant 同事总让我帮忙，我不想但不好意思拒绝
```

**输出：**
```
✂️ 课题分离分析

【这是谁的课题？】
• 是否帮忙 → 你的课题
• 同事是否失望 → 同事的课题

【你的课题】
✓ 评估自己是否有时间和精力
✓ 根据自己的情况决定
✓ 诚实表达你的想法

【同事的课题】
• 被拒绝后的情绪
• 如何理解你的拒绝
• 是否找其他人帮忙

【建议行动】
1. 诚实说"我现在不方便"
2. 不必过度解释或道歉
3. 理解对方可能失望但不为此负责
4. 记住：不畏惧被讨厌才是自由

【关键洞察】
不畏惧被人讨厌而勇往直前，
不随波逐流而激流勇进，
这才是对人而言的自由。
```

## 📚 核心理念

**关键洞察：**
1. 一切人际关系矛盾都起因于对别人的课题妄加干涉
2. 满足别人的期望而活是对自己撒谎
3. 不畏惧被讨厌才是自由
4. 因反对就能崩塌的关系不必缔结
5. 课题分离不是疏远，是解开人际关系之线

**健康公式：**
```
人际关系自由 = 课题分离 + 横向关系 + 被讨厌的勇气
```
