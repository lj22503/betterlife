# 解释风格分析器 🔍

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户分析自己的解释风格，识别永久性、普遍性、人格化三个维度的倾向。

**适用场景：**
- 想了解自己的解释习惯
- 分析对事件的解释方式
- 识别悲观思维模式

---

## 🎯 核心功能

### 1. 解释风格三维度

**永久性维度（Permanence）：**
```
• 决定一个人会放弃多久
• 对厄运做永久性解释的人会有长期的无助
• 对同样事件做暂时性解释的人很快就能恢复
```

**普遍性维度（Pervasiveness）：**
```
• 具有普遍性特点的解释风格会在许多不同情境中造成无助
• 特定性的解释风格只会在某些问题领域内造成无助
• 一点小事都被看成大难临头 = 普遍性
```

**人格化维度（Personalization）：**
```
• 控制你如何看待自己，对自己的感觉
• 当不好的事情发生时，悲观的人怪罪自己
• 乐观的人怪罪旁人或环境
```

### 2. 分析框架

**遇到坏事时：**
| 维度 | 悲观解释 | 乐观解释 |
|------|---------|---------|
| 永久性 | "永远都会这样" | "只是暂时的" |
| 普遍性 | "所有事都糟了" | "只是这件事" |
| 人格化 | "都是我的错" | "是环境/运气" |

**遇到好事时：**
| 维度 | 悲观解释 | 乐观解释 |
|------|---------|---------|
| 永久性 | "只是暂时的" | "会一直这样" |
| 普遍性 | "只是这件事" | "所有事都会好" |
| 人格化 | "是运气好" | "是我的功劳" |

---

## ⚠️ 常见错误

**错误 1：忽视永久性维度**
```
问题：
• 认为"这次失败"不会影响下次
• 但潜意识认为是永久的

解决：
✓ 永久性维度决定放弃多久
✓ 识别自己的永久性解释
✓ 学习暂时性解释
```

## 📚 核心理念

**关键洞察：**
1. 解释风格有三个维度
2. 永久性决定放弃多久
3. 普遍性决定无助感的范围
4. 人格化影响自我感觉

---

*解释风格有三个维度——永久性、普遍性和人格化。* 🔍
