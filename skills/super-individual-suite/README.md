## 📚 核心理论

### 超级个体 49 个关键动作

**认知思维：**
- 像经营公司一样经营自己
- 一份时间卖多次
- 挣信息差的钱
- 稀缺：效果>知识

**个人品牌：**
- 火箭模型：作品 + 日更 + 口碑
- 人设三要素：真实、聚焦、稳定
- 内容筛选用户

### 个人画布 9 要素

1. 客户群体
2. 价值主张
3. 渠道通路
4. 客户关系
5. 收入来源
6. 核心资源
7. 关键业务
8. 重要合作
9. 成本结构

### 人生设计

**人生仪表盘：** 健康/工作/娱乐/爱
**人生指南针：** 工作观 + 人生观
**奥德赛计划：** 三个 5 年版本
**原型设计：** 对话 + 体验
