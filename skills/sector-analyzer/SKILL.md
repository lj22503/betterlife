# 行业板块分析器 🏭

**基于《回归投资常识》（齐俊杰）核心理念**

---

## 📋 功能描述

帮助用户理解四大行业板块（金融/消费/周期/科技）的投资逻辑，把握行业轮动机会。

**适用场景：**
- 行业分析
- 板块轮动
- 个股选择
- 行业基金配置

---

## 🎯 核心功能

### 1. 金融板块

**银行：**
- 逆周期防御
- 信用中介 + 支付中介 + 信用创造

**保险：**
- 顺周期进攻
- PEV、NBV、十年期国债收益率

**证券：**
- 牛市旗手
- 右侧交易，平台突破追击

### 2. 消费板块

**特点：**
- 长期稳定增长
- 品牌护城河

**白酒：**
- 毛利率 60%+，高端 80%+
- 定价权：G 端>B 端>C 端

**家电：**
- 重点关注白电
- 行业集中度高

### 3. 周期板块

**特点：**
- 强周期性
- 均值回归

**有色金属：**
- 铜是"经济学博士"
- 铜价预示经济复苏

**煤炭：**
- 动力煤价格决定利润

### 4. 科技板块

**特点：**
- 赢家通吃
- 波动大，易有泡沫

**TMT：**
- 宏观小年 + 货币宽松 + 科技周期

**半导体：**
- 周期 + 成长双重特性
- 费城半导体指数、销售额增速
