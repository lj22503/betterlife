# 人生和谐平衡器 ⚖️

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户实现人生和谐，平衡工作、家庭、育儿、交友、兴趣等各方面。

**适用场景：**
- 工作生活失衡
- 无暇顾及家庭
- 某一方面过度突出

**边界条件：**
- 不追求完美平衡
- 接受动态调整
- 不同阶段有不同侧重

---

## 🎯 核心功能

### 1. 人生五大领域

```
• 工作（公司上班）
• 家庭（家务）
• 育儿
• 交友
• 兴趣
```

### 2. 和谐原则

```
✓ 不认可任何一方面突出
✓ 平衡关注各方面
✓ "工作忙"是逃避的借口
```

---

## ⚠️ 常见错误

**错误 1：把工作当全部**
```
问题：
• "工作就是全部"
• "工作忙所以不管其他"

解决：
✓ 工作只是人生的一部分
✓ 家庭、育儿、交友、兴趣同样重要
✓ "工作忙"是逃避的借口
```

**错误 2：追求完美平衡**
```
问题：
• "每天都要平衡所有方面"
• 做不到就自责

解决：
✓ 接受动态调整
✓ 不同阶段有不同侧重
✓ 长期平衡而非每天平衡
```

## 🧪 使用示例

**输入：**
```
@ant 工作太忙，无暇顾及家庭
```

**输出：**
```
⚖️ 人生和谐分析

【人生五大领域】
1. 工作 2. 家庭 3. 育儿 4. 交友 5. 兴趣

【问题诊断】
• 工作过度突出
• 其他方面被忽视

【建议行动】
1. 重新分配时间
2. 关注家庭、交友、兴趣
3. 不认可工作至上的生活方式

【关键洞察】
"工作忙，所以无暇顾及家庭"
是人生的谎言，是逃避其他责任的借口。
```
