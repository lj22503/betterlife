# 习得性无助打破器 🔨

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户识别并打破习得性无助，从"无论怎么努力都于事无补"的想法中走出来。

**适用场景：**
- 感到无助、放弃
- 觉得努力没用
- 抑郁情绪
- 想重新振作

---

## 🎯 核心功能

### 1. 习得性无助特征

**习得性无助：**
```
• 放弃的反应
• 源自"无论你怎么努力都于事无补"的想法
• 行为上的放弃
• 情绪上的抑郁
```

### 2. 打破方法

**步骤 1：识别无助想法**
```
"无论怎么努力都于事无补"
```

**步骤 2：检验想法**
```
• 真的"都于事无补"吗？
• 有什么证据？
• 有没有例外？
```

**步骤 3：小步行动**
```
• 从小的可控的事开始
• 积累成功经验
• 重建掌控感
```

---

## ⚠️ 常见错误

**错误 1：认为无助是真实的**
```
问题：
• "真的做什么都没用"
• 把想法当事实

解决：
✓ 无助是想法，不是事实
✓ 检验这个想法
✓ 从小事开始行动
```
