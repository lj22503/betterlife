# 当下生活实践器 ⏰

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户认真的人生"活在当下"，不纠结过去，不担忧未来。

**适用场景：**
- 过度担忧未来
- 纠结过去的事情
- 无法专注当下

---

## 🎯 核心功能

### 1. 当下原则

```
✓ 过去不能决定现在
✓ 未来不必过度担忧
✓ 重要的是"此时此刻"
```

### 2. 人生是连续的刹那

```
• 人生不是线性的
• 而是每一个瞬间的连续
• 认真过好每一个"现在"
```

---

