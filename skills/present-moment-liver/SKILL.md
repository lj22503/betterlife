# 当下生活实践器 ⏰

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户认真的人生"活在当下"，不纠结过去，不担忧未来。

**适用场景：**
- 过度担忧未来
- 纠结过去的事情
- 无法专注当下

**边界条件：**
- 不否定规划未来
- 接纳过去的经历
- 专注当下的行动

---

## 🎯 核心功能

### 1. 当下原则

```
✓ 过去不能决定现在
✓ 未来不必过度担忧
✓ 重要的是"此时此刻"
```

### 2. 人生是连续的刹那

```
• 人生不是线性的
• 而是每一个瞬间的连续
• 认真过好每一个"现在"
```

---

## ⚠️ 常见错误

**错误 1：把当下当借口**
```
问题：
• "活在当下就是不规划"
• "不想未来"

解决：
✓ 活在当下不等于不规划
✓ 规划未来但不过度担忧
✓ 专注当下的行动
```

**错误 2：纠结过去**
```
问题：
• "如果当初..."
• 为过去的事情自责

解决：
✓ 过去不能决定现在
✓ 接纳过去的经历
✓ 重要的是"此时此刻"的你
```
