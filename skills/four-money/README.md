# 管好四笔钱 Skill 套件 💰

**基于且慢《管好四笔钱》的家庭账户投资管理技能**

---

## 📖 简介

这套技能将且慢的四笔钱投资理念转化为实用的 AI 工具，帮助家庭建立科学的账户体系，实现资产合理配置。

**核心理念：**
- 💵 **活钱管理** —— 3-6 个月生活费，货币基金
- 📈 **稳健理财** —— 3 年内明确用途，债券基金
- 🚀 **长期投资** —— 3 年以上闲钱，指数基金定投
- 🛡️ **保险保障** —— 风险缓冲，年收入 5-20%

## 📚 四笔钱账户体系

| 账户 | 用途 | 投资品种 | 金额 |
|------|------|----------|------|
| 活钱管理 | 日常开支、应急 | 货币基金 | 3-6 个月支出 |
| 稳健理财 | 3 年内明确用途 | 债券基金、固收 + | 根据需求 |
| 长期投资 | 3 年以上闲钱 | 指数基金、股票基金 | 剩余资金 |
| 保险保障 | 风险缓冲 | 重疾/医疗/意外/寿险 | 年收入 5-20% |
