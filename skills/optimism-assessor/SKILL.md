# 乐观水平评估器 🌞

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户评估自己的乐观水平，了解解释风格，认识自己是悲观者还是乐观者。

**适用场景：**
- 想了解自己的乐观程度
- 测试解释风格
- 心理咨询自我评估
- 想改变悲观思维

**边界条件：**
- 评估不是诊断
- 乐观可以习得
- 解释风格可以改变

---

## 🎯 核心功能

### 1. 乐观 vs 悲观画像

**悲观者的特征：**
```
• 相信坏事都是因为自己的错
• 这件事会毁掉一切
• 会持续很久
• 把好运看成暂时性的
```

**乐观者的特征：**
```
• 认为失败是暂时性的
• 每个失败都有它的原因，不是自己的错
• 可能是环境、运气或其他人为原因
• 不会被失败击倒
• 把困难看成挑战
```

### 2. 解释风格三维度

| 维度 | 悲观解释 | 乐观解释 |
|------|---------|---------|
| **永久性** | 永远都会这样 | 只是暂时的 |
| **普遍性** | 所有事都糟了 | 只是这件事 |
| **人格化** | 都是我的错 | 是环境/运气/别人 |

### 3. 快速评估

**评估问题：**
- [ ] 遇到坏事时，你认为是暂时的还是永久的？
- [ ] 你认为这件事会影响其他方面吗？
- [ ] 你认为是自己的错还是外部原因？
- [ ] 遇到好事时，你认为是自己的功劳还是运气？

## ⚠️ 常见错误

**错误 1：认为乐观是天生的**
```
问题：
• "我天生就是悲观的人"
• "乐观的人天生乐观"

解决：
✓ 乐观是可以习得的
✓ 解释风格可以改变
✓ 通过 ABCDE 技术可以转变
```

**错误 2：认为乐观就是盲目积极**
```
问题：
• "乐观就是无视问题"
• "盲目积极就好"

解决：
✓ 乐观不是盲目积极
✓ 是理性的解释风格
✓ 乐观需要与智慧结合
```

## 📚 核心理念

**关键洞察：**
1. 解释风格有三个维度：永久性、普遍性、人格化
2. 悲观者将坏事归因于永久的、普遍的、内在的因素
3. 乐观者将坏事归因于暂时的、特定的、外在的因素
4. 乐观是可以习得的

**健康公式：**
```
乐观水平 = 永久性维度 + 普遍性维度 + 人格化维度
```
