# 宁静情绪发现器 🧘

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户寻找和品味宁静情绪，在安全熟悉的环境中感受低调的平和。

**适用场景：**
- 想感到平静
- 压力后放松
- 睡前放松
- 日常情绪培养

**边界条件：**
- 宁静需要安全熟悉的环境
- 自身不需要付出太多努力
- 比喜悦更低调

---

## 🎯 核心功能

### 1. 宁静的特征

**与喜悦的异同：**
```
相同点：
• 周围安全而熟悉
• 自身不需要付出太多努力

不同点：
• 宁静要低调得多
• 喜悦更活跃
```

### 2. 宁静的场景

**典型场景：**
```
• 叹出长长的、舒爽的一口气
• 感到目前的状况是如此舒服和顺畅
• 经过辛苦而有意义的一天后，躺在花园里的吊床上小憩
• 在明媚的早晨，随着大海的声音撞击脑海、凉爽的微风轻触肌肤，在沙滩上散步
• 捧着一本好书蜷缩在沙发上，腿上趴着一只温暖的猫咪，身边放着一杯你最爱的清茶
• 瑜伽练习中的放松体式，沉入蒲垫之中
```

### 3. 培养方法

**微习惯：**
```
• 叹出长长的、舒爽的一口气
• 品味辛苦一天后的放松
• 在自然中散步
• 与好书、好茶、宠物相伴
• 瑜伽放松体式
```

---

## ⚠️ 常见错误

**错误 1：追求强烈的感受**
```
问题：
• "我要非常平静"
• 追求强烈放松

解决：
✓ 宁静是低调的积极情绪
✓ 不需要追求强烈
✓ 自然感受即可
```
