# 人生坐标定位器 🧭

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户定位自己的人生坐标，从"我不好"走向"你好我好"。

**适用场景：**
- 感到自己是受害者
- 焦虑、抑郁、自责
- 挑剔、指责别人
- 想建立健康人际关系

---

## 🎯 核心功能

### 1. 四个人生坐标

| 坐标 | 信念 | 情绪 | 行为 |
|------|------|------|------|
| **你好，我不好** | 我是受害者 | 焦虑、抑郁、自责 | 顺从、取悦 |
| **我好，你不好** | 我是对的 | 愤怒、挑剔、指责 | 攻击、批评 |
| **我不好，你也不好** | 都没希望 | 反复无常、报复 | 自毁、攻击 |
| **你好，我也好** | 彼此都好 | 安全、信任、合作 | 合作、促进 |

### 2. 坐标识别

**你在"你好，我不好"吗？**
- [ ] 认为自己是生活的受害者
- [ ] 无法掌控人生
- [ ] 焦虑、抑郁、自责

**你在"我好，你不好"吗？**
- [ ] 占据权力和权威的制高点
- [ ] 攻击、指责别人
- [ ] 愤怒、挑剔

**你在"我不好，你也不好"吗？**
- [ ] 既自卑又自大
- [ ] 自我贬低也攻击他人
- [ ] 情绪反复无常

**你在"你好，我也好"吗？**
- [ ] 相信自己是好的
- [ ] 相信别人是好的
- [ ] 对自己和别人展现美好

---

## ⚠️ 常见错误

**错误 1：认为坐标是固定的**
```
问题：
• "我就是这样的人"
• 坐标无法改变

解决：
✓ 坐标是一种选择和承诺
✓ 可以主动选择改变
✓ "你好我也好"是终极目标
```
