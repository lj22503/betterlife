# 悲观思维转换器 🔄

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户转变悲观思维，从悲观解释风格转向乐观解释风格。

**适用场景：**
- 觉得自己太悲观
- 想变得乐观
- 经常负面思考
- 想改变解释习惯

---

## 🎯 核心功能

### 1. 悲观思维特征

**悲观者的解释风格：**
```
• 坏事 = 永久的 + 普遍的 + 自己的错
• 好事 = 暂时的 + 特定的 + 运气好
```

### 2. 转换方法

**从永久到暂时：**
```
"永远都会这样" → "只是暂时的"
```

**从普遍到特定：**
```
"所有事都糟了" → "只是这件事"
```

**从内在到外在：**
```
"都是我的错" → "是环境/运气/别人"
```

### 3. 日常练习

**每天记录 5 件 ABC：**
```
• 不好的事
• 想法
• 后果
• 反驳
• 激发
```

---

## ⚠️ 常见错误

**错误 1：急于求成**
```
问题：
• "我想马上变乐观"
• 几天没效果就放弃

解决：
✓ 解释风格是习惯，需要时间改变
✓ 每天练习 ABCDE
✓ 持续 2-4 周见效
```
