# CFO 财务管理 Skill 套件 💼

**基于《从总账到总监》的财务管理与领导力技能**

---

## 📖 简介

这套技能帮助财务专业人士从总账会计成长为财务总监，提供完整的工具速查手册和领导力发展路径。

**核心理念：**
- 📊 **CFO 工具** —— 六大类工具速查
- 👔 **财务领导** —— 从价值贡献者到资源组织者
- 👥 **团队建设** —— 留才 3M 法和学习型组织
- 💎 **价值创造** —— 精益财务四象限
- 🎯 **决策支持** —— OCA 模型和批判性思考

## 📚 核心工具速查

### 职场成长类

| 工具 | 用途 | 关键点 |
|------|------|--------|
| 晋升三要素 | 职业规划 | 能力/关系/可视度 |
| 关注圈与影响圈 | 精力分配 | 聚焦可控事项 |
| 职业性格三色 | 自我认知 | 红宝石/绿宝石/蓝宝石 |

### 团队管理类

| 工具 | 用途 | 关键点 |
|------|------|--------|
| 留才 3M 法 | 人才保留 | Meaning/Menu/Mentor |
| 管理者三大杠杆 | 角色转型 | 营运/资本/资源 |
| 学习型组织 | 能力建设 | 每周学习会/知识库/季度研习营 |

### 沟通协作类

| 工具 | 用途 | 关键点 |
|------|------|--------|
| 沟通次序清单 | 敏感事项 | 优先与最重要的人沟通 |
| 情感账户 | 信任积累 | 从点滴做起 |
| 关键对话 | 冲突解决 | 内容 + 氛围 |

### 决策分析类

| 工具 | 用途 | 关键点 |
|------|------|--------|
| OCA 模型 | 多方案选择 | 选项/比较/行动 |
| 六顶思考帽 | 全面思考 | 蓝/白/红/黄/黑/绿 |
| 卡诺价值模型 | 资源投入 | 基本/兴奋/无差异 |

### 时间任务类

| 工具 | 用途 | 关键点 |
|------|------|--------|
| 大石头优先 | 优先级 | 先做重要不紧急 |
| 四象限 | 时间管理 | 第一/二/三/四象限 |
| TFT 模型 | 基础技能 | Task/Feedback/Time |

### 财务专项类

| 工具 | 用途 | 关键点 |
|------|------|--------|
| 精益财务四象限 | 工作优先级 | 身边的金子/地上的果子/该打的硬仗/问题墙 |
| 决策成本三要素 | 决策支持 | 取舍/节约/优化 |
| 标准成本 | 成本监控 | 管理思维 vs 记账思维 |
