# 职场燃料评估器 ⛽

**基于《远见：如何规划职业生涯 3 大阶段》核心理念**

---

## 📋 功能描述

帮助用户评估和积累三种职场燃料：可迁移技能、有意义的经验、持久的关系。

**适用场景：**
- 职业规划
- 职业转型
- 职场进阶
- 退休规划

---

## 🎯 核心功能

### 1. 三种职场燃料评估

**可迁移技能：**
- 学术学位、专业证书
- 语言能力
- 优点/特长
- 情商
- 人才账户

**有意义的经验：**
- 个人旅行
- 海外工作经验
- 企业管理/创业经验
- 社区/志愿者活动
- 公开演讲/写作/表演经验
- 教学/咨询/指导经验

**持久的关系：**
- 联系人
- 专家团
- 关键同事（5-10 人）
- 支持者/导师（3-5 人）

### 2. 5 个关键数字

1. **职业生涯长度** = 62 - 目前年龄
2. **精通技能时间** = 10000 小时
3. **40 岁后财富占比** = 85%-90%
4. **社交货币** = 不是越多越好
5. **导师人数** = 3-5 个

### 3. 关键同事识别

**影响力排序：**
1. 直接上司（头号人物）
2. 同事
3. 下属
4. 跨部门合作者

### 4. 4 大黄金问题

1. 我是否正在学习和成长？
2. 我是否正在对某些人、公司、社会拥有影响力？
3. 我体验到乐趣了吗？
4. 我是否得到了适当的奖励，并创造了经济价值？
