# 自豪情绪庆祝器 🏆

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户庆祝自豪情绪，在成就后绽放自信，分享好消息。

**适用场景：**
- 想为自己自豪
- 庆祝成就
- 提升自信
- 分享好消息

**边界条件：**
- 自豪紧随着成就而绽放
- 需要投入努力并取得成功
- 值得称道的事情

---

## 🎯 核心功能

### 1. 自豪的特征

**引发自豪的条件：**
```
• 你投入了努力
• 并取得了成功
• 做了一些值得称道的事情
• 通过你的努力成就了一些什么
```

### 2. 自豪的感受

**身体感受：**
```
• 对自己的能力充满信心和自我肯定
• 昂首挺胸
• 想要与别人分享你的好消息
• 被吸引去梦想未来可能取得的成就
```

### 3. 培养方法

**微习惯：**
```
• 记录自己的成就
• 庆祝小胜利
• 分享好消息
• 肯定自己的能力
• 梦想未来的成就
```

---

## ⚠️ 常见错误

**错误 1：忽视小成就**
```
问题：
• "这不算什么成就"
• 忽视小胜利

解决：
✓ 自豪紧随着成就而绽放
✓ 小成就也值得庆祝
✓ 肯定自己的努力
```
