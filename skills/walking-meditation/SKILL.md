# 行走冥想引导器 🚶

**基于《十分钟冥想》（安迪·普迪科姆）**

---

## 📋 功能描述

帮助用户行走时的正念觉察。

**适用场景：**
- 行走时的正念觉察
- 提升正念意识
- 培养觉察力

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 轻轻地闭上眼睛
2. 将注意力集中在当下的体验上
3. 不要试图阻止或改变想法
4. 当心灵游离时，温和地带回
5. 保持适度的好奇心
```

---

## ⚠️ 常见错误

**错误 1：努力控制**
```
问题：
• "我要努力控制心灵"

解决：
✓ 冥想不是要去阻止想法、控制心灵
✓ 在这个过程中，我们放弃控制
✓ 任由想法来去
```

## 📚 核心理念

**关键洞察：**
1. 冥想不是控制心灵
2. 放弃控制，不插手，不介入
3. 保持适度的好奇心
4. 温和地带回注意力

---

*冥想不是要去阻止想法、控制心灵。
