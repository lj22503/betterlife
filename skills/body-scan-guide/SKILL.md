# 身体扫描引导器 🦶

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行身体扫描练习，系统地关注身体每个部位的感觉。

**适用场景：**
- 身体紧张需要放松
- 睡前放松
- 正念入门练习
- 身体觉察训练

**边界条件：**
- 练习中可能入睡是正常的
- 不要试图改变感觉
- 没有感觉也是正常的

---

## 🎯 核心功能

### 1. 准备姿势

**推荐姿势：**
```
• 仰卧在温暖舒适的地方
• 可以躺在床上或地板
• 如果舒服可以裹毯子
• 闭眼或睁眼都可以
```

### 2. 扫描顺序

**扫描路径：**
```
脚趾 → 脚心 → 脚背 → 脚跟 → 脚踝
→ 小腿 → 膝盖 → 大腿
→ 骨盆 → 腹股沟 → 臀部
→ 腹部 → 后背 → 胸部
→ 双手 → 手臂 → 肩膀
→ 脖子 → 面部 → 整个头部
```

### 3. 练习要点

**每个部位：**
```
1. 将意念引导到该部位
2. 探索这种感觉（麻刺/温暖/麻木/无感觉）
3. 不要判断，让感觉自由运行
4. 吸气时想象气息进入，呼气时离开
5. 停留 20-30 秒
```

### 4. 处理意念游离

**正确态度：**
```
✓ 意念游离非常正常
✓ 温和承认事实
✓ 记下意念转移到哪里
✓ 温和引导回关注的部位
```

---

## ⚠️ 常见错误

**错误 1：试图改变感觉**
```
问题：
• "我应该放松"
• 刻意尝试更加放松

解决：
✓ 不要试图改变任何东西
✓ 充分关注真实感受
✓ 没有感觉也是正常的
```

**错误 2：责备自己入睡**
```
问题：
• "我怎么又睡着了"
• 责备自己无法保持清醒

解决：
✓ 身体扫描容易产生放松感
✓ 入睡是正常的
✓ 如果总是入睡，可以坐着练习
```
