# 希望情绪增强器 🌅

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户增强希望情绪，在境况紧迫时保持乐观，相信事情会往好的方向发展。

**适用场景：**
- 想感到有希望
- 面对不确定性
- 境况紧迫时
- 情绪低落时

**边界条件：**
- 希望在境况紧迫时发挥作用
- 事情发展对你不利时
- 存在相当大的不确定性

---

## 🎯 核心功能

### 1. 希望的特征

**与其他积极情绪的不同：**
```
• 大多数的积极情绪都是在你感到安全和满足的时候出现
• 但是希望是一个例外
• 希望在你境况紧迫的时候发挥作用
• 事情的发展对你不利
• 或者是关于事情将如何发展存在着相当大的不确定性
```

### 2. 希望的触发条件

**引发希望的条件：**
```
• 事情看来将要无望或绝望的时候
• 你对最坏的结果感到恐惧
• 却又在某种程度上仍然相信事情会往好的方向发展
• 真真切切地渴望某些更好的情况发生
```

### 3. 培养方法

**微习惯：**
```
• 发掘创造性去为更好的未来做出努力
• 相信事情会往好的方向发展
• 渴望某些更好的情况发生
• 在不确定性中寻找可能性
```

---

## ⚠️ 常见错误

**错误 1：在顺境中期待希望**
```
问题：
• "我希望一切都好"
• 在顺境中期待希望

解决：
✓ 希望在境况紧迫时发挥作用
✓ 如果一切都已经在按照你希望的方式发展
✓ 那么你基本上就没有什么需要希望的了
```
