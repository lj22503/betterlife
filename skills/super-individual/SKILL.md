# 超级个体成长器 🚀

**基于《单干：成为超级个体的 49 个关键动作》核心理念**

---

## 📋 功能描述

帮助用户像经营公司一样经营自己，成为超级个体，实现一份时间卖多次。

**适用场景：**
- 想发展副业
- 想成为自由职业者
- 想打造个人品牌
- 想实现财务自由

---

## 🎯 核心功能

### 1. 一人公司思维

**商业理解 3 阶段：**
- 小白看规模
- 高手看现金流
- 大神看人效

**核心资源：** 客户的信任

### 2. 时间销售模式

**三种模式：**
- 小白：一份时间卖一次
- 高手：一份时间卖多次（课程/书/程序）
- 大神：买卖别人的时间

### 3. 杠杆思维

**四种杠杆：**
- 内容杠杆（书/课程/视频）
- 代码杠杆（程序/工具）
- 媒体杠杆（播客/直播）
- 资本杠杆（投资/收购）

### 4. 个人品牌火箭模型

- 作品：代表作
- 日更：持续输出
- 口碑：用户故事

## 💡 使用建议

**完整超级个体成长流程：**

1. **一人公司画布** - 像经营公司一样经营自己
2. **个人品牌火箭** - 作品 + 日更 + 口碑
3. **动力体系** - 习惯 + 体系 + 心力
4. **选择决策** - 河流模型 + 难而正确

**关键原则：**
- 一份时间卖多次（边际成本趋近于零）
- 先卖再做（纸板模型→MVP→正式产品）
- 品牌才是持续的流量
- 离钱近就是主动降低不确定性
- 面对选择时选难而正确的事

**49 个关键动作分类：**
- 认知思维（14 个）：经营自己/一人公司/杠杆/对标/跨界等
- 个人品牌（11 个）：火箭模型/定位/流量/内容/写作/讲故事等
- 动力（8 个）：推石模型/习惯/体系/心力/节奏感/聚焦等
- 学习（8 个）：大树模型/终身学习/研究驱动/知识图谱/访谈等
- 选择（8 个）：河流模型/难而正确/长期主义/先胜后战/边界等
