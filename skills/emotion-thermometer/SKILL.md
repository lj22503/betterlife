# 情绪温度计 🌡️

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户量化情绪状态，真实呈现内心感受，追踪情绪变化。

**适用场景：**
- 不知道自己的情绪状态
- 需要追踪情绪变化
- 心理咨询自我评估
- 情绪量化记录

**边界条件：**
- 不用刻意在乎分数本身
- 只需真实呈现内心感受
- 1 分最低（非常糟糕），10 分最高（非常愉悦）

---

## 🎯 核心功能

### 1. 情绪温度计（1-10 分）

| 分数 | 状态 | 描述 |
|------|------|------|
| **1-2 分** | 非常糟糕 | 可能想自杀，极度抑郁 |
| **3-4 分** | 糟糕 | 悲哀、焦虑、无精打采 |
| **5-6 分** | 还能承受 | 日常状态，有起伏 |
| **7-8 分** | 良好 | 心情愉悦，有活力 |
| **9-10 分** | 非常愉悦 | 感觉好极了，身心愉悦 |

### 2. 使用步骤

**步骤 1：静心觉察**
```
闭上眼睛，感受此刻的内心
```

**步骤 2：诚实评分**
```
1-10 分，选择最符合的分数
```

**步骤 3：记录感受**
```
为什么是这个分数？
```

**步骤 4：追踪变化**
```
每天记录，观察趋势
```

---

## ⚠️ 常见错误

**错误 1：刻意追求高分**
```
问题：
• "我应该打高分"
• 把打分当考试

解决：
✓ 只需真实呈现内心感受
✓ 不用刻意在乎分数本身
✓ 有助于了解自己、作出调整
```

**错误 2：习惯性说"挺好的"**
```
问题：
• 习惯说"挺好的"
• 隐瞒真实情绪

解决：
✓ 真实才是了解内心世界的入口
✓ 允许自己表达负面情绪
✓ 表现悲伤不是无能懦弱的表现
```
