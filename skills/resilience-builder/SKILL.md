# 心理韧性构建器 💪

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户提升心理韧性，从挫折中快速恢复，增强抗压能力。

**适用场景：**
- 想提升心理韧性
- 抗压能力差
- 容易被打倒
- 想从挫折中恢复

---

## 🎯 核心功能

### 1. 心理韧性特征

**有韧性的人：**
```
• 从挫折中快速恢复
• 把困难看成挑战
• 不会被失败击倒
• 更努力地去克服困难
```

### 2. 构建方法

**方法 1：乐观解释风格**
```
• 将失败归因于暂时的原因
• 将失败归因于特定的原因
• 不怪罪自己
```

**方法 2：社会支持**
```
• 建立深厚的友谊和爱情
• 与配偶或情人有亲密关系
• 寻求支持
```

**方法 3：工作/目标**
```
• 外出工作
• 有明确的目标
• 有意义感
```

---

## ⚠️ 常见错误

**错误 1：认为韧性是天生的**
```
问题：
• "我天生就脆弱"
• 韧性无法改变

解决：
✓ 韧性可以习得
✓ 通过乐观解释风格
✓ 通过社会支持
```
