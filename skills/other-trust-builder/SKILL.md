# 他者信赖构建器 🤝

**基于《被讨厌的信任》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户建立他者信赖，区分"信任"和"怀疑"，建立深度人际关系。

**适用场景：**
- 难以信任别人
- 害怕被背叛
- 人际关系疏离
- 无法建立深度连接

---

## 🎯 核心功能

### 1. 他者信赖三原则

**原则 1：信任是你的课题**
```
✓ 选择信任别人（你的课题）
✓ 如何对待信任是别人的课题
✓ 不因害怕背叛而不信任
```

**原则 2：无条件信赖**
```
✓ 不是基于条件的信任
✓ 而是基于选择的信任
✓ 即使可能被背叛
```

**原则 3：接纳结果**
```
✓ 信任被辜负时接纳痛苦
✓ 但不因此关闭信任
✓ 继续选择信任值得的人
```

---

## ⚠️ 常见错误

**错误 1：把信任当赌注**
```
问题：
• "我信任你，你不能背叛我"
• 信任附带条件

解决：
✓ 信任是你的选择
✓ 别人如何对待是别人的课题
```

## 📚 核心理念

**关键洞察：**
1. 信任是你的课题
2. 如何对待信任是别人的课题
3. 无条件信赖才能建立深度关系
4. 不因害怕背叛而关闭信任

**健康公式：**
```
深度关系 = 课题分离 + 无条件信赖 + 自我接纳
```
