# 排除饮食计划

**配合 `lectin-avoidance` 技能使用**

---

## 第 1-2 周：排除期

### 避免的食物

**高凝集素食物：**
- [ ] 豆类（黄豆、黑豆、红豆等）
- [ ] 全谷物（小麦、大麦、燕麦等）
- [ ] 茄科植物（番茄、茄子、辣椒、土豆）
- [ ] 坚果（花生、腰果等）

### 可以吃的食物

**低凝集素食物：**
- [ ] 肉类（草饲牛肉、鸡肉等）
- [ ] 鱼类（野生鱼等）
- [ ] 蔬菜（绿叶蔬菜、十字花科等）
- [ ] 健康脂肪（橄榄油、牛油果等）
