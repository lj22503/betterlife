# 友善禅修引导器 💗

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行友善禅修（慈心禅），培养对自己和他人的爱心和善意。

**适用场景：**
- 培养自我接纳
- 增强对他人的善意
- 正念进阶练习
- 情绪调节

**边界条件：**
- 如果难以善待自己，想象爱你的人
- 感到压力过大时回到呼吸
- 不要压制或评判感受

---

## 🎯 核心功能

### 1. 准备：稳定精神

**步骤：**
```
1. 选择温暖舒适的环境
2. 确保姿势端正
3. 将注意力放在呼吸
4. 扩展至全身
5. 直到精神稳定
```

### 2. 对自己释放友善

**祝福语句：**
```
• 愿我远离痛苦
• 愿我尽可能的快乐健康
• 愿我生活幸福
```

**练习方法：**
```
1. 让词句进入脑海
2. 每次只投放一颗"石子"
3. 倾听思想、感受、知觉的反应
4. 不要判断出现的反应
```

### 3. 扩展友善

**扩展顺序：**
```
1. 对自己
2. 一个你爱的人
3. 一个陌生人
4. 一个让你不舒服的人
5. 所有人（包括自己）
```

### 4. 难以善待自己时

**方法：**
```
1. 想象一个无条件爱你的人（或宠物）
2. 清晰感受他们的疼爱
3. 将这种爱释放到自己身上
```

---

## ⚠️ 常见错误

**错误 1：难以善待自己**
```
问题：
• "我不值得被爱"
• 无法对自己释放友善

解决：
✓ 想象无条件爱你的人
✓ 清晰感受他们的疼爱
✓ 将这种爱释放到自己身上
```

**错误 2：压制感受**
```
问题：
• "我不应该有这种感觉"
• 评判出现的反应

解决：
✓ 不要压制或评判
✓ 倾听反应
✓ 所有反应都是属于你的
```
