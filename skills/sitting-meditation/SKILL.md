# 坐禅引导器 🪑

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行坐禅练习，培养全身觉察和呼吸觉察。

**适用场景：**
- 正念基础练习
- 提升觉察力
- 情绪调节
- 身心放松

**边界条件：**
- 不控制呼吸
- 不改变感觉
- 循序渐进练习

---

## 🎯 核心功能

### 1. 准备姿势

**坐姿要求：**
```
• 坐在椅子、坐垫或凳子上
• 后背挺直但不僵硬
• 脊柱自行支撑
• 双脚平放地板
• 双手放在大腿上
• 闭眼或视线放低
```

### 2. 练习序列

**步骤 1：呼吸觉察（5-8 分钟）**
```
• 注意力放在腹部
• 感受气息进出
• 不控制呼吸
```

**步骤 2：全身觉察（5-8 分钟）**
```
• 扩展注意力到全身
• 感受身体与支撑物接触的感觉
• 感受身体作为一个整体呼吸
```

**步骤 3：开放觉察（5-10 分钟）**
```
• 关注任何出现的感受
• 愉悦/不愉悦/中性
• 怀有善意的好奇
```

### 3. 处理强烈感受

**方法：**
```
1. 短暂转移注意力到该部位
2. 探索感觉的品质、位置、变化
3. 用呼吸将意识运送到该部位
4. 然后回到呼吸或全身觉察
```

---

## ⚠️ 常见错误

**错误 1：身体僵硬**
```
问题：
• "我必须坐得笔直"
• 过度紧张

解决：
✓ 身体竖直但不僵硬
✓ 舒适而端正
✓ 双肩放松
```

**错误 2：抗拒不愉悦感受**
```
问题：
• "这个感觉不好，我要摆脱"
• 试图改变感受

解决：
✓ 如实感受所有感觉
✓ 怀有善意的好奇
✓ 不试图改变
```
