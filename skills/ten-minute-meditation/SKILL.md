# 十分钟冥想引导器 🧘

**基于《十分钟冥想》（安迪·普迪科姆）**

---

## 📋 功能描述

引导用户进行十分钟冥想练习，培养头脑空间，提升正念意识。

**适用场景：**
- 日常冥想练习
- 减压放松
- 提升专注力
- 培养正念意识

**边界条件：**
- 每天练习效果最佳
- 不需要努力创造心境
- 允许想法来去

---

## 🎯 核心功能

### 1. 十分钟冥想四部分

**第一部分：签到（1-2 分钟）**
```
1. 做 5 次深呼吸，鼻子吸气，嘴巴呼气
2. 轻轻闭上眼睛
3. 将注意力集中在身体的躯体感觉上
   - 身体落座的感觉
   - 脚放在地板上的感觉
4. 扫描全身，留意舒适和紧张的区域
5. 留意自己的情感——你现在处于什么心情
```

**第二部分：驯服野马（5-6 分钟）**
```
1. 把心灵带到自然而舒服的安定之地
2. 将注意力集中在呼吸的起伏上
3. 不要试图阻止或改变想法
4. 当心灵游离时，温和地带回呼吸
5. 像看云卷云舒一样观察想法
```

**第三部分：沉浸（1-2 分钟）**
```
1. 完全解放自己的心灵
2. 享受静默
3. 允许想法自由来去
4. 感受头脑空间的宁静
```

**第四部分：带入生活（1 分钟）**
```
1. 有意识地将沉浸感带入日常生活
2. 将觉醒意识带入人际关系
3. 留意细微的事情
4. 保持正念状态
```

### 2. 核心理念

**蓝色的天空：**
```
天空总是蓝的
云相当于我们的思想
当心灵忙于这些想法的时候，蓝色的天空就暂时被遮蔽了
冥想不是努力把阴云赶走
而是：搬把椅子到花园里，坐看云卷云舒
```

**头脑空间：**
```
一种深层的宁静感
一种满足感
一种不可动摇的富足感
无关乎当时的情绪
```

## ⚠️ 常见错误

**错误 1：努力创造心境**
```
问题：
• "我要努力放松"
• 努力创造头脑空间

解决：
✓ 冥想不是需要努力去创造一种人为的心境
✓ 不要努力把阴云赶走
✓ 坐看云卷云舒
```

**错误 2：抗拒想法**
```
问题：
• "我不要这些想法"
• 抗拒心中的想法

解决：
✓ 冥想不是要去阻止想法、控制心灵
✓ 在这个过程中，我们放弃控制
✓ 任由想法来去
```
