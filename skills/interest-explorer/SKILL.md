# 兴趣情绪探索器 🔍

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户培养兴趣情绪，被新颖或奇怪的事物吸引，探索和学习。

**适用场景：**
- 想培养好奇心
- 想学习新东西
- 提升探索欲
- 日常情绪培养

**边界条件：**
- 需要新颖或奇怪的事物
- 需要努力和更多关注
- 感到绝对安全

---

## 🎯 核心功能

### 1. 兴趣的特征

**与喜悦和宁静的不同：**
```
• 虽然你感到绝对安全
• 但一些新颖的或奇怪的事物吸引了你的注意
• 用一种带着可能性和神秘性的感觉将你填满
• 需要你的努力和更多的关注
• 你完全被吸引了
```

### 2. 兴趣的感受

**身体感受：**
```
• 被牵引着去探索
• 将自己沉浸于正在接触的事物当中
• 觉得极其开放和活跃
• 内心的视野在不断扩展
• 有强烈的欲望去更多地探索和学习
```

### 3. 培养方法

**微习惯：**
```
• 尝试新的事物
• 学习新的知识
• 探索未知的领域
• 保持好奇心
• 将自己沉浸于新发现中
```

---

## ⚠️ 常见错误

**错误 1：被动等待兴趣**
```
问题：
• "等我有兴趣再做"
• 被动等待

解决：
✓ 主动寻找新颖的事物
✓ 尝试新的活动
✓ 保持好奇心
```
