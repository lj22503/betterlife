# 理性感性平衡器 ⚖️

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户在理性与感性之间找到平衡，真正厉害的人是理性与感性并存。

**适用场景：**
- 太理性显得没人情味
- 太情绪化容易冲动
- 想平衡理性与感性
- 情绪管理困扰

**边界条件：**
- 人人在理性与感性之间徘徊
- 真正厉害的人是理性与感性并存
- 不同事情需要不同处理方式

---

## 🎯 核心功能

### 1. 理性 vs 感性

**理性思维：**
```
• 对事物进行理智分析和判断
• 头脑不发热
• 决策不冲动
• 适合处理复杂问题
```

**感性思维：**
```
• 凭借心情和感受
• 感情用事
• 不计后果
• 适合体验快乐和连接
```

### 2. 平衡原则

**何时用理性：**
- [ ] 做重要决策时
- [ ] 处理复杂问题时
- [ ] 感到痛苦时（用理性干预）

**何时用感性：**
- [ ] 感受快乐时
- [ ] 与人连接时
- [ ] 体验美好时

---

## ⚠️ 常见错误

**错误 1：否定感性**
```
问题：
• "感性是不好的"
• "我要完全理性"

解决：
✓ 少了感性会显得没人情味
✓ 感性之心可以使我们触目生春
✓ 快乐时让感性尽情飞驰
```

**错误 2：否定理性**
```
问题：
• "理性太冷漠"
• "跟着感觉走就好"

解决：
✓ 少了理性会显得太死板
✓ 理性是在嘈杂世界中拯救生命的力量
✓ 痛苦时让理性干预感性
```

