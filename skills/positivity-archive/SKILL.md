# 积极情绪档案袋 📁

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户建立 10 个积极情绪档案袋，收藏与每种积极情绪相关联的事物和纪念品。

**适用场景：**
- 想建立积极情绪档案袋
- 收藏美好回忆
- 品味积极情绪
- 保持积极情绪活跃

**边界条件：**
- 档案袋包含照片、信件、名言、物品
- 对你而言带有深刻个人意义
- 10 个不断成长的档案袋

---

## 🎯 核心功能

### 1. 10 个档案袋

**10 种积极情绪档案袋：**
```
1. 喜悦档案袋
2. 感激档案袋
3. 宁静档案袋
4. 兴趣档案袋
5. 希望档案袋
6. 自豪档案袋
7. 逗趣档案袋
8. 激励档案袋
9. 敬佩档案袋
10. 爱档案袋
```

### 2. 档案袋内容

**可以包含：**
```
• 照片
• 信件
• 名言
• 对你而言带有深刻个人意义的物品
```

### 3. 建立方法

**喜悦档案袋思考问题：**
```
• 当你觉得安全、轻松和喜悦，是在什么时候？
• 当事情完全按照你的心意发展，是在什么时候？
• 当你感到脚步轻快、止不住地微笑，是在什么时候？
• 当你觉得好玩，想要一跃而入并参与其中，是在什么时候？
```

**感激档案袋思考问题：**
```
• 当你觉得感激或感谢，是在什么时候？
• 你最珍惜的礼物是什么？
• 当你因为自己是那么的幸运而心里美滋滋，是在什么时候？
• 当你对善意有做出回报的冲动，是在什么时候？
```

（其他 8 个档案袋各有 4 个思考问题）

### 4. 好处

**拥有 10 个档案袋的好处：**
```
• 有机会跳出跑步机的步调
• 保持积极情绪的活跃和精彩
• 随时可以回味美好
• 积累积极情绪资源
```

---

## ⚠️ 常见错误

**错误 1：认为档案袋是一次性的**
```
问题：
• "建好就不用管了"
• 不更新档案袋

解决：
✓ 10 个不断成长的档案袋
✓ 持续添加新的内容
✓ 保持积极情绪的活跃和精彩
```
