# 积极情绪档案袋建立指南

**配合 `positivity-archive` 技能使用**

---

## 10 个档案袋清单

### 1. 喜悦档案袋
**思考问题：**
- [ ] 当你觉得安全、轻松和喜悦，是在什么时候？
- [ ] 当事情完全按照你的心意发展，是在什么时候？
- [ ] 当你感到脚步轻快、止不住地微笑，是在什么时候？
- [ ] 当你觉得好玩，想要一跃而入并参与其中，是在什么时候？

**收藏内容：**
- [ ] 照片
- [ ] 信件
- [ ] 名言
- [ ] 物品

---

### 2. 感激档案袋
**思考问题：**
- [ ] 当你觉得感激或感谢，是在什么时候？
- [ ] 你最珍惜的礼物是什么？
- [ ] 当你因为自己是那么的幸运而心里美滋滋，是在什么时候？
- [ ] 当你对善意有做出回报的冲动，是在什么时候？

**收藏内容：**
- [ ] 照片
- [ ] 信件
- [ ] 名言
- [ ] 物品

---

### 3. 宁静档案袋
**思考问题：**
- [ ] 当你感到完全地平和与宁静，是在什么时候？
- [ ] 当你感觉自己的生活是如此舒心，是在什么时候？
- [ ] 当你的身体完全放松，全部的身体紧张都消失不见，是在什么时候？
- [ ] 当你休息并完全沉浸在其中，品味你感受到的美好，是在什么时候？

**收藏内容：**
- [ ] 照片
- [ ] 信件
- [ ] 名言
- [ ] 物品

---

### 4. 兴趣档案袋
**思考问题：**
- [ ] 当你对于在面前展开的奥秘或可能性深感兴趣，是在什么时候？
- [ ] 当你觉得很安全，但又被一些新的未知的东西所吸引，是在什么时候？
- [ ] 当你感到极其开放和活跃，就好像你自己内心的视野在不断扩展，是在什么时候？
- [ ] 当你感到有一种强烈的欲望去更多地探索和学习，是在什么时候？

**收藏内容：**
- [ ] 照片
- [ ] 信件
- [ ] 名言
- [ ] 物品

---

### 5. 希望档案袋
**思考问题：**
- [ ] 当你感到充满希望和乐观，受到一个美好结果的可能性的鼓舞，是在什么时候？
- [ ] 当你面临不确定性时，你对最坏的结果感到恐惧，却又在某种程度上仍然相信事情会往好的方向发展，是在什么时候？
- [ ] 当你真真切切地渴望某些更好的情况发生，是在什么时候？
- [ ] 当你发掘你的创造性去为一个更好的未来做出努力，是在什么时候？

**收藏内容：**

- [ ] 名言
- [ ] 物品

### 6. 自豪档案袋
**思考问题：**
- [ ] 当你最为自己感到自豪，对自己的能力充满信心和自我肯定，是在什么时候？
- [ ] 你在什么时候做了一些值得称道的事情？
- [ ] 是什么令你昂首挺胸？是什么让你想要与别人分享你的好消息？
- [ ] 是什么吸引你去梦想你可能在未来取得的成就？

**收藏内容：**

### 7. 逗趣档案袋

- [ ] 当你和其他人发现或激起了某种不可预见的些许幽默，是在什么时候？

- [ ] 当你产生与其他人分享愉悦的冲动，并且也许在这个过程中建立起友谊，是在什么时候？

### 10. 爱档案袋
**思考问题：**
- [ ] 当你最容易感受到在你和另一个人之间的爱意涌现，是在什么时候？
- [ ] 当你的某一种人际关系引发了某种或某几种其他形式的积极情绪，是在什么时候？
- [ ] 当你发现自己情不自禁地靠向心爱的人，肯定他的与众不同，是在什么时候？
- [ ] 当你享受挚爱的人陪伴的感觉，从而珍惜他们或是在他们的怀抱中笑容满溢，是在什么时候？

- [ ] 物品

*拥有 10 个不断成长的档案袋，
其好处就在于你有机会跳出跑步机的步调，
保持积极情绪的活跃和精彩。* 📁
