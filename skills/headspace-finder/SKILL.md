# 头脑空间发现器 🌌

**基于《十分钟冥想》（安迪·普迪科姆）**

---

## 📋 功能描述

帮助用户寻找头脑空间，感受深层的宁静感、满足感和富足感。

**适用场景：**
- 想感受深层宁静
- 压力大需要放松
- 想培养满足感
- 提升幸福感

---

## 🎯 核心功能

### 1. 头脑空间的本质

**定义：**
```
一种深层的宁静感
一种满足感
一种不可动摇的富足感
无关乎当时的情绪
```

### 2. 寻找方法

**步骤：**
```
1. 承认心灵在限定你的体验
2. 通过改变你看待世界的方式，改变你周围的世界
3. 不要努力创造心境
4. 坐看云卷云舒
```

---

## ⚠️ 常见错误

**错误 1：努力创造头脑空间**
```
问题：
• "我要努力获得宁静"

解决：
✓ 冥想不是需要努力去创造一种人为的心境
✓ 不要努力把阴云赶走
✓ 坐看云卷云舒
```
