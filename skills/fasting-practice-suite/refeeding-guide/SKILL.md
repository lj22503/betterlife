# 复食指南 🍽️

**基于《疗愈的饮食与断食》第 53 章**

---

## 📋 功能描述

指导用户正确结束断食，避免复食综合征，最大化断食效益。

**适用场景：**
- 结束间歇性断食
- 结束 24 小时 + 断食
- 结束长时间断食
- 避免断食后暴食

**边界条件：**
- 断食时间越长，复食越重要
- 长时间断食需专业指导
- 复食不当可能抵消断食效益

---

## 🎯 核心功能

### 1. 复食原则

**核心原则：**
```
✓ 从少量开始
✓ 从简单开始
✓ 从不刺激胰岛素开始
✓ 慢慢增加复杂度
✓ 细嚼慢咽
✓ 觉察饱足感
```

**为什么重要：**
- 断食后身体对食物敏感
- 胰岛素反应可能过强
- 容易吃过量
- 可能引起不适

### 2. 复食时间线

**短断食（16-18 小时）：**
```
• 可直接正常进食
• 仍建议从蛋白质/脂肪开始
• 避免第一口就吃精制碳水
```

**中断食（24-36 小时）：**
```
断食结束前 1 小时：
• 1-2 茶匙脂肪（MCT 油、橄榄油、牛油果）
• 或咖啡/茶加印度酥油

第一餐：
• 少量蛋白质（蛋、鱼）
• 大量蔬菜
• 好脂肪
• 避免淀粉类

第二餐（4-6 小时后）：
• 正常饮食
• 仍避免精制碳水
```

**长断食（48 小时+）：**
```
第 1 天：
• 骨汤、蔬菜汤
• 少量发酵食品
• 好脂肪（橄榄油、牛油果）
• 避免蛋白质过量

第 2 天：
• 加入少量蛋白质（蛋、鱼）
• 继续大量蔬菜
• 可加入少量根茎类

第 3 天：
• 恢复正常饮食
• 仍避免精制糖和淀粉
• 观察身体反应
```

### 3. 复食食物选择

**✅ 推荐食物：**
```
第一口（断食结束前 1 小时）：
• MCT 油 1 茶匙
• 橄榄油 1 茶匙
• 牛油果 1-2 片
• 咖啡/茶 + 印度酥油

第一餐：
• 骨汤/蔬菜汤
• 水煮蛋
• 蒸鱼
• 大量蔬菜（橄榄油拌）
• 发酵食品（酸菜、泡菜）

后续餐：
• 草饲肉
• 根茎类（地瓜、芋头）
• 继续大量蔬菜
```

**❌ 避免食物：**
```
× 精制糖（甜点、含糖饮料）
× 精制淀粉（白米饭、面包、面条）
× 大量蛋白质（可能刺激胰岛素）
× 加工食品
× 酒精
× 大量果糖（果汁、大量水果）
```

### 4. 复食综合征预防

**什么是复食综合征：**
- 长时间断食后快速进食导致
- 电解质急剧变化
- 可能危及生命

**症状：**
```
• 虚弱、疲劳
• 心跳异常
• 呼吸困难
• 水肿
• 意识模糊
• 抽搐
```

**预防：**
```
✓ 长时间断食（5 天+）需医疗监督
✓ 缓慢复食（至少用断食时间的一半）
✓ 补充电解质（钠、钾、镁、磷）
✓ 从流质开始
✓ 避免高碳水
```

### 5. 复食后饮食维持

**维持断食效益：**
```
✓ 继续间歇性断食（16-8）
✓ 低糖高脂饮食
✓ 大量蔬菜
✓ 好脂肪
✓ 适量蛋白质
✓ 避免精制糖和淀粉
```

**常见错误：**
```
× 断食后报复性进食
× 第一餐就吃大量碳水
× 忽略饱足信号
× 立即恢复旧饮食习惯
```

## 🧪 使用示例

```
@ant 我刚完成 24 小时断食，如何复食？

信息：
- 断食时长：24 小时
- 断食类型：中断食
- 之前有经验：是
- 健康状况：无
```

**预期输出：**
- 复食方案：断食前 1 小时少量脂肪
- 第一餐：蛋白质 + 蔬菜 + 好脂肪
- 避免食物：精制碳水
- 维持计划：继续 16-8 断食

---

## 📚 核心理念

**关键洞察：**
1. 复食和断食同样重要
2. 从脂肪开始，不刺激胰岛素
3. 断食越长，复食越慢
4. 避免报复性进食
5. 维持新饮食习惯

**复食公式：**
```
好复食 = 少量 + 简单 + 好脂肪 + 慢
```
