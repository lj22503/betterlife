# 间歇性断食指南 ⏰

**基于《疗愈的饮食与断食》第 46-48 章**

---

## 📋 功能描述

指导用户安全有效地进行间歇性断食，从 16-8 开始逐步深入。

**适用场景：**
- 减重需求
- 改善代谢健康
- 简化生活
- 肠道修复
- 自噬作用激活

**边界条件：**
- 孕妇/哺乳期不建议
- 体重不足者谨慎
- 糖尿病患者需医师指导
- 生理期前避开断食

---

## 🎯 核心功能

### 1. 断食方案选择

**入门级：16-8 断食**
```
• 断食 16 小时，进食窗口 8 小时
• 例：中午 12 点 - 晚上 8 点进食
• 最容易执行，适合新手
```

**进阶级：18-6 断食**
```
• 断食 18 小时，进食窗口 6 小时
• 例：下午 2 点 - 晚上 8 点进食
• 自噬作用更强
```

**高级：23-1 断食（OMAD）**
```
• 断食 23 小时，进食窗口 1 小时
• 一天一餐
• 适合有经验者
```

### 2. 断食期间可以喝什么

**✅ 可以：**
```
• 水（天然好水）
• 气泡水
• 草本茶（无糖）
• 黑咖啡（无糖无奶）
• 稀释苹果醋（1-2 茶匙/杯水）
```

**❌ 不可以：**
```
• 含糖饮料
• 果汁
• 牛奶/奶精
• 代糖饮料（可能刺激胰岛素）
• 酒精
```

### 3. 饥饿感管理

**饥饿感曲线：**
```
• 饥饿感是波浪式的
• 持续约 1 小时会自然消退
• 多喝水帮助度过
• 分散注意力
```

**应对策略：**
- 喝气泡水增加饱足感
- 喝草本茶
- 散步或轻度运动
- 工作/活动分散注意
- 提醒自己：这是身体在清理

### 4. 女士断食注意事项

**生理周期配合：**
```
✓ 月经后 2 周：断食黄金期
× 月经前 1 周：避开断食（黄体素下降，血糖易波动）
× 月经期间：不建议长时间断食
```

**特殊人群：**
```
× 体重不足（BMI<18.5）
× 怀孕/哺乳期
× 有饮食失调史
× 压力过大时期
```

### 5. 断食 + 饮食配合

**进食窗口饮食原则：**
```
✓ 大量蔬菜（生吃最佳）
✓ 好脂肪（橄榄油、椰子油、牛油果）
✓ 适量蛋白质（鱼、蛋、草饲肉）
✓ 低糖水果（莓果类）
× 精制糖、精制淀粉
× 种籽油
× 加工食品
```

**晚餐建议：**
- 早点吃（睡前 3-4 小时）
- 热汤 + 蔬菜
- 减轻肠胃负担
- 不影响睡眠

## 📚 核心理念

**关键洞察：**
1. 16-8 最容易执行且有效
2. 饥饿感会自然消退
3. 女士需配合生理周期
4. 断食 + 好饮食效果最佳
5. 断食是给身体休息时间

**断食好处：**
- 激活自噬作用
- 改善胰岛素敏感性
- 简化生活
- 给肠道休息
- 降低发炎
