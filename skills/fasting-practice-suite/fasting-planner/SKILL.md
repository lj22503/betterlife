# 断食计划器 📅

**基于《疗愈的饮食与断食》第 46、51 章**

---

## 📋 功能描述

根据用户情况设计个性化断食计划，从日常到深度清理。

**适用场景：**
- 新手开始断食
- 优化现有断食计划
- 安排深度清理断食
- 配合生活节奏

**边界条件：**
- 需考虑个人健康状况
- 女士需配合生理周期
- 循序渐进为原则

---

## 🎯 核心功能

### 1. 断食方案库

**日常维护方案：**
```
16-8 断食（入门）：
• 适合：新手、日常维护
• 时间：例如 12:00-20:00 进食
• 频率：每天

18-6 断食（进阶）：
• 适合：有经验者、更强自噬
• 时间：例如 14:00-20:00 进食
• 频率：每周 3-5 次

23-1 断食（OMAD）：
• 适合：高级、简化生活
• 时间：1 小时内吃完一天食物
• 频率：每周 1-3 次
```

**深度清理方案：**
```
24 小时断食：
• 适合：月度清理
• 频率：每月 1-2 次
• 例：晚餐后到次日晚餐

36 小时断食：
• 适合：季度清理
• 频率：每季 1 次
• 例：第 1 天晚餐→第 3 天早餐

48 小时断食：
• 适合：深度清理
• 频率：每季 1 次或半年 1 次
• 需准备和复食计划

72 小时 + 断食：
• 适合：年度深度清理
• 频率：每年 1-2 次
• 需专业指导
```

### 2. 新手入门计划

**第 1-2 周：适应期**
```
• 12-12 断食（7:00-19:00 进食）
• 戒除睡前零食
• 稳定血糖
```

**第 3-4 周：进阶期**
```
• 14-10 断食（9:00-19:00 进食）
• 跳过早餐或晚餐
• 适应饥饿感
```

**第 5-8 周：稳定期**
```
• 16-8 断食（12:00-20:00 进食）
• 形成习惯
• 感受效益
```

**第 9 周+：优化期**
```
• 尝试 18-6 断食
• 或加入月度 24 小时断食
• 找到适合自己的节奏
```

### 3. 女士生理周期配合

**月经周期 4 阶段：**
```
卵泡期（月经后 1-14 天）：
✓ 断食黄金期
✓ 可进行 18-6 或 24 小时断食
✓ 胰岛素敏感性好

排卵期（第 14-16 天）：
✓ 可继续断食
✓ 注意能量水平

黄体期（第 17-28 天）：
× 避免长时间断食
× 特别是月经前 1 周
× 改为 14-10 或正常饮食

月经期：
× 不建议断食
× 注重营养
× 休息为主
```

### 4. 生活节奏整合

**上班族方案：**
```
工作日：
• 跳过早餐（省时间）
• 12:00 午餐、19:00 晚餐
• 16-8 断食

周末：
• 可尝试 18-6
• 或家庭聚餐时灵活调整
```

**轮班工作者：**
```
• 根据工作时间调整进食窗口
• 保持 16 小时断食
• 例如：夜班 20:00-4:00，进食窗口 12:00-20:00
```

**社交活跃者：**
```
• 工作日严格 16-8
• 周末灵活调整
• 重要聚餐时暂停断食
• 次日恢复
```

### 5. 断食日历

**月度计划示例：**
```
每日：16-8 断食
第 7 天：18-6 断食（加深）
第 14 天：24 小时断食（月度清理）
第 21 天：18-6 断食
第 28 天：正常饮食（女士月经期）
```

**季度计划示例：**
```
每月：1-2 次 24 小时断食
每季：1 次 36-48 小时断食
每日：16-8 维持
```

## 📚 核心理念

**关键洞察：**
1. 断食需循序渐进
2. 女士配合生理周期
3. 整合生活节奏才可持续
4. 日常 + 深度清理结合
5. 灵活调整，不僵化

**成功关键：**
- 从短时间开始
- 形成习惯再进阶
- 倾听身体
- 长期可持续
