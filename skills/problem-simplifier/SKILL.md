# 问题简化器 ✂️

**基于《关键冲突》（科里·帕特森等）**

---

## 📋 功能描述

帮助用户简化问题，消除能力障碍，化人际关系危机为合作共赢。

**适用场景：**
- 简化问题，消除能力障碍
- 改善人际关系
- 提升沟通能力

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 明确选择（目标和意义）
2. 梳理头绪（保持理智）
3. 问题描述（准备对话）
4. 制造动机（帮助行动）
5. 简化问题（轻松履约）
6. 灵活关注（应对突发）
7. 制订计划（做出承诺）
```

---

## ⚠️ 常见错误

**错误 1：沉默或暴力**
```
问题：
• 保持沉默，任其变得愈加猖狂
• 冒失争执，带来新的问题

解决：
✓ 有礼有节地和对方进行对话
✓ 在诚实面对问题的同时保持对对方的充分尊重
✓ 关键冲突是提升人际关系的最大机遇
```

**错误 2：想当然地虚构情节**
```
问题：
• 根据一知半解的信息自以为是
• 带着强烈的负面情绪面对冲突

解决：
✓ 在开口之前调整自己的思维方式和感受
✓ 了解事情的确切经过
✓ 假设自己是事件中的当事人
```
