# 智慧乐观平衡器 ⚖️

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户平衡乐观与智慧，在合适的情况下使用乐观技术，避免盲目乐观。

**适用场景：**
- 想避免盲目乐观
- 不知道何时该乐观
- 想理性使用乐观技术

---

## 🎯 核心功能

### 1. 何时使用乐观

**应该用乐观技术的情况：**
```
✓ 你想成功（升职、推销、完成报告、赢球等）
✓ 你关心你的感觉（想摆脱抑郁、提高士气等）
✓ 事情迟滞无进展，而你的健康已有问题时
✓ 你想领导别人、激励别人、赢得选票
```

### 2. 何时不用乐观

**不应该用乐观技术的情况：**
```
✗ 你要为一件有危险且不确定的事制定计划
✗ 你要为一个前途黯淡的人出谋划策
✗ 你想对陷入困境的人表示同情（先建立信赖与共情）
```

### 3. 基本原则

**使用乐观技术的基本原则：**
```
先问：在某一个特定情况下失败的代价是什么？
如果失败的代价很高，那么就不应该乐观。
```

---

## ⚠️ 常见错误

**错误 1：盲目乐观**
```
问题：
• 所有情况都用乐观
• 不考虑失败的代价

解决：
✓ 先问失败的代价
✓ 代价高时不要乐观
✓ 乐观只有跟智慧结合才有用
```

## 📚 核心理念

**关键洞察：**
1. 乐观不是所有情况都用
2. 先问失败的代价
3. 乐观需要与智慧结合

---

*乐观只有跟智慧结合在一起时才有用，
它单独存在时没有任何意义。
乐观是帮助一个人达到他所设定的目标的工具，
乐观是否有意义取决于目标的选择。* ⚖️
