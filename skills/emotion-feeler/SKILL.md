# 情绪感受器 💗

**基于《当下的力量》（埃克哈特·托利）**

---

## 📋 功能描述

帮助用户感受情绪，与情绪联结，避免在生理层面体验情绪。

**适用场景：**
- 切断与情绪的联系
- 无法感受情绪
- 想与情绪联结
- 情绪以生理问题出现

**边界条件：**
- 情绪比思维更真实
- 感受而非思考情绪
- 接受情绪的存在

---

## 🎯 核心功能

### 1. 情绪 vs 思维

**关键洞察：**
```
如果两者之间有明显的分歧
那么思维永远是说谎的一方
情绪则始终是真实的
```

### 2. 切断情绪联系的后果

**后果：**
```
如果你不能感受到你的情绪
或是切断了与情绪的联系
那么你最终会在纯生理这一层面体验到它们
它们会以生理问题或疾病的形式出现
```

### 3. 感受情绪的方法

**步骤：**
```
1. 将注意力集中在你内心的感受上
2. 别去想它
3. 别让你的感受变成大脑和思维
4. 不要去判断或分析它
5. 保持临在，继续观察你的内在
```

---

## ⚠️ 常见错误

**错误 1：用思维分析情绪**
```
问题：
• "我为什么会有这种情绪"
• 分析情绪的原因

解决：
✓ 别去想它
✓ 别让你的感受变成大脑和思维
✓ 就只是感受
```
