# 自我接纳练习器 🌱

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户接纳自我，区分"能改变的"和"不能改变的"，减少自我否定。

**适用场景：**
- 自我否定、自我嫌恶
- 过度在意自己的缺点
- 无法接纳不完美的自己
- 与他人比较感到自卑

**边界条件：**
- 不鼓励放弃改变
- 接纳≠放弃努力
- 区分"不能改变"和"不想改变"

---

## 🎯 核心功能

### 1. 自我接纳三步法

**步骤 1：区分能改变与不能改变**
```
能改变的：
• 我的行为和习惯
• 我的态度和选择
• 我的努力和行动

不能改变的：
• 我的出身和过去
• 我的天赋和外貌
• 别人的想法和评价
```

**步骤 2：接纳不能改变的**
```
✓ 承认现实
✓ 不为此自责
✓ 把精力放在能改变的
```

**步骤 3：改变能改变的**
```
✓ 设定可行目标
✓ 采取行动
✓ 接纳过程中的不完美
```

### 2. 自我接纳检查清单

**我在接纳自己吗？**
- [ ] 我是否接纳自己的出身和过去？
- [ ] 我是否接纳自己的不完美？
- [ ] 我是否不过度在意别人的评价？

**我在努力改变吗？**
- [ ] 我是否在改变能改变的？
- [ ] 我是否采取行动？
- [ ] 我是否接纳改变过程中的挫折？

---

## ⚠️ 常见错误

**错误 1：把接纳当成放弃**
```
问题：
• "反正我就是这样了"
• "接纳就是什么都不做"

解决：
✓ 接纳是改变的前提
✓ 接纳不能改变的，改变能改变的
✓ 接纳≠放弃努力
```

**错误 2：过度关注不能改变的**
```
问题：
• 纠结于出身、外貌
• 为过去的事情自责

解决：
✓ 关注能改变的
✓ 过去不能决定现在
✓ 重要的是"此时此刻"
```

**输入：**


```
🌱 自我接纳分析

【不能改变的】（接纳）

• 别人的评价

【能改变的】（行动）
• 你的行为和习惯
• 你的态度和选择
• 你的努力和成长

【建议行动】
1. 列出 3 个你能改变的方面
2. 为每个方面设定小目标
3. 每天进步一点点
4. 接纳过程中的不完美

【关键洞察】
不是肯定自我，而是接纳自我。
接纳不完美的自己，然后改变能改变的。
```

## 📚 核心理念

**关键洞察：**
1. 不是肯定自我，而是接纳自我
2. 接纳不能改变的，改变能改变的
3. 过去的经历不能决定现在
4. 重要的是"此时此刻"的你
5. 自我接纳是改变的前提

**健康公式：**
```
自我接纳 = 接纳不能改变的 + 改变能改变的
```
