# 自我认知构建器 🪞

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户认识自己，探索内心世界，这是改变的唯一前提。

**适用场景：**
- 想了解自己
- 迷茫、不知道自己要什么
- 想改变但不知道从何开始
- 心理咨询自我探索

**边界条件：**
- 改变的唯一前提是认识你自己
- 能帮你的人只有你自己
- 真实才是了解内心世界的入口

---

## 🎯 核心功能

### 1. 自我认知三层次

**层次 1：认识情绪**
```
• 我此刻的感受是什么？
• 为什么会有这个感受？
• 情绪背后的需求是什么？
```

**层次 2：认识模式**
```
• 我的行为习惯是什么？
• 这些习惯从哪里来？
• 哪些模式在限制我？
```

**层次 3：认识选择**
```
• 我选择了什么样的生活？
• 我可以选择不同吗？
• 什么对我真正重要？
```

### 2. 自我探索问题

**你是谁？**
- [ ] 你的价值观是什么？
- [ ] 你的优势是什么？
- [ ] 你的局限是什么？

**你从哪里来？**
- [ ] 你的童年经历如何影响你？
- [ ] 你从父母那里学到了什么？
- [ ] 哪些影响是积极的？哪些是消极的？

**你要到哪里去？**
- [ ] 你想要什么样的生活？
- [ ] 什么对你真正重要？
- [ ] 你愿意为什么负责？

## ⚠️ 常见错误

**错误 1：期待别人给答案**
```
问题：
• "告诉我该怎么做"
• 期待别人解决自己的问题

解决：
✓ 改变的唯一前提是认识你自己
✓ 能帮你的人只有你自己
✓ 真实才是了解内心世界的入口
```

**错误 2：逃避真实感受**
```
问题：
• 说"挺好的"隐瞒真实情绪
• 不敢面对内心的痛苦

解决：
✓ 真实才是了解内心世界的入口
✓ 允许自己感受所有情绪
✓ 认识自己是改变的第一步
```
