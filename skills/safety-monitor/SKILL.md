# 安全监控器 🛡️

**基于《关键对话：如何高效能沟通》**

---

## 📋 功能描述

帮助用户监控对话氛围是否安全，把关键对话转变成行动和结果。

**适用场景：**
- 监控对话氛围是否安全
- 改善沟通效果
- 提升人际关系

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 从心开始（确定真正目的）
2. 注意观察（监控安全氛围）
3. 保证安全（让对方畅所欲言）
4. 陈述观点（循循善诱）
5. 了解动机（帮助对方走出沉默或暴力）
6. 掌控自我（保持韧性）
7. 开始行动（转变成行动和结果）
```

---

## ⚠️ 常见错误

**错误 1：傻瓜式选择**
```
问题：
• 在"说真话"和"维持友谊"之间二选一
• 以为必须权衡取舍

解决：
✓ 拒绝傻瓜式选择
✓ 寻找既能百分之百坦诚又能百分之百尊重对方的方法
✓ 展开对话：双方自由地交流观点
```

**错误 2：三种错误想法**
```
问题：
• 受害者想法："这可不是我的错！"
• 大反派想法："都是你的错！"
• 无助者想法："这事我也没办法！"

解决：
✓ 把受害者变成参与者
✓ 把大反派变成正常人
✓ 把无助者变成行动者
```

## 📚 核心理念

**关键洞察：**
1. 监控对话氛围是否安全
2. 关键对话有三个特征：观点不同/风险巨大/情绪强烈
3. 拒绝傻瓜式选择
4. 营造安全氛围，让双方都愿意向共享观点库中添加信息

**健康公式：**
```
关键对话 = 从心开始 + 注意观察 + 保证安全 + 陈述观点 + 了解动机 + 掌控自我 + 开始行动
```

---

*关键对话指的是两人或多人之间的一种讨论，
这种讨论具有三个特点：①观点不同；②风险巨大；③情绪强烈。
决定对话成败的关键因素是从出现问题到当事人找到方法
实实在在、有礼有节地解决问题的间隔时间。* 🛡️
