# 勇气增强器 💪

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户获得勇气，特别是"被讨厌的勇气"，活出自由人生。

**适用场景：**
- 害怕被讨厌
- 不敢表达真实想法
- 为他人而活

---

## 🎯 核心功能

### 1. 勇气来源

**勇气 = 自我接纳 + 他者信赖 + 他者贡献**

```
✓ 接纳不完美的自己
✓ 信任别人（即使可能被背叛）
✓ 感受对别人的贡献
```

### 2. 被讨厌的勇气

```
✓ 不畏惧被人讨厌
✓ 不随波逐流
✓ 激流勇进
```

---

**输入：**
```
@ant 我不敢拒绝别人，怕被讨厌
```

**输出：**
```

【被讨厌的勇气】

✓ 不随波逐流而激流勇进

【建议行动】
1. 诚实表达你的想法
2. 接纳别人可能讨厌你
3. 记住：这是自由的生活方式

【关键洞察】
不畏惧被人讨厌而勇往直前，
不随波逐流而激流勇进，

```

---

## 📚 核心理念

**关键洞察：**
1. 被讨厌是自由的代价
2. 勇气来自自我接纳、他者信赖、他者贡献
3. 不畏惧被讨厌才是自由

---
