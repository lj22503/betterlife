# 勇气增强器 💪

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户获得勇气，特别是"被讨厌的勇气"，活出自由人生。

**适用场景：**
- 害怕被讨厌
- 不敢表达真实想法
- 为他人而活

**边界条件：**
- 不鼓励故意惹人讨厌
- 勇气是为了自由而非伤害
- 结合课题分离使用

---

## 🎯 核心功能

### 1. 勇气来源

**勇气 = 自我接纳 + 他者信赖 + 他者贡献**

```
✓ 接纳不完美的自己
✓ 信任别人（即使可能被背叛）
✓ 感受对别人的贡献
```

### 2. 被讨厌的勇气

```
✓ 不畏惧被人讨厌
✓ 不随波逐流
✓ 激流勇进
```

---

## ⚠️ 常见错误

**错误 1：把被讨厌当目的**
```
问题：
• "我要故意惹人讨厌"
• 把被讨厌当成勇敢

解决：
✓ 被讨厌是自由的代价，不是目的
✓ 为了活出自由，不怕被讨厌
✓ 不是故意惹人讨厌
```

**错误 2：没有准备的勇气**
```
问题：
• 没有自我接纳就盲目行动
• 没有他者信赖就孤军奋战

解决：
✓ 先自我接纳
✓ 建立他者信赖
✓ 感受他者贡献
```
