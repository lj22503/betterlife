# 霍兰德职业匹配器 🎯

**基于《金树人：生涯咨询与辅导》核心理念**

---

## 📋 功能描述

基于霍兰德类型论，帮助用户进行兴趣类型与职业类型的匹配，找到适合的职业方向。

**适用场景：**
- 职业选择
- 专业选择
- 职业转型
- 自我认知

---

## 🎯 核心功能

### 1. 霍兰德六型评估

**RIASEC 六型：**

**R 现实型 (Realistic)**
- 特点：动手操作、机械、户外
- 适合职业：工程师、技师、农民、运动员

**I 研究型 (Investigative)**
- 特点：思考、分析、研究
- 适合职业：科学家、研究员、程序员、分析师

**A 艺术型 (Artistic)**
- 特点：创造、表达、美感
- 适合职业：艺术家、作家、设计师、音乐家

**S 社会型 (Social)**
- 特点：助人、教导、服务
- 适合职业：教师、咨询师、护士、社工

**E 企业型 (Enterprising)**
- 特点：领导、说服、管理
- 适合职业：企业家、销售、管理者、政治家

**C 常规型 (Conventional)**
- 特点：秩序、精确、系统
- 适合职业：会计、行政、图书管理员、数据分析师

### 2. 兴趣图与工作世界图结合

**兴趣图：** 个人兴趣类型分布
**工作世界图：** 职业类型分布
**匹配度：** 兴趣与职业的匹配程度

### 3. 职业自我探索

**三工具：**
1. **生涯憧憬**：你向往什么样的职业生涯？
2. **我最愉快的经验**：什么活动让你最投入？
3. **生涯度假计划**：如果有无限时间，你想做什么？

### 4. 匹配建议

**高匹配度：** 兴趣类型与职业类型一致
**中等匹配度：** 相邻类型（如 R 与 I）
**低匹配度：** 相对类型（如 R 与 S）
