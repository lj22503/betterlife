## 10 个省钱妙招

### 1. 不去购物
- [ ] 只买需要的，不被广告打动
- [ ] 取消购物 APP 推送
- [ ] 卸载购物软件

### 2. 量入为出
- [ ] 等有钱再买，避免利息
- [ ] 设置购物冷静期（7 天）
- [ ] 使用现金支付

### 3. 爱惜所有
- [ ] 缝补衣物
- [ ] 维修而非更换
- [ ] 定期保养物品

### 4. 用坏为止
- [ ] 不追求新品
- [ ] 用到不能再用
- [ ] 物尽其用

### 5. 自己动手
- [ ] DIY 而非购买服务
- [ ] 自己做饭
- [ ] 自己修理

### 6. 预见需求
- [ ] 提前计划，避免冲动消费
- [ ] 制定购物清单
- [ ] 批量采购

### 7. 全面研究
- [ ] 比较价值、质量、耐用性
- [ ] 查看评测
- [ ] 比价

### 8. 降低花销
- [ ] 寻找更便宜的替代方案
- [ ] 使用优惠券
- [ ] 团购

### 9. 换个方式满足需求
- [ ] 创意而非消费
- [ ] 体验而非物质
- [ ] 共享而非独占

### 10. 遵循本计划
- [ ] 持续追踪
- [ ] 定期 review
- [ ] 不断优化
