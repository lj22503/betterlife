# MBTI 人格类型测评模板

## 四维度评估

**说明：** 每个维度选择更符合你的选项

---

## 维度 1：能量来源 (E/I)

**外倾 (E) vs 内倾 (I)**

### 外倾 (E) 特征
- [ ] 从与人交往中获得能量
- [ ] 喜欢成为注意的焦点
- [ ] 行动前先说出来
- [ ] 朋友众多，交游广泛
- [ ] 喜欢热闹和团体活动

### 内倾 (I) 特征
- [ ] 从独处中获得能量
- [ ] 喜欢安静和隐私
- [ ] 思考后再说
- [ ] 朋友较少但关系深厚
- [ ] 喜欢深度交流

**你的偏好：** □ E  □ I

---

## 维度 2：信息获取 (S/N)

**感觉 (S) vs 直觉 (N)**

### 感觉 (S) 特征
- [ ] 关注具体事实和细节
- [ ] 相信经验和实际
- [ ] 喜欢按部就班
- [ ] 注重现实和当下
- [ ] 擅长记忆事实

### 直觉 (N) 特征
- [ ] 关注可能性和未来
- [ ] 相信灵感和想象
- [ ] 喜欢创新和变化
- [ ] 注重愿景和远景
- [ ] 擅长联想和模式识别

**你的偏好：** □ S  □ N

---

## 维度 3：决策方式 (T/F)

**思考 (T) vs 情感 (F)**

### 思考 (T) 特征
- [ ] 基于逻辑和原则做决定
- [ ] 注重公平和客观
- [ ] 喜欢分析和批判
- [ ] 对事不对人
- [ ] 重视真理胜过和谐

### 情感 (F) 特征
- [ ] 基于价值观和人情做决定
- [ ] 注重和谐和关系
- [ ] 喜欢欣赏和赞美
- [ ] 对人不对事
- [ ] 重视和谐胜过真理

**你的偏好：** □ T  □ F

---

## 维度 4：生活方式 (J/P)

**判断 (J) vs 感知 (P)**

### 判断 (J) 特征
- [ ] 喜欢计划和结构
- [ ] 做事井井有条
- [ ] 喜欢做决定
- [ ] 按时完成任务
- [ ] 喜欢控制和确定性

### 感知 (P) 特征
- [ ] 喜欢灵活和自由
- [ ] 做事随遇而安
- [ ] 喜欢保持选择开放
- [ ] 最后一刻完成任务
- [ ] 喜欢变化和可能性

**你的偏好：** □ J  □ P
