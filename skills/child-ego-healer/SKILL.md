# 儿童自我疗愈器 🧸

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户疗愈童年创伤，打破依赖关系，成为独立自主的人。

**适用场景：**
- 童年创伤影响现在
- 依赖型人格
- 讨好型人格
- 想打破童年模式

---

## 🎯 核心功能

### 1. 童年影响识别

**你的童年模式：**
- [ ] 顺从父母期望
- [ ] 取悦别人获得爱
- [ ] 压抑负面情绪
- [ ] 依赖别人肯定

### 2. 疗愈步骤

**步骤 1：认识童年影响**
```
父母把观念和行为传递给后代
```

**步骤 2：打破依赖关系**
```
成长的本质是减少并打破依赖关系
```

**步骤 3：成为独立自主的人**
```
很少有人能完全达成，但要努力
```

---

## ⚠️ 常见错误

**错误 1：责怪父母**
```
问题：
• "都是父母的错"
• 责怪原生家庭

解决：
✓ 父母也是人，会传递观念
✓ 关键是如何应对和防御
✓ 为自己负责
```

1. 父母会传递观念和行为
2. 成长是打破依赖关系
3. 为自己负责
