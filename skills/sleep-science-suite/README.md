# 好睡科学套件 😴

**基于《绝佳时间》《睡觉》《好睡》**

---

## 📖 简介

好睡科学套件包含 5 个技能，帮助用户理解睡眠科学，优化睡眠环境，转变失眠观念，获得优质睡眠。

**核心理念：**
- 🌙 睡眠由 4-5 个 90 分钟周期组成
- ⏰ 光照是生物钟最强同步信号
- 🫁 鼻呼吸促进放松和深睡
- 🧘 接纳失眠比对抗更有效
- 🛏️ 完全黑暗是睡眠环境关键

## 🔑 关键概念

### 睡眠周期
- 4-5 个周期/晚
- 每周期 70-90 分钟
- 深睡清理脑内废物
- 从快速眼动睡眠中醒来最佳

### 生物钟
- 光照是最强同步信号
- 清晨光照提前生物钟
- 傍晚光照延迟生物钟
- 运动和进食也影响

### 鼻呼吸
- 激活副交感神经
- 舌抵上颚自然闭嘴
- 胶带法简单有效
- 呼吸暂停需医疗干预

### 失眠观念
- 失眠本身没有绝对重要性
- 接纳比对抗更有效
- 没有统一睡眠标准
- 自己感觉睡饱是关键

### 睡眠环境
- 卧室完全黑暗
- 白噪声压制干扰
- 凉爽促进睡眠
- 舒适寝具支撑
