# 睡眠呼吸优化 🫁

**基于《睡觉》《好睡》**

---

## 📋 功能描述

指导用户优化睡眠呼吸，预防呼吸暂停，改善血氧含量和睡眠质量。

**适用场景：**
- 打鼾
- 口呼吸
- 睡眠呼吸暂停
- 醒来口干/喉咙痛
- 白天疲劳

**边界条件：**
- 不替代医疗诊断
- 严重呼吸暂停需就医
- 可能需要专业设备

---

## 🎯 核心功能

### 1. 呼吸方式识别

**鼻呼吸 vs 口呼吸：**
```
鼻呼吸（健康）：
✓ 过滤空气（鼻毛、黏液）
✓ 加温加湿
✓ 产生一氧化氮（扩张血管）
✓ 激活副交感神经
✓ 呼吸更深更慢
✓ 促进放松

口呼吸（不健康）：
× 无过滤功能
× 空气干燥寒冷
× 激活交感神经（压力反应）
× 呼吸浅快
× 导致打鼾
× 影响睡眠质量
```

**口呼吸信号：**
```
• 醒来口干/喉咙痛
• 打鼾
• 枕头有口水痕迹
• 白天疲劳
• 注意力不集中
• 儿童：面部发育异常、注意力问题
```

### 2. 舌抵上颚练习

**正确姿势：**
```
• 舌尖轻抵上颚（门牙后方）
• 舌面贴紧上颚
• 嘴唇闭合
• 牙齿轻合或微张
• 用鼻子呼吸
```

**练习方法：**
```
随时练习：
✓ 白天随时提醒
✓ 设置手机提醒
✓ 贴便签在电脑/镜子

睡前练习：
✓ 躺下后检查舌位
✓ 确认嘴唇闭合
✓ 确认鼻呼吸

习惯养成：
✓ 持续 2-3 周
✓ 逐渐自动化
✓ 睡觉时自然保持
```

### 3. 口呼吸矫正

**胶带法（简单有效）：**
```
方法：
1. 选择医用纸胶带
2. 剪一小段（约 2-3cm）
3. 垂直贴在嘴唇中央
4. 不要完全封死，留缝隙

原理：
• 提醒闭嘴
• 强制鼻呼吸
• 安全（可随时揭开）

注意：
✓ 从短时间开始
✓ 适应后整晚使用
✓ 鼻塞时不使用
× 饮酒后不使用
× 感冒时不使用
```

**其他方法：**
```
• 口呼吸矫正器
• 特殊设计的枕头
• 呼吸训练器
```

### 4. 呼吸暂停识别

**睡眠呼吸暂停信号：**
```
• 响亮打鼾（伴侣告知）
• 睡眠中呼吸暂停（伴侣观察）
• 睡眠中窒息/喘气
• 白天极度困倦
• 早晨头痛
• 注意力不集中
• 情绪多变/易怒
• 夜间频繁醒来
• 夜尿增多
```

**风险评估：**
```
高风险人群：
• 超重/肥胖
• 男性
• 年龄>40 岁
• 颈部粗大
• 下颌后缩
• 家族史
• 饮酒
• 吸烟
```

**就医指征：**
```
✓ 伴侣观察到呼吸暂停
✓ 白天极度困倦（影响工作/驾驶）
✓ 打鼾响亮且频繁
✓ 夜间窒息感
✓ 高血压难以控制
```

### 5. 呼吸练习

**深呼吸练习：**
```
时间：随时，睡前最重要

方法：
1. 闭嘴，用鼻孔呼吸
2. 横膈膜呼吸（丹田充气）
3. 吸气至少 3 秒（可到 5 秒+）
4. 吐气至少 3 秒（可到 5 秒+）
5. 吐气顺序：肚子先缩→胸腔塌下
6. 慢，不要求快

效果：
✓ 激活副交感神经
✓ 降低压力
✓ 促进放松
✓ 准备入睡
```

**规律呼吸（6 次/分钟）：**
```
方法：
• 吸气 5 秒
• 吐气 5 秒
• 每分钟 6 次呼吸
• 持续 5-10 分钟

效果：
✓ 守住根本波动
✓ 达到谐振状态
✓ 深度放松
```

**鼻孔交替呼吸法：**
```
方法：
1. 右手拇指按住右鼻孔
2. 左鼻孔吸气，默数至 4
3. 按住左鼻孔，闭气，默数至 16
4. 放开右鼻孔，呼气，默数至 8
5. 右鼻孔吸气，默数至 4
6. 闭气，默数至 16
7. 放开左鼻孔，呼气，默数至 8
8. 为一个循环，重复 3-5 次

效果：
✓ 改善鼻塞
✓ 平复情绪
✓ 活化左右脑
✓ 改善睡眠质量
```

**四短一长净化呼吸：**
```
方法：
• 四组短呼吸
• 一组长呼吸
• 循环进行
• 吸气缓，吐气一次全部吐出

效果：
✓ 净化呼吸系统
✓ 提升血氧
✓ 放松身心
```

### 6. 血氧优化

**影响血氧的因素：**
```
降低血氧：
× 口呼吸
× 打鼾/呼吸暂停
× 缺乏运动
× 缺乏晒太阳
× 肥胖

提升血氧：
✓ 鼻呼吸
✓ 深呼吸练习
✓ 适度运动
✓ 晒太阳
✓ 泡澡/温泉
✓ 轻断食
```

**血氧与睡眠：**
```
• 血氧降低→睡眠节律打破
• 深睡时造血功能最佳
• 睡眠质量影响免疫力
• 血氧充足→深度睡眠
```

## 📚 核心理念

**关键洞察：**
1. 鼻呼吸激活副交感神经
2. 口呼吸引发压力反应
3. 舌抵上颚自然闭嘴
4. 胶带法简单有效
5. 呼吸暂停需医疗干预

**健康公式：**
```
优质睡眠 = 鼻呼吸 + 舌抵上颚 + 规律呼吸
```
