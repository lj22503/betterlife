# 生物钟同步 ⏰

**基于《绝佳时间》《好睡》**

---

## 📋 功能描述

指导用户通过光照、运动、进食时间管理，同步生物钟，改善睡眠和健康。

**适用场景：**
- 昼夜节律紊乱
- 时差调整
- 夜班工作
- 睡眠时相延迟

**边界条件：**
- 不替代医疗建议
- 严重节律紊乱需就医
- 循序渐进调整

---

## 🎯 核心功能

### 1. 生物钟基础

**什么是生物钟：**
```
• 体内昼夜节律系统
• 调控 24 小时生理机能
• 主钟位于视交叉上核（下丘脑）
• 调控睡眠 - 觉醒周期
```

**受生物钟调控的生理机能：**
```
• 体温波动
• 激素分泌（皮质醇、褪黑素）
• 血压变化
• 代谢功能
• 认知能力
• 免疫功能
```

### 2. 光照同步

**光照是最强同步信号：**
```
清晨光照：
• 生物钟提前
• 早睡早起
• 抑制褪黑素分泌
• 提升血清素（白天）

傍晚光照：
• 生物钟延迟
• 晚睡晚起
• 抑制褪黑素分泌
• 影响入睡
```

**光照建议：**
```
早晨（起床后 1 小时内）：
✓ 接触自然光 15-30 分钟
✓ 拉开窗帘
✓ 户外散步最佳
✓ 阴天也有效（强度足够）

白天：
✓ 尽量多接触自然光
✓ 室内使用全光谱照明
✓ 每 1-2 小时到户外

傍晚（睡前 2-3 小时）：
✓ 减少强光暴露
✓ 使用暖色光
✓ 避免蓝光（手机、电脑）
✓ 可使用蓝光过滤眼镜

睡前：
✓ 卧室完全黑暗
✓ 遮住所有光源
✓ 必要时戴眼罩
```

### 3. 运动同步

**运动影响生物钟：**
```
早晨/上午运动：
• 生物钟提前
• 促进早睡早起
• 提升白天警觉性

下午运动（15:00-18:00）：
• 体温达到峰值
• 运动表现最佳
• 不影响夜间睡眠

晚上运动（睡前 3 小时内）：
• 可能延迟生物钟
• 提高核心体温
• 影响入睡
• 建议避免剧烈运动
```

**运动建议：**
```
最佳时间：
✓ 早晨：调整生物钟
✓ 下午：运动表现最佳

避免：
× 睡前 3 小时剧烈运动
× 晚上高强度训练
```

### 4. 进食时间同步

**进食时间与代谢：**
```
• 肠道菌群有昼夜节律
• 胰岛素敏感性白天更高
• 晚上代谢效率下降
• 深夜进食增加肥胖风险
```

**进食建议：**
```
早餐（起床后 1-2 小时内）：
✓ 丰富营养
✓ 蛋白质充足
✓ 启动代谢

午餐（12:00-14:00）：
✓ 主要热量摄入
✓ 营养均衡

晚餐（睡前 3-4 小时完成）：
✓ 清淡为主
✓ 减少碳水
✓ 避免过饱

避免：
× 睡前 3 小时内进食
× 深夜零食
× 夜间大量饮水
```

### 5. 时差调整

**东行（提前生物钟）：**
```
出发前：
✓ 提前 2-3 天早睡早起
✓ 早晨接触强光
✓ 晚上避免强光

飞行中：
✓ 按目的地时间调整
✓ 机上尽量睡眠
✓ 多喝水

到达后：
✓ 早晨接触强光
✓ 白天避免小睡
✓ 按当地时间进食
```

**西行（延迟生物钟）：**
```
出发前：
✓ 推迟 2-3 天晚睡晚起
✓ 傍晚接触强光

到达后：
✓ 傍晚接触强光
✓ 早晨避免强光
✓ 按当地时间进食
```

**褪黑素辅助（可选）：**
```
• 东行：睡前 0.5-3mg
• 西行：通常不需要
• 短期使用（3-5 天）
```

### 6. 夜班适应

**夜班挑战：**
```
• 生物钟与工作时间冲突
• 年轻人适应能力强于中老年人
• 长期夜班增加健康风险
```

**适应策略：**
```
工作期间：
✓ 明亮照明（模拟白天）
✓ 定时小睡（20-30 分钟）
✓ 健康饮食（避免大量碳水）

白天睡眠：
✓ 卧室完全黑暗
✓ 使用耳塞/白噪声
✓ 告知家人不要打扰
✓ 固定睡眠时间

下班后：
✓ 戴墨镜回家（避免晨光）
✓ 尽快入睡
✓ 避免社交活动干扰
```

**健康支持：**
```
✓ 整体健康饮食
✓ 复合维生素（可选）
✓ 维生素 D（缺乏时）
✓ 定期体检
```

## 📚 核心理念

**关键洞察：**
1. 光照是最强生物钟同步信号
2. 清晨光照提前生物钟
3. 傍晚光照延迟生物钟
4. 运动和进食也影响生物钟
5. 肠道菌群有昼夜节律

**健康公式：**
```
生物钟同步 = 早晨光照 + 规律作息 + 适时进食
```
