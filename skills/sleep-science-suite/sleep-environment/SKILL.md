# 睡眠环境优化 🛏️

**基于《绝佳时间》《睡觉》《好睡》**

---

## 📋 功能描述

指导用户优化睡眠环境，创造有利于入睡和深睡的条件。

**适用场景：**
- 入睡困难
- 睡眠浅易醒
- 环境干扰
- 出差旅行

**边界条件：**
- 不替代医疗建议
- 严重睡眠障碍需就医
- 个体差异需调整

---

## 🎯 核心功能

### 1. 光线管理

**卧室黑暗度：**
```
理想状态：
✓ 完全黑暗
✓ 无路灯/招牌灯光干扰
✓ 无电子设备指示灯
✓ 无小夜灯

实现方法：
✓ 遮光窗帘（100% 遮光）
✓ 眼罩（备用方案）
✓ 遮盖/移除所有光源
✓ 拔掉电子设备插头
```

**睡前光线：**
```
睡前 2-3 小时：
✓ 减少强光
✓ 使用暖色光（黄/红色）
✓ 避免蓝光（手机、电脑）
✓ 可使用蓝光过滤眼镜

暖色光作用：
• 不触发黑视蛋白
• 促进褪黑素分泌
• 帮助放松入睡
```

**早晨光线：**
```
起床后：
✓ 立即接触自然光
✓ 拉开窗帘
✓ 户外散步最佳
• 抑制褪黑素
• 启动清醒模式
• 调整生物钟
```

### 2. 声音管理

**理想声音环境：**
```
安静：
✓ 隔音良好的卧室
✓ 远离马路/噪音源
✓ 告知家人睡眠时间

白噪声（可选）：
✓ 单调重复的声音
✓ 压制外界噪声
✓ 整晚使用
✓ 音量<56 分贝

自然声音：
✓ 雨声
✓ 海浪声
✓ 风扇声
```

**白噪声使用指南：**
```
正确方法：
✓ 整晚使用
✓ 音量适中（<56 分贝）
✓ 稳定持续
✓ 不关心的声音

作用：
• 压制外界噪声
• 转移注意力
• 释放焦虑
• 让神经疲劳
```

**避免的声音：**
```
× 突然的噪音
× 人声对话
× 电话铃声
× 电视声音（除非稳定背景音）
```

### 3. 温度管理

**理想温度：**
```
卧室温度：18-22℃
• 凉爽有助于入睡
• 核心体温下降促进睡眠
• 过热影响深睡

被窝温度：
• 温暖但不闷热
• 可调节的被子
• 根据季节调整
```

**温度调节方法：**
```
夏季：
✓ 空调/风扇
✓ 透气床品
✓ 凉爽睡衣
✓ 睡前洗澡降温

冬季：
✓ 暖气/电热毯（睡前关闭）
✓ 保暖床品
✓ 穿袜子（暖脚）
✓ 睡前泡澡升温后自然下降

四季通用：
✓ 分层盖被（易调节）
✓ 透气床垫
✓ 保持空气流通
```

### 4. 寝具选择

**枕头：**
```
选择标准：
✓ 保持肩颈头在一条线
✓ 合适高度（侧睡/仰睡）
✓ 合适硬度
✓ 透气材质

药枕（中医）：
• 肝火扰心型
• 痰热扰心型
• 心脾两虚型
• 心胆气虚型
• 胃气不和型
```

**床垫：**
```
选择标准：
✓ 适度支撑
✓ 不过软/过硬
✓ 符合体型
✓ 减少翻身干扰

类型：
• 记忆棉（贴合身体）
• 乳胶（弹性好）
• 弹簧（支撑好）
• 棕榈（偏硬）
```

**床品：**
```
材质：
✓ 纯棉（透气）
✓ 亚麻（透气、吸湿）
✓ 真丝（亲肤）
✓ 天丝（凉爽）

颜色：
✓ 柔和色调
✓ 避免鲜艳刺激
✓ 个人喜好为主
```

### 5. 气味管理

**助眠气味：**
```
精油：
✓ 薰衣草（放松）
✓ 洋甘菊（镇静）
✓ 檀香（安神）
✓ 雪松（稳定）

使用方法：
• 香薰机
• 枕边滴 1-2 滴
• 泡澡加入
• 喷雾喷洒
```

**避免气味：**
```
× 刺激性气味
× 浓烈香水
× 新家具味道
× 化学清洁剂
```

### 6. 方位与布局

**床头朝向（传统建议）：**
```
避免：
× 头对空调（直吹）
× 头朝马路（噪音/气流）
× 头朝马桶（心理/卫生）
× 头朝电磁波密集处

建议：
✓ 安静一侧
✓ 远离门窗直冲
✓ 有靠山感（实墙）
```

**卧室布局：**
```
原则：
✓ 简洁整洁
✓ 减少杂物
✓ 仅用于睡眠/亲密
✓ 不在卧室工作/娱乐

电子设备：
✓ 移除或远离床
✓ 不充电在床边
✓ 飞行模式
✓ 屏幕朝下
```

### 7. 安全感营造

**声音安全感：**
```
• 稳定的背景音
• 熟悉的白噪声
• 家人在附近的感觉
• 门锁好（心理安全）
```

**视觉安全感：**
```
• 柔和的灯光（睡前）
• 熟悉的物品
• 整洁的环境
• 无阴影角落
```

**触觉安全感：**
```
• 舒适的寝具
• 合适的包裹感
• 重力被（可选）
• 抱枕/玩偶
```

## 🧪 使用示例

```
@ant 我睡眠很浅，容易醒，如何改善环境？

信息：
- 目前环境：光线一般、有噪音、温度合适
- 睡眠干扰：路灯、车声、伴侣打鼾
- 居住情况：与伴侣同住
- 预算：中等
```

**预期输出：**
- 评估：光线和噪音是主要问题
- 光线建议：遮光窗帘/眼罩
- 声音建议：白噪声机/耳塞
- 寝具建议：支撑性枕头

## 📚 核心理念

**关键洞察：**
1. 卧室完全黑暗最重要
2. 白噪声压制外界干扰
3. 凉爽环境促进睡眠
4. 寝具支撑影响质量
5. 气味影响放松程度

**健康公式：**
```
优质睡眠环境 = 完全黑暗 + 安静/白噪声 + 凉爽 + 舒适寝具
```
