# 睡眠周期优化 🌙

**基于《绝佳时间》《睡觉》《好睡》**

---

## 📋 功能描述

帮助用户理解睡眠周期，优化睡眠质量，从深睡中获得最大恢复效益。

**适用场景：**
- 睡眠质量差
- 醒来仍感疲劳
- 多梦易醒
- 睡眠时间短

**边界条件：**
- 不替代医疗诊断
- 严重睡眠障碍需就医
- 个体差异需考虑

---

## 🎯 核心功能

### 1. 睡眠周期基础

**睡眠阶段：**
```
非快速眼动睡眠 (NREM)：
• 阶段 1：浅睡（5%）
• 阶段 2：中度睡眠（45%）
• 阶段 3：深睡/慢波睡眠（25%）
  - 头脑最深休息
  - 记忆整合与学习
  - 清理脑内废物（淀粉样蛋白）

快速眼动睡眠 (REM)：
• 做梦阶段（25%）
• 脑波接近清醒
• 身体肌肉受抑制
• 情绪处理与记忆巩固
```

**周期规律：**
```
• 每周期 70-90 分钟
• 每晚 4-5 个周期
• 从快速眼动睡眠中自然醒来最佳
• 深睡需求优先满足
```

### 2. 深睡优化

**深睡的重要性：**
```
✓ 为头脑带来最深休息
✓ 神经细胞层次重塑大脑
✓ 记忆整合与学习
✓ 自由基和氧化物自然降低
✓ 脑部胶质淋巴系统清理废物
✓ 抗氧化和清理机制运作最佳
```

**增加深睡的方法：**
```
✓ 规律作息（固定时间睡觉起床）
✓ 睡前放松（深呼吸、静坐）
✓ 适度运动（但睡前 3 小时避免剧烈）
✓ 睡前远离蓝光（手机、电脑）
✓ 保持卧室凉爽（18-22℃）
✓ 睡前不饮酒（影响深睡比例）
```

### 3. 睡眠周期计算

**理想起床时间：**
```
基于 90 分钟周期：
• 睡 4 个周期 = 6 小时
• 睡 5 个周期 = 7.5 小时
• 睡 6 个周期 = 9 小时

例：23:00 入睡
• 4 周期：05:00 起床
• 5 周期：06:30 起床
• 6 周期：08:00 起床
```

**避免在深睡期醒来：**
```
× 睡 7 小时 = 可能在深睡期被唤醒
✓ 睡 7.5 小时 = 更可能在浅睡期醒来

感觉：
× 深睡期醒来：昏沉、疲惫
✓ 浅睡期醒来：清醒、精神
```

### 4. 睡眠时型识别

**三种时型：**
```
早晨型（早鸟型）10%：
• 早睡早起
• 上午认知能力最强
• 晚上较早疲倦

夜晚型（夜猫子型）25%：
• 晚睡晚起
• 下午/晚上认知能力最强
• 早上难以醒来

中间型（鸽子型）65%：
• 介于两者之间
• 可适度调整
```

**时型测试：**
```
问题：
1. 你自然醒来的时间是？
2. 你什么时候认知能力最强？
3. 你什么时候最容易疲倦？

结果：
• 多数指向早晨→早晨型
• 多数指向夜晚→夜晚型
• 混合→中间型
```

### 5. 睡眠需求评估

**睡眠时长：**
```
• 婴儿：16 小时左右
• 儿童：10-12 小时
• 青少年：8-10 小时
• 成人：7-8 小时（平均）
• 老年人：6-7 小时（正常减少）

关键：
• 没有统一标准
• 自己觉得有没有睡饱
• 休息到的满足感是关键
```

**睡眠质量指标：**
```
✓ 入睡时间<30 分钟
✓ 夜间醒来≤1 次
✓ 醒来后 20 分钟内重新入睡
✓ 白天不极度困倦
✓ 醒来感觉精神恢复
```

### 6. 睡眠日记

**记录内容：**
```
每日记录：
1. 上床时间
2. 尝试入睡时间
3. 入睡花费时间
4. 夜间醒来次数
5. 醒来总时长
6. 最终醒来时间
7. 起床时间
8. 睡眠质量评价（1-5 分）
9. 是否做梦
10. 影响睡眠的白天事件
```

**分析指标：**
```
• 睡眠效率 = 睡眠时间/在床时间 × 100%
  （>85% 为良好）
• 睡眠潜伏期 = 尝试入睡到睡着的时间
  （<30 分钟为正常）
• 醒来次数 = 夜间醒来次数
  （≤1 次为良好）
```

## 📚 核心理念

**关键洞察：**
1. 睡眠由 4-5 个 90 分钟周期组成
2. 深睡为头脑带来最深休息
3. 从快速眼动睡眠中醒来最佳
4. 睡眠时长没有统一标准
5. 自己感觉睡饱是关键

**健康公式：**
```
优质睡眠 = 完整周期 + 充足深睡 + 自然醒来
```
