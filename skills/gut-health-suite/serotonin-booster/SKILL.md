# 血清素提升 😊

**基于川本彻《肠活生活》**

---

## 📋 功能描述

指导用户通过肠道健康提升血清素分泌，改善情绪和抑制食欲。

**适用场景：**
- 情绪低落
- 抑郁倾向
- 食欲失控
- 睡眠问题

**边界条件：**
- 不替代医疗治疗
- 重度抑郁需就医
- 需配合整体健康生活方式

---

## 🎯 核心功能

### 1. 血清素基础知识

**什么是血清素：**
```
• 人称"幸福激素"
• 95% 在肠道制造
• 5% 在大脑制造
• 调节情绪、食欲、睡眠
```

**血清素的作用：**
```
✓ 情绪积极
✓ 抑制食欲
✓ 预防抑郁症
✓ 调节睡眠（转化为褪黑素）
✓ 稳定自主神经
```

### 2. 血清素分泌机制

**分泌流程：**
```
1. 早晨阳光 → 打开血清素分泌开关
2. 肠道有益菌 → 产生短链脂肪酸
3. 短链脂肪酸 → 刺激肠道嗜铬细胞
4. 嗜铬细胞 → 分泌血清素
5. 血清素 → 通过血液输送到大脑
```

**关键开关：**
```
• 早晨阳光（必需）
• 肠道有益菌（必需）
• 短链脂肪酸（信号分子）
```

### 3. 血清素不足症状

**情绪方面：**
```
• 情绪低落
• 容易焦虑
• 缺乏动力
• 抑郁倾向
```

**食欲方面：**
```
• 渴望碳水化合物
• 暴饮暴食
• 夜间进食
• 难以满足
```

**睡眠方面：**
```
• 入睡困难
• 睡眠质量差
• 早醒
• 白天疲倦
```

**身体方面：**
```
• 便秘（肠道血清素不足）
• 慢性疼痛
• 偏头痛
```

### 4. 血清素提升方案

**早晨例行程序：**
```
起床后 1 小时内：
✓ 沐浴晨光（15-30 分钟）
✓ 深呼吸（拉长呼气）
✓ 轻度运动（散步、伸展）
✓ 吃早餐（哪怕水果、酸奶）
```

**肠道支持：**
```
每日：
✓ 可溶性膳食纤维
✓ 发酵食品
✓ 充足水分
✓ 适度运动
```

**营养支持：**
```
血清素原料：
✓ 色氨酸（蛋白质）
  来源：鸡蛋、奶制品、大豆、坚果
✓ 维生素 B6
  来源：香蕉、土豆、鸡肉
✓ 镁
  来源：绿叶蔬菜、坚果、全谷物
```

### 5. 血清素 - 褪黑素循环

**白天（血清素）：**
```
• 阳光照射
• 血清素分泌
• 情绪积极
• 食欲稳定
```

**夜晚（褪黑素）：**
```
• 光线减弱
• 血清素→褪黑素
• 准备睡眠
• 体温下降
```

**破坏循环的行为：**
```
× 晚上暴露在强光下（10 点后）
× 睡眠不足
× 肠道环境恶化
× 不吃早餐
```
