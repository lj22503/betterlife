# 肠道菌群平衡 🦠

**基于川本彻《肠活生活》**

---

## 📋 功能描述

帮助用户了解肠道菌群构成，通过饮食调整达到 2:1:7 健康比例。

**适用场景：**
- 便秘/腹泻
- 减肥困难
- 免疫力低下
- 皮肤问题

**边界条件：**
- 不替代医疗诊断
- 需 1-2 个月持续调整
- 个体差异需观察

---

## 🎯 核心功能

### 1. 肠道菌群 2:1:7 法则

**理想比例：**
```
有益菌：有害菌：中性菌 = 2 : 1 : 7
```

**菌群分类：**
| 类型 | 代表菌 | 比例 | 特点 |
|------|--------|------|------|
| 有益菌 | 双歧杆菌、乳酸菌、丁酸梭菌 | 20% | 产生短链脂肪酸 |
| 有害菌 | 产气荚膜梭菌等 | 10% | 产生毒素、恶臭 |
| 中性菌 | 大肠杆菌等 | 70% | 墙头草，站多数方 |

### 2. 肠道菌群失衡信号

**有益菌不足：**
```
• 便秘或腹泻
• 皮肤粗糙
• 免疫力下降
• 容易感冒
• 情绪低落
```

**有害菌过多：**
```
• 粪便恶臭
• 胀气严重
• 体臭加重
• 容易疲劳
• 肥胖倾向
```

### 3. 有益菌增殖方案

**核心策略：可溶性膳食纤维**
```
为什么是可溶性膳食纤维？
✓ 不被胃酸消化，顺利抵达大肠
✓ 有益菌的主要食物
✓ 效率高，各种有益菌都能利用
✓ 不存在人体不匹配问题
```

**富含可溶性膳食纤维的食物：**
```
🥇 黏糊糊食材：
   • 山药、秋葵、光帽鳞伞
   • 长蒴黄麻、落葵

🥈 海藻类：
   • 海带、裙带菜、海蕴

🥉 谷物：
   • 大麦、燕麦、黑麦、荞麦

其他：
   • 牛蒡、胡萝卜、卷心菜
   • 苹果、西梅、无花果
```

### 4. 轻断食减害菌

**方案：**
```
周末轻断食：
• 1-2 天/月
• 蔬菜汤 + 蔬果汁

日常调整：
• 1 日 3 餐→1 日 2 餐（偶尔）
• 拉长空腹时间
• 让肠胃休息
```

**效果：**
- 减缓有害菌增殖速度
- 有益菌更耐饥饿
- 肠道得到休息

### 5. 发酵食品补充

**日本传统发酵食品：**
```
• 纳豆（丁酸梭菌）
• 味噌（乳酸菌）
• 泡菜（乳酸菌）
• 米酒、酱油、醋
```

**西方发酵食品：**
```
• 酸奶（乳酸菌）
• 奶酪
• 丹贝（印尼大豆发酵）
• 葡萄酒
```

**摄取原则：**
- 每日至少 1 种
- 传统发酵优于工业发酵
- 不加热食用（保留活菌）

## 📚 核心理念

**关键洞察：**
1. 肠道菌群决定健康 90%
2. 理想比例：有益菌 2:有害菌 1:中性菌 7
3. 可溶性膳食纤维是有益菌首选食物
4. 1-2 个月可改变菌群平衡
5. 中性菌是墙头草，站多数方

**健康公式：**
```
肠道健康 = 可溶性膳食纤维 + 发酵食品 + 适度断食
```
