# 排便调理 🚽

**基于川本彻《肠活生活》第 4 章**

---

## 📋 功能描述

帮助用户解决便秘和腹泻问题，建立规律排便习惯。

**适用场景：**
- 便秘（3 天以上无排便）
- 腹泻（每日 3 次以上稀便）
- 排便不规律
- 排便费力

**边界条件：**
- 不替代医疗诊断
- 严重症状需就医
- 需排除器质性疾病

---

## 🎯 核心功能

### 1. 便秘类型识别

**交感神经优势型（最常见）：**
```
原因：压力、紧张、忙碌
机制：交感神经抑制肠道蠕动
特征：
• 有便意但排不出
• 粪便干硬
• 排便费力
• 压力大时加重
```

**蠕动不足型：**
```
原因：运动不足、纤维不足
机制：肠道蠕动缓慢
特征：
• 便意少
• 粪便在肠道停留久
• 水分被过度吸收
• 粪便干硬
```

**弛缓型：**
```
原因：腹肌无力、年老
机制：排便力量不足
特征：
• 有便意但无力排出
• 排便时间长
• 需要用力
```

### 2. 腹泻类型识别

**小肠腹泻：**
```
原因：细菌、病毒毒素
特征：
• 水样便
• 腹痛明显
• 可能发烧
• 急性发作
```

**大肠腹泻：**
```
原因：细菌感染最常见
特征：
• 黏液便
• 里急后重
• 腹痛
• 可能带血
```

**压力性腹泻：**
```
原因：焦虑、紧张
机制：副交感神经过度兴奋
特征：
• 紧张时发作
• 肠易激综合征
• 反复发作
```

### 3. 便秘解决方案

**激活副交感神经：**
```
✓ 腹式呼吸（拉长呼气）
✓ 温水泡澡
✓ 睡前放松
✓ 适度运动
✓ 静坐/冥想
```

**促进肠道蠕动：**
```
✓ 可溶性膳食纤维
✓ 充足水分（2-3L/天）
✓ 腹部按摩
✓ 肠道活力保健操
✓ 早晨排便习惯
```

**营养支持：**
```
✓ 泛酸（维生素 B5）
  来源：肝脏、鸡蛋、花椰菜
✓ 镁
  来源：绿叶蔬菜、坚果
✓ 益生菌
  来源：发酵食品
```

### 4. 腹泻解决方案

**急性期处理：**
```
✓ 补充水分和电解质
✓ 清淡饮食（米汤、稀饭）
✓ 避免油腻、辛辣
✓ 温暖肠道（暖宝宝）
✓ 休息
```

**恢复期调理：**
```
✓ 逐渐增加纤维
✓ 益生菌补充
✓ 发酵食品
✓ 避免生冷
```

**慢性腹泻：**
```
✓ 排查食物不耐受
✓ 压力管理
✓ 肠道菌群调整
✓ 就医排查
```

### 5. 肠道活力保健操

**扭转运动：**
```
1. 双腿并拢前伸，坐地板/床上
2. 左膝弯曲，左脚放右腿外侧
3. 右手抱左膝，上身向左侧扭转
4. 左手撑后方地板
5. 保持姿势，缓慢呼吸 3-5 次
6. 换边重复

次数：左右各 1-3 次
时间：早晨或睡前
```

**腹部按摩：**
```
画圈按摩：
• 以肚脐为圆心
• 手指并拢，适度压力
• 按压腹部凹下 1-2 厘米
• 顺时针画圈 10 圈

穴位按压（各 3 秒）：
• 天枢穴：肚脐横向 3 指（左右）
• 大巨穴：天枢下方 3 指（左右）
• 关元穴：肚脐下方 4 指
• 中脘穴：胸口窝与肚脐中点
```

### 6. 排便习惯建立

**早晨例行程序：**
```
起床后：
1. 喝一杯温水
2. 沐浴晨光
3. 轻度活动
4. 吃早餐（触发胃结肠反射）
5. 固定时间如厕（即使无便意）
```

**理想排便：**
```
• 频率：每日 1-2 次
• 时间：早晨最佳
• 时长：5 分钟内
• 感觉：轻松、无费力
• 粪便：成形、香蕉状
```

## 🧪 使用示例

```
@ant 我经常便秘，很痛苦

信息：
- 排便类型：便秘
- 频率：每 3-4 天一次
- 粪便质量：硬、费力
- 伴随症状：腹胀、皮肤粗糙
- 压力水平：高
```

**预期输出：**
- 类型：交感神经优势型便秘
- 原因：压力大、副交感神经不足
- 立即缓解：腹部按摩、温水
- 长期方案：压力管理、纤维增加

## 📚 核心理念

**关键洞察：**
1. 便秘根本原因是交感神经优势
2. 激活副交感神经是关键
3. 温暖肠道有效缓解
4. 早晨例行程序建立习惯
5. 肠道活力操简单有效

**健康公式：**
```
规律排便 = 副交感神经优势 + 充足纤维 + 适度蠕动
```
