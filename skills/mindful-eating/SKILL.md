# 正念饮食引导器 🍽️

**基于《5-8 周正念禅修练习》+《十分钟冥想》**

---

## 📋 功能描述

引导用户进行正念饮食练习，品味食物，培养对饮食的觉察，感恩食物。

**适用场景：**
- 正念入门练习
- 改善饮食习惯
- 品味食物
- 减少暴食
- 培养感恩心

**边界条件：**
- 选择平时不吃的食物
- 不受打扰的环境
- 慢慢品味
- 感恩食物

---

## 🎯 核心功能

### 1. 葡萄干禅修（8 步骤）

**步骤 1：持（20-30 秒）**
```
• 拿起葡萄干放在手掌
• 认真观察
• 感受重量、阴影
```

**步骤 2：看（20-30 秒）**
```
• 认真审视
• 滑过每个部分
• 审视光亮、阴暗、皱褶
```

**步骤 3：触（20-30 秒）**
```
• 翻转葡萄干
• 探索纹理结构
• 感受触感
```

**步骤 4：嗅（20-30 秒）**
```
• 放在鼻子下闻
• 注意气味
• 让气味充满意识
```

**步骤 5：放（20-30 秒）**
```
• 慢慢放入口中
• 注意手臂动作
• 感受舌头接纳
• 不要咀嚼
```

**步骤 6：嚼（20-30 秒）**
```
• 有意识地咬一口
• 感受变化
• 注意味道释放
• 继续慢慢咀嚼
```

**步骤 7：咽（20-30 秒）**
```
• 感受吞咽欲望
• 跟踪吞咽过程
• 感受落入胃部
```

**步骤 8：感（20-30 秒）**
```
• 感受吃完后的影响
• 口腔余味
• 是否有自动拿取的倾向
```

### 2. 饮食冥想（来自《十分钟冥想》）

**餐前准备：**
```
1. 在餐桌旁坐下来，最好是一个人
2. 远离所有外在的分心事物
3. 深呼吸几次，鼻子吸气，嘴巴呼气
4. 让身体和心灵安顿下来
```

**欣赏食物：**
```
1. 它们来自哪里，出自哪个国家
2. 是自发生长出来的，还是人为制造出来的
3. 想象它们生长环境中的各种要素
4. 想象照看这些庄稼或者动物的是什么人
```

**感恩：**
```
1. 花一点儿时间感恩一下
2. 感恩你的碗里有食物可吃
3. 不要因为你想吃它而有愧疚感
4. 感谢和感恩是稳定的正念修习的中心所在
```

**进餐觉察：**
```
1. 比往常慢一些，自然地做，不要想太多
2. 用手拿着吃的食物：
   - 注意它的纹理、温度以及颜色
3. 食物放在盘子里的：
   - 留意餐具的质地和温度
   - 花些时间观察餐盘上的颜色
4. 将食物送进嘴里时：
   - 把关注焦点从手上移开
   - 转到眼睛、鼻子和嘴巴
   - 食物闻起来是什么味道
   - 近距离观察，它看起来如何
   - 口感、纹理、温度如何
```

**心灵反应：**
```
1. 留意心灵对食物做出的反应
2. 喜欢还是不喜欢
3. 接受了食物原本的样子，还是对某些方面感到抗拒
4. 太烫或者太凉、太甜或者太酸
5. 留意心灵如何忙着对食物进行评判
6. 如何跟以前的饭菜做比较
```

**习惯觉察：**
```
1. 心灵开始游移时，温和地把注意力带回
2. 留意自己是否会习惯性地吃得很快
3. 是否有偏好甜食的强烈冲动
4. 对正在吃的东西感到不安时，留意这些想法
5. 留意吃饭的时候你的呼吸是怎样的
```

**结束觉察：**
```
1. 快要吃完时，留意是否对快吃完饭有种失望感或解脱感
2. 再花些时间品味一下最后一口
3. 深呼吸几次
4. 回想一下盘子里满是食物的时候是什么样子
5. 现在里面空空如也的时候又是什么样子
6. 比较之前胃里的空荡感和现在的饱腹感
7. 留意不断变化和有始有终的事物
```

## ⚠️ 常见错误

**错误 1：狼吞虎咽**
```
问题：
• 习惯性地快速吃
• 不注意感受

解决：
✓ 慢慢品味
✓ 每个步骤 20-30 秒
✓ 注意每个细节
```

**错误 2：注意力分散**
```
问题：
• 边吃边想其他事
• 不注意当下

解决：
✓ 发现分散时留意
✓ 柔和地带回
✓ 专注当前一刻
```

**错误 3：评判食物**
```
问题：
• "这个不好吃"
• 与以前的饭菜比较

解决：
✓ 留意心灵如何忙着评判
✓ 接受食物原本的样子
✓ 不抗拒
```

## 📚 核心理念

**关键洞察：**
1. 每个步骤 20-30 秒
2. 像从未见过一样观察
3. 品味 300 多种口味
4. 分散时柔和带回
5. 感恩是正念修习的中心

**健康公式：**
```
正念饮食 = 8 步骤 + 每步 20-30 秒 + 感恩 + 专注当下
```
