# 正念饮食引导器 🍇

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行正念饮食练习，品味食物，培养对饮食的觉察。

**适用场景：**
- 正念入门练习
- 改善饮食习惯
- 品味食物
- 减少暴食

**边界条件：**
- 选择平时不吃的食物
- 不受打扰的环境
- 慢慢品味

---

## 🎯 核心功能

### 1. 葡萄干禅修（8 步骤）

**步骤 1：持（20-30 秒）**
```
• 拿起葡萄干放在手掌
• 认真观察
• 感受重量、阴影
```

**步骤 2：看（20-30 秒）**
```
• 认真审视
• 滑过每个部分
• 审视光亮、阴暗、皱褶
```

**步骤 3：触（20-30 秒）**
```
• 翻转葡萄干
• 探索纹理结构
• 感受触感
```

**步骤 4：嗅（20-30 秒）**
```
• 放在鼻子下闻
• 注意气味
• 让气味充满意识
```

**步骤 5：放（20-30 秒）**
```
• 慢慢放入口中
• 注意手臂动作
• 感受舌头接纳
• 不要咀嚼
```

**步骤 6：嚼（20-30 秒）**
```
• 有意识地咬一口
• 感受变化
• 注意味道释放
• 继续慢慢咀嚼
```

**步骤 7：咽（20-30 秒）**
```
• 感受吞咽欲望
• 跟踪吞咽过程
• 感受落入胃部
```

**步骤 8：感（20-30 秒）**
```
• 感受吃完后的影响
• 口腔余味
• 是否有自动拿取的倾向
```

### 2. 巧克力禅修

**步骤：**
```
1. 打开包装，享受芳香
2. 掰下一块，仔细端详
3. 放入嘴中，让它融化
4. 留意吮吸欲望
5. 品味 300 多种口味
6. 慢慢吞咽
7. 再拿一块重复
```

---

## ⚠️ 常见错误

**错误 1：狼吞虎咽**
```
问题：
• 习惯性地快速吃
• 不注意感受

解决：
✓ 慢慢品味
✓ 每个步骤 20-30 秒
✓ 注意每个细节
```

**错误 2：注意力分散**
```
问题：
• 边吃边想其他事
• 不注意当下

解决：
✓ 发现分散时留意
✓ 柔和地带回
✓ 专注当前一刻
```

## 📚 核心理念

**关键洞察：**
1. 每个步骤 20-30 秒
2. 像从未见过一样观察
3. 品味 300 多种口味
4. 分散时柔和带回

**健康公式：**
```
正念饮食 = 8 步骤 + 每步 20-30 秒 + 专注当下
```
