# 感受需要识别器 💗

**基于《非暴力沟通：让表达温柔有力》（马歇尔·卢森堡）**

---

## 📋 功能描述

帮助用户识别感受和需要，改善沟通品质。

**适用场景：**
- 识别感受和需要
- 改善人际关系
- 提升沟通品质

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 观察具体行为
2. 识别感受
3. 识别需要
4. 提出请求
```

---

## ⚠️ 常见错误

**错误 1：混淆观察和评论**
```
问题：
• "你总是..."（评论）

解决：
✓ 区分观察和评论
✓ 在特定的时间和情境中观察
✓ 不主张绝对化的结论
```

---


---

## 📚 核心理念

**关键洞察：**
1. 识别感受和需要
2. 区分观察和评论
3. 直接说出需要
4. 请求而非命令

---

*不区分观察和评论，人们将倾向于听到批评，并反驳我们。
直接说出我们的需要，其他人就较有可能作出积极的回应。* 💗
