# 成功日记 📔

**基于《小狗钱钱》核心理念**

---

## 📋 功能描述

帮助用户每天记录至少 5 条成功事项，建立自信，克服恐惧，形成积极的自我认知。

**适用场景：**
- 儿童/青少年自信心建立
- 成年人自我激励
- 克服恐惧和焦虑
- 培养积极心态

**边界条件：**
- 不替代心理咨询
- 聚焦日常小成就
- 适合 8 岁以上人群

---

## 🎯 核心功能

### 1. 每日成功记录

引导用户记录当天成功的 5 件事。

**输入：**
- 日期
- 5 条成功事项（任何小事都可以）
- 心情评分（1-10）

**输出：**
```json
{
  "date": "2026-03-18",
  "successes": [
    {
      "id": 1,
      "content": "今天早起完成了晨跑",
      "category": "健康",
      "confidence_boost": "高"
    },
    {
      "id": 2,
      "content": "主动和陌生人打招呼",
      "category": "社交",
      "confidence_boost": "中"
    },
    {
      "id": 3,
      "content": "完成了工作报告",
      "category": "工作",
      "confidence_boost": "中"
    },
    {
      "id": 4,
      "content": "给妈妈打了电话",
      "category": "家庭",

      "confidence_boost": "中"
    }
  ],
  "mood_score": 8,
  "streak_days": 7
}
```

### 2. 自信建立分析

分析成功日记对自信心的影响。

**分析维度：**
- 连续记录天数
- 成功事项类别分布
- 自信心提升趋势
- 恐惧克服案例

**输出：**
```json
{
  "streak_days": 7,
  "total_successes": 35,
  "categories": {
    "健康": 10,
    "工作": 8,
    "学习": 7,
    "社交": 5,
    "家庭": 5
  },
  "confidence_trend": "上升",
  "fear_overcome": [
    "公开演讲",
    "主动社交"
  ],
  "insight": "你在健康领域最有成就感，社交恐惧正在克服"
}
```

### 3. 恐惧克服辅助

在遇到困难时，从成功日记中汲取力量。

**使用场景：**
- 面临挑战时
- 感到恐惧时
- 自我怀疑时

**输出：**
```markdown
# 恐惧克服卡片

## 当前挑战
公开演讲

## 你的成功证据
1. 上周完成了部门汇报（成功！）
2. 上个月主动发言 3 次（成功！）
3. 去年参加了演讲比赛（成功！）

## 自信 affirmations
- "我曾经做到过，这次也能做到"
- "每一次尝试都是成功"
- "恐惧是正常的，但我有能力应对"

## 行动建议
翻一翻成功日记，找到 3 件类似的成功案例
```

### 4. 周期性回顾

每周/每月回顾成功日记，发现成长模式。

**输出：**
```markdown
# 本周成功回顾（2026-03-11 ~ 2026-03-17）

## 本周亮点
- 连续 7 天记录成功日记 ✅
- 克服了公开演讲恐惧 🎉
- 完成了 3 次主动社交 🌟

## 成长洞察
- 自信心从 6 分提升到 8 分
- 社交类成功从 1 次增加到 5 次
- 开始主动挑战舒适区

## 下周目标
- 继续保持每日记录
- 挑战 1 次更大的公开演讲
- 帮助他人建立成功日记习惯
```

```json

  "action": {
    "type": "string",
    "required": true,
    "description": "操作类型（record/review/overcome）"
  },
  "date": {
    "type": "string",
    "required": false,
    "description": "日期（YYYY-MM-DD）"
  },
  "successes": {
    "type": "array",
    "required": false,
    "description": "成功事项列表"

  },

    "type": "string",
    "required": false,
    "description": "回顾周期（weekly/monthly）"
  }
}
```

## ⚠️ 错误处理

| 错误码 | 含义 | 处理建议 |
|--------|------|----------|
| 400 | 参数缺失 | 检查必填字段（action） |
| 400 | 成功事项不足 | 建议至少记录 5 件事 |
| 400 | 日期无效 | 日期格式应为 YYYY-MM-DD |
| 500 | 内部错误 | 重试或联系开发者 |

## 📚 政策依据

**核心理论：**
- 《小狗钱钱》- 博多·舍费尔
- 积极心理学（Positive Psychology）
- 自我效能理论（Self-Efficacy Theory）

**关键概念：**
- 成功日记（每天 5 条成功）
- 自信建立（从小事开始）
- 恐惧克服（用证据对抗恐惧）
- 过于自信比不够自信要好
