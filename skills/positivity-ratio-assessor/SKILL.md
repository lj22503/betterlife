# 积极率评估器 📊

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户评估自己的积极率（P/N），了解积极情绪与消极情绪的配比，预测生活轨迹。

**适用场景：**
- 想了解自己的积极率
- 测试积极情绪水平
- 预测生活轨迹
- 想提升幸福感

**边界条件：**
- 积极率不是越多越好
- 3:1 是临界点
- 上限约 11:1

---

## 🎯 核心功能

### 1. 积极率定义

**公式：**
```
积极率 = 积极情绪出现频率 ÷ 消极情绪出现频率
即：P/N
```

### 2. 临界点

| 积极率 | 状态 | 说明 |
|--------|------|------|
| **< 1:1** | 枯萎失败 | 消极情绪过多 |
| **1:1 - 3:1** | 普通 | 积极情绪不足 |
| **≥ 3:1** | 欣欣向荣 | 临界点以上 |
| **> 11:1** | 过度 | 积极情绪过多 |

### 3. 10 种积极情绪

**积极情绪项目（10 个）：**
- [ ] 逗趣、好玩或可笑
- [ ] 敬佩、惊奇或叹为观止
- [ ] 感激、赞赏或感恩
- [ ] 希望、乐观或备受鼓舞
- [ ] 激励、振奋或兴高采烈
- [ ] 兴趣、吸引注意或好奇
- [ ] 快乐、高兴或幸福
- [ ] 爱、亲密感或信任
- [ ] 自豪、自信或自我肯定
- [ ] 宁静、满足或平和

**消极情绪项目（10 个）：**
- [ ] 愤怒、生气或懊恼
- [ ] 羞愧、屈辱或丢脸
- [ ] 轻蔑、藐视或鄙夷
- [ ] 反感、讨嫌或厌恶
- [ ] 尴尬、难为情或羞愧
- [ ] 内疚、忏悔或应受谴责
- [ ] 仇恨、不信任或怀疑
- [ ] 悲伤、消沉或不幸
- [ ] 恐惧、害怕或担心
- [ ] 压力、紧张或不堪重负

### 4. 计分方法

**积极情绪得分：**
- 评定为 2 或以上的数量

**消极情绪得分：**
- 评定为 1 或以上的数量

**积极率计算：**
```
积极率 = 积极情绪得分 ÷ 消极情绪得分
（如消极情绪为 0，用 1 代替）
```

---

## ⚠️ 常见错误

**错误 1：认为积极情绪越多越好**
```
问题：
• "我要追求 100% 积极"
• 认为消极情绪越少越好

解决：
✓ 积极情绪过多会使人狂妄浮夸
✓ 消极情绪过多使人悲观消沉
✓ 适度配比（3:1）才能提升幸福感
✓ 上限约 11:1
```

**错误 2：忽视消极情绪的价值**
```
问题：
• "消极情绪没用"
• 试图消除所有消极情绪

解决：
✓ 消极情绪能帮助生存
✓ 愤怒产生攻击冲动
✓ 恐惧引起逃避冲动
✓ 关键是配比，不是消除
```

## 📚 核心理念

**关键洞察：**
1. 积极率 = 积极情绪频率 ÷ 消极情绪频率
2. 3:1 是临界点
3. 上限约 11:1
4. 积极情绪过多或过少都不好

```
欣欣向荣 = 积极率 ≥ 3:1 且 ≤ 11:1
```
