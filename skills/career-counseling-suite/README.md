**核心理念：**
- 🧭 **生涯咨询师** - 完整的生涯咨询工具箱

- 📈 **生涯发展** - 规划人生各阶段的发展任务
- 📖 **生涯故事** - 用故事重构生涯意义
- 🎲 **生涯决定** - 做出聪明的生涯选择

## 📚 核心理论

### 生涯辅导基本概念

**命定 + 自定：**
- 命定：先天条件、环境限制
- 自定：个人选择、主动规划

### 霍兰德六型 (RIASEC)

| 类型 | 特点 | 适合职业 |
|------|------|----------|
| R 现实型 | 动手操作、机械 | 工程师、技师 |
| I 研究型 | 思考、分析 | 科学家、程序员 |
| A 艺术型 | 创造、表达 | 艺术家、设计师 |
| S 社会型 | 助人、教导 | 教师、咨询师 |
| E 企业型 | 领导、说服 | 企业家、销售 |
| C 常规型 | 秩序、精确 | 会计、行政 |

### 生涯发展五阶段

| 阶段 | 年龄 | 发展任务 |
|------|------|----------|
| 成长期 | 0-14 岁 | 发展自我概念 |
| 探索期 | 15-24 岁 | 尝试、学习、初步选择 |
| 建立期 | 25-44 岁 | 稳定、巩固、提升 |
| 维持期 | 45-64 岁 | 维持成就、更新技能 |
| 衰退期 | 65 岁+ | 适应退休生活 |

### 生涯即故事

**6 种创新思潮：**
1. 不再有专家 - 当事人是自己生涯的专家
2. 授权取代适配 - 赋能而非匹配
3. 重写"总体叙事"
4. 生涯是专属于这个人的
5. 生涯发展理论并非咨询理论
6. "故事"取代"分数"

### 生涯决定理论

**职业选择状态四种分类：**
1. 没有选择
2. 不确定的选择
3. 兴趣与能力倾向的落差
4. 不聪明的选择

**CASVE 循环决策：**
- C 沟通：识别问题
- A 分析：分析自我和环境
- S 综合：产生选项
- V 评估：评估选项
- E 执行：实施选择
