# 分手恢复引导器 💔

**基于《亲密关系》（罗兰·米勒）**

---

## 📋 功能描述

帮助用户从分手/离婚中恢复，改善亲密关系品质。

**适用场景：**
- 从分手/离婚中恢复
- 改善亲密关系
- 提升关系满意度

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 评估当前状况
2. 识别问题所在
3. 采取改善行动
4. 持续维护关系
```

---

## ⚠️ 常见错误

**错误 1：忽视个体差异**
```
问题：
• 过度关注性别差异
• 忽视个体差异

解决：
✓ 两性性别内的行为和观点差异通常远大于两性之间的平均差异
✓ 个体差异才是影响人际交往的更重要的力量
```

## 📚 核心理念

**关键洞察：**
1. 从分手/离婚中恢复
2. 个体差异比性别差异更重要
3. 亲密关系满足的秘诀是欣赏和感激
4. 人际关系的质量比数量更重要
