# 小狗钱钱 Skill 套件 🐕

**基于博多·舍费尔《小狗钱钱》的财商教育技能**

---

## 📖 简介

这套技能将《小狗钱钱》的核心理念转化为实用的 AI 工具，帮助儿童、青少年和成年人建立正确的金钱观，实现财务自由。

**核心理念：**
- 💰 **梦想储蓄罐** —— 为每个梦想单独储蓄
- 📔 **成功日记** —— 每天记录 5 件成功事项
- 🪿 **金鹅账户** —— 养鹅生蛋，本金永远不花
- 🎯 **72 公式** —— 快速估算复利和通胀
- 💪 **第一桶金** —— 72 小时内开始行动

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/lj22503/betterlife.git

# 进入目录
cd betterlife/skills
```

### 使用示例

#### 1. 设置梦想储蓄罐

```
@ant 帮我设置梦想储蓄罐

信息：
- 年龄：12 岁
- 月收入：500 元（零花钱）
- 梦想：
  1. 变速自行车（2000 元）
  2. 全套漫画（800 元）
  3. 夏令营（3000 元）
```

#### 2. 记录成功日记

```
@ant 帮我记录今天的成功日记

今天成功的 5 件事：
1. 早起晨跑 3 公里
2. 完成了工作报告
3. 主动和同事打招呼
4. 读了 30 分钟书
5. 给妈妈打了电话
```

#### 3. 计算金鹅账户

```
@ant 我想规划财务自由

信息：
- 年龄：30 岁
- 月收入：20000 元
- 月开销：12000 元
- 当前储蓄：100000 元
```

#### 4. 72 公式计算

```
@ant 年化 8% 的收益，多久翻倍？
```

#### 5. 发现赚钱机会

```
@ant 我想赚零花钱，有什么建议？

信息：
- 年龄：12 岁
- 技能：喜欢动物、会拍照
- 时间：周末 5 小时
```

## 📚 《小狗钱钱》核心课程

### 第 1 课：梦想清单
**技能：** `dream-saver`

- 列出你的 10 个梦想
- 排序优先级
- 为每个梦想准备储蓄罐

### 第 2 课：成功日记
**技能：** `success-diary`

- 每天记录 5 件成功事项
- 建立自信心
- 用证据克服恐惧

### 第 3 课：金鹅账户
**技能：** `golden-goose`

- 理解"鹅"与"蛋"的关系
- 设立投资本金账户
- 实现被动收入

### 第 4 课：72 魔法
**技能：** `money-magic`

- 用 72 公式计算翻倍时间
- 理解复利的魔力
- 认识通胀的侵蚀

### 第 5 课：第一桶金
**技能：** `first-income`

- 发现赚钱机会
- 克服"谈钱羞耻"
- 72 小时内开始行动

## 📊 学习路径

### 儿童版（10-15 岁）

```
第 1 周：梦想储蓄罐（设立 3 个梦想）
第 2 周：成功日记（每天 5 条）
第 3 周：第一桶金（发现赚钱机会）
第 4 周：金鹅账户（开始存钱）
第 5 周：72 公式（理解复利）
```

### 青少年版（16-22 岁）

```
第 1 周：梦想储蓄罐 + 成功日记
第 2 周：第一桶金（发展副业）
第 3 周：金鹅账户（开始投资）
第 4 周：72 公式（深入理解）
第 5-8 周：实践 + 复盘
```

### 成年人版（23 岁+）

```
第 1 周：成功日记（重建自信）
第 2 周：金鹅账户（财务诊断）
第 3 周：72 公式（投资规划）
第 4 周：梦想储蓄罐（人生目标）
第 5-12 周：财务自由路径实施
```
