# 穷爸爸富爸爸 Skill 套件 💰

**基于罗伯特·清崎《穷爸爸富爸爸》的财商教育技能**

---

## 📖 简介

这套技能将《穷爸爸富爸爸》的核心理念转化为实用的 AI 工具，帮助用户从雇员思维转变为投资者思维，建立资产项，逃脱老鼠赛跑。

**核心理念：**
- 🧠 **思维转换** —— 从"为钱工作"到"让钱为自己工作"
- 📊 **资产负债** —— 分清什么是资产什么是负债
- 🐭 **老鼠赛跑** —— 逃脱为钱工作的循环
- 💡 **财商提升** —— 学习财务/投资/市场/法律知识
- 🎮 **现金流游戏** —— 在游戏中学习投资

## 📚 核心内容

### 富爸爸 vs 穷爸爸

| 维度 | 穷爸爸 | 富爸爸 |
|------|--------|--------|
| 金钱观 | 工作回报 | 投资回报 |
| 工作观 | 为钱工作 | 让钱为自己工作 |
| 风险观 | 避免风险 | 管理风险 |
| 口头禅 | 我可付不起 | 我怎样才能付得起 |

### 资产 vs 负债

**资产：** 能把钱放进我口袋里的东西
- 出租房产
- 分红股票
- 债券利息
- 业务收入

**负债：** 把钱从我口袋里取走的东西
- 自住房贷
- 车贷
- 信用卡债务
- 奢侈品消费

### 老鼠赛跑循环

```
恐惧（没钱）→努力工作→得到报酬→贪婪（想买）→消费→没钱→恐惧
```

**逃脱路径：**
1. 建立资产项
2. 关注自己的事业
3. 先支付自己

### 财商四大支柱

1. 财务知识（读懂报表）
2. 投资知识（钱生钱）
3. 市场知识（供需/情绪）
4. 法律知识（税收/合规）
