# 库存周期定位器 📦

**基于周金涛《涛动周期论》核心理念**

---

## 📋 功能描述

帮助用户识别基钦周期（库存周期，约 3 年），把握短期经济波动和交易机会。

**适用场景：**
- 短期经济判断
- 股票/商品交易时机
- 企业库存管理
- 宏观经济研究

**边界条件：**
- 不提供具体投资建议
- 基于历史规律，非精确预测
- 适合 18 岁以上人群

---

## 🎯 核心功能

### 1. 库存周期定位

识别当前库存周期位置。

**库存周期：** 约 3 年（36 个月）轮回

**周期阶段：**
- 补库存（12-18 个月）：需求上升，生产扩张
- 去库存（12-18 个月）：需求下降，生产收缩

**输出：**
```json
{
  "current_cycle": 18,
  "cycle_period": "2024-2027",
  "current_phase": "去库存",
  "phase_start": "2025-06",
  "phase_duration": "18 个月",
  "next_phase": "补库存",
  "next_phase_start": "2026-12",
  "insight": "当前处于去库存阶段，2026 年底进入补库存周期"
}
```

### 2. 三周期嵌套分析

分析康波 + 房地产 + 库存三周期嵌套关系。

**嵌套关系：**
- 1 个康波周期（60 年）= 3 个房地产周期（20 年）
- 1 个房地产周期（20 年）= 2 个固定资产投资周期（10 年）
- 1 个固定资产投资周期（10 年）= 3 个库存周期（3 年）

**人生周期：**
- 人生 = 1 次康波 + 3 次房地产周期 + 9 个固定资产投资周期 + 18 次库存周期

**输出：**
```json
{
  "kondratieff": {
    "phase": "萧条期",
    "period": "2018-2030"
  },
  "real_estate": {
    "phase": "调整期",
    "period": "2021-2028"
  },
  "inventory": {
    "phase": "去库存",
    "period": "2025-2026"
  },
  "nested_implication": "三周期叠加调整，经济下行压力大",
  "strategy": "防守为主，等待周期反转"
}
```

### 3. 库存周期与资产价格

分析库存周期对各资产类别的影响。

**规律：**
- 补库存期：股票 > 大宗商品 > 债券
- 去库存期：债券 > 现金 > 股票 > 大宗商品

**输出：**
```json
{
  "inventory_phase": "去库存",
  "asset_performance": {
    "stocks": "熊市（企业盈利下降）",
    "bonds": "牛市（利率下行）",
    "commodities": "熊市（需求疲软）",
    "cash": "保值（流动性好）"
  },
  "sector_rotation": {
    "defensive": "公用事业、必需消费、医疗",
    "cyclical": "避免（工业、材料、能源）"
  },
  "recommendation": "配置债券 + 防御性股票 + 现金"
}
```

### 4. 库存周期交易策略

基于库存周期制定交易策略。

**策略：**
- 补库存初期：买入股票/商品
- 补库存后期：减仓
- 去库存初期：持有债券/现金
- 去库存后期：准备买入

**输出：**
```json
{
  "current_phase": "去库存中期",
  "trading_strategy": {
    "stocks": "定投优质资产，等待低点",
    "bonds": "持有（利率下行受益）",
    "commodities": "避免（需求疲软）",
    "cash": "保持充足流动性"
  },

    "PPI 同比（判断价格拐点）",
    "工业企业利润（判断盈利拐点）",
    "PMI（判断景气度）"
  ],
  "reversal_signal": "PPI、CPI 转头向上，准备买入"
}
```

## 🧪 使用示例

### 示例 1：当前经济形势

```
@ant 现在经济形势如何？
```

**预期输出：**
- 康波：萧条期（2018-2030）
- 房地产：调整期（2021-2028）
- 库存：去库存（2025-2026）
- 结论：三周期叠加调整，防守为主

### 示例 2：股票投资时机

```
@ant 现在适合买股票吗？
```

**预期输出：**
- 库存周期：去库存期（企业盈利下降）
- 策略：定投优质资产，等待低点
- 反转信号：PPI、CPI 转头向上

### 示例 3：大宗商品投资

```
@ant 现在适合投资大宗商品吗？
```

**预期输出：**
- 康波萧条期：商品熊市（2011-2030）
- 库存去化：需求疲软
- 建议：避免，等待 2030 年后
