# 抑郁对抗器 🛡️

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户对抗抑郁，使用认知疗法和乐观技术摆脱抑郁情绪。

**适用场景：**
- 抑郁情绪
- 想摆脱抑郁
- 负面思维循环
- 情绪低落

**边界条件：**
- 不替代专业治疗
- 严重抑郁需就医
- 配合专业治疗使用

---

## 🎯 核心功能

### 1. 抑郁类型

**一般性抑郁：**
```
• 来自痛苦和失落
• 正常的情绪反应
```

**单极抑郁症：**
```
• 没有躁狂症状
• 需要专业治疗
```

**双极抑郁症（躁郁症）：**
```
• 一端躁狂，一端抑郁
• 遗传几率大
• 需要专业治疗
```

### 2. 认知疗法五策略

**策略 1：认识自动想法**
```
学会去认识在情绪最低沉时自动冒出来的想法
```

**策略 2：与想法抗争**
```
举出各种与之相反的例子
```

**策略 3：重新归因**
```
用不同的解释去对抗原有的想法
```

**策略 4：转移注意力**
```
把自己从抑郁的思绪中引开
```

**策略 5：质疑假设**
```
认识并且质疑那些控制你并引起抑郁的假设
```

### 3. 保护因素

**三个保护因素（有其一就不易得抑郁症）：**
1. 与配偶或情人有非常亲密、无话不谈的关系
2. 外出工作
3. 家中 14 岁以下的孩子少于 3 个

---

## ⚠️ 常见错误

**错误 1：忽视反刍**
```
问题：
• 不断想负面事情
• 告诉自己事情有多糟

解决：
✓ 悲观的反刍者会出现问题
✓ 停止反刍
✓ 转移注意力
```
