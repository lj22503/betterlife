# 归因风格训练器 🎯

**基于《活出最乐观的自己》（马丁·塞利格曼）**

---

## 📋 功能描述

帮助用户训练乐观的归因风格，改变对事件的因果分析习惯。

**适用场景：**
- 想改变归因习惯
- 经常错误归因
- 想学习乐观归因

---

## 🎯 核心功能

### 1. 归因风格

**悲观归因：**
```
• 坏事 = 内在 + 永久 + 普遍
• 好事 = 外在 + 暂时 + 特定
```

**乐观归因：**
```
• 坏事 = 外在 + 暂时 + 特定
• 好事 = 内在 + 永久 + 普遍
```

### 2. 训练方法

**每天练习：**
```
1. 记录发生的事
2. 识别自己的归因
3. 调整为乐观归因
4. 重复练习
```

---

## ⚠️ 常见错误

**错误 1：忽视妈妈的影响力**
```
问题：
• 忽视家庭影响
• 忽视父母的归因风格

解决：
✓ 孩子的解释风格受妈妈影响
✓ 学习乐观的归因风格
✓ 改变代际传递
```

---

## 🧪 使用示例

**输入：**
```
@ant 我总是把失败归因于自己
```

**输出：**
```
🎯 归因风格分析

【你的归因】
失败 = 自己的错（内在）

【乐观归因】
失败 = 外在 + 暂时 + 特定
• 可能是环境原因
• 只是暂时的
• 只是这件事

【训练方法】
1. 记录发生的事
2. 识别自己的归因
3. 调整为乐观归因
4. 重复练习

【关键洞察】
孩子每天从父母身上学到各种事件的因果分析，
尤其是妈妈的。如果你是乐观的，孩子也会是乐观的。
```

---

## 📚 核心理念

**关键洞察：**
1. 归因风格可以训练
2. 受父母影响
3. 可以代际传递

---

*孩子的解释风格会受到三种因素的影响，
其中最重要的是从父母身上学到的因果分析。* 🎯
