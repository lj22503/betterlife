# 非暴力沟通实践器 💗

**基于《非暴力沟通：让表达温柔有力》（马歇尔·卢森堡）**

---

## 📋 功能描述

帮助用户学习非暴力沟通四要素（观察/感受/需要/请求），让表达温柔有力。

**适用场景：**
- 想学习非暴力沟通
- 沟通中有冲突
- 想提升表达能力
- 改善人际关系

**边界条件：**
- 区分观察和评论
- 直接说出需要
- 请求而非命令

---

## 🎯 核心功能

### 1. 非暴力沟通四要素

**四要素过程：**
```
1. 观察：什么是我的观察
   （看、听、回忆、想）到的具体行为
   
2. 感受：我的感受如何
   （情感而非思想）
   
3. 需要：哪些需要导致那样的感受
   （需要或价值，而非偏好）
   
4. 请求：我的请求是什么
   （清楚地请求，而非命令）
```

### 2. 表达模式

**诚实地表达自己：**
```
当我（看、听、想到我看到的/听到的）……
我感到……
因为我需要/看重……
你是否愿意……？
```

**关切地倾听他人：**
```
当你（看、听、想到你看到的/听到的）……
你感到……吗？
因为你需要/看重……
所以，你想……
```

### 3. 四种选择

**听到不中听的话时的四种选择：**
```
1. 责备自己
2. 指责他人
3. 体会自己的感受和需要
4. 体会他人的感受和需要
```

**推荐选择 3 和 4**

### 4. 异化的沟通方式

**避免的沟通方式：**
```
• 道德评判：将不符合价值观的人看作不道德的
• 进行比较：蒙蔽对人对己的爱意
• 回避责任：淡化对自己的责任意识
• 强人所难：认为"某人应当受到惩罚"
```

---

## ⚠️ 常见错误

**错误 1：混淆观察和评论**
```
问题：
• "你总是迟到"（评论）
• "你从不关心我"（评论）

解决：
✓ 区分观察和评论
✓ "这周你有三次会议迟到"（观察）
✓ 不主张绝对化的结论
✓ 在特定的时间和情境中进行观察
```

**错误 2：用批评提出主张**
```
问题：
• "你太自私了"
• 通过批评来提出主张

解决：
✓ 直接说出我们的需要
✓ 对他人的指责、批评反映了我们的需要和价值观
✓ 直接说出需要，其他人较有可能作出积极的回应
```

## 🧪 使用示例

**输入：**
```
@ant 我老公总是不关心我，怎么办？
```

**输出：**
```
💗 非暴力沟通引导

【四要素表达】

❌ 暴力沟通：
"你总是不关心我"（评论）

✅ 非暴力沟通：
1. 观察："这周你有三个晚上都在工作，没有和我说话"
2. 感受："我感到孤单和难过"
3. 需要："因为我需要陪伴和关爱"
4. 请求："你是否愿意每天晚上花 30 分钟和我聊聊天？"

【关键洞察】
不区分观察和评论，人们将倾向于听到批评，并反驳我们。
直接说出我们的需要，其他人就较有可能作出积极的回应。
```

## 📚 核心理念

**关键洞察：**
1. 非暴力沟通四要素：观察/感受/需要/请求
2. 区分观察和评论
3. 直接说出需要
4. 请求而非命令

**健康公式：**
```
非暴力沟通 = 观察 + 感受 + 需要 + 请求
```
