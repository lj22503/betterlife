# 自我状态识别器 🎭

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户识别当前处于哪种自我状态（儿童/父母/成人），从痛苦走向自由。

**适用场景：**
- 感到痛苦纠结
- 情绪化反应
- 挑剔别人或自己
- 想进入成人状态

**边界条件：**
- 三种状态都是正常的
- 成人状态是唯一能学习的状态
- 目标是适时调动成人状态

---

## 🎯 核心功能

### 1. 三种自我状态

**儿童自我状态：**
```
• 下意识用童年习惯反应
• 像孩子一样的感受和行为
• 顺从、依赖、取悦
• 体验过去的情绪
• 学不到新东西
```

**父母自我状态：**
```
• 挑剔别人或自己
• 重复父母的是非观
• 审判、批评、教育
• 自我批判
• 学不到新东西
```

**成人自我状态：**
```
• 摆脱情绪化控制
• 合理计划、考虑、决定、行动
• 用知识和技能解决当下问题
• 唯一能学到新知识的状态
```

### 2. 状态识别检查清单

**你在儿童状态吗？**
- [ ] 顺从别人的期望
- [ ] 依赖别人的肯定
- [ ] 取悦别人
- [ ] 感到无助、弱小

**你在父母状态吗？**
- [ ] 挑剔别人的缺点
- [ ] 批评、指责别人
- [ ] 自我批判、自责
- [ ] 说"应该""必须"

**你在成人状态吗？**
- [ ] 理性分析当前情况
- [ ] 用知识和技能解决问题
- [ ] 不为情绪所控制
- [ ] 学习新的东西

---

## ⚠️ 常见错误

**错误 1：否定儿童和父母状态**
```
问题：
• "儿童状态不好，我要避免"
• "父母状态是错的"

解决：
✓ 三种状态都是正常的
✓ 关键是适时调动成人状态
✓ 儿童和父母状态也有价值
```

**错误 2：以为自己在成人状态**
```
问题：
• 实际上在儿童状态
• 以为自己很理性

解决：
✓ 诚实觉察自己的状态
✓ 使用检查清单识别
✓ 成人状态是唯一能学习的状态
```
