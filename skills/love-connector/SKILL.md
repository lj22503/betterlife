# 爱情绪连接者 💗

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户连接爱情绪，感受与他人之间的爱意涌现，建立亲密、安全、可靠和信任的关系。

**适用场景：**
- 想感受爱
- 想爱别人
- 建立亲密关系
- 提升信任感

**边界条件：**
- 爱不是一种单一的积极情绪
- 而是所有积极情绪的集合
- 需要社会关系

---

## 🎯 核心功能

### 1. 爱的特征

**爱与其他积极情绪的关系：**
```
• 爱不是一种单一的积极情绪
• 而是上述的所有
• 包括喜悦、感激、宁静、兴趣、希望、自豪、逗趣、激励和敬佩
• 爱之所以被称作是一件多彩的事物，是有道理的
```

### 2. 爱的生化反应

**身体变化：**
```
• 提高了催产素（一种健康的神经肽）的水平
• 提高了黄体酮（一种天然孕激素）的水平
• 这些都是与终身的牵绊、信任以及亲密相联系的生化反应
```

### 3. 爱的感受

**身体感受：**
```
• 在社会关系中感觉到亲近、安全、可靠和信任
• 某一种人际关系引发了某种或某几种其他形式的积极情绪
• 情不自禁地靠向心爱的人
• 肯定他的与众不同
• 享受挚爱的人陪伴的感觉
```

### 4. 培养方法

**微习惯：**
```
• 与心爱的人在一起
• 肯定他人的与众不同
• 珍惜他们
• 在他们的怀抱中笑容满溢
• 建立亲密而友好的联系
```

---

## ⚠️ 常见错误

**错误 1：认为爱是单一情绪**
```
问题：
• "爱就是一种感觉"
• 认为爱是单一的

解决：
✓ 爱不是一种单一的积极情绪
✓ 而是所有积极情绪的集合
✓ 包括喜悦、感激、宁静等 9 种
```
