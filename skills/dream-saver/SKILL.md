# 梦想储蓄罐 💰

**基于《小狗钱钱》核心理念**

---

## 📋 功能描述

帮助用户将抽象的梦想转化为具体的储蓄目标，通过可视化方法激励持续储蓄。

**适用场景：**
- 儿童/青少年财商教育
- 成年人目标管理
- 家庭财务规划
- 梦想可视化激励

**边界条件：**
- 不提供投资建议
- 聚焦目标设定和储蓄规划
- 适合 10 岁以上人群

---

## 🎯 核心功能

### 1. 梦想清单创建

帮助用户列出并排序梦想清单。

**输入：**
- 年龄
- 梦想列表（最多 10 个）
- 每个梦想的预算
- 优先级排序

**输出：**
```json
{
  "dreams": [
    {
      "id": 1,
      "name": "变速自行车",
      "cost": 2000,
      "priority": "高",
      "deadline": "2026-06-01",
      "monthly_saving": 500
    },
    {
      "id": 2,
      "name": "美国交换学生",
      "cost": 50000,
      "priority": "中",
      "deadline": "2027-07-01",
      "monthly_saving": 3000
    }
  ],
  "total_cost": 52000,
  "total_monthly_saving": 3500
}
```

### 2. 梦想储蓄罐分配

为每个梦想创建独立的储蓄罐。

**原则：**
- 一个梦想一个储蓄罐
- 专款专用，不混用
- 可视化进度（贴纸/图表）

**输出：**
```json
{
  "dream": "变速自行车",
  "target": 2000,
  "current": 500,
  "progress": 0.25,
  "remaining": 1500,
  "monthly_target": 500,
  "months_remaining": 3,
  "status": "进行中",
  "visual": "🟩🟩⬜⬜"
}
```

### 3. 收入分配建议

将收入分配到不同储蓄罐。

**50/30/20 法则（儿童版）：**
- 50% 梦想储蓄（长期目标）
- 30% 日常开销（零花钱）
- 20% 金鹅账户（投资本金）

**输出：**
```json
{
  "income": 1000,
  "allocation": {
    "dream_saving": {
      "amount": 500,
      "percentage": 0.5,
      "purpose": "梦想目标储蓄"
    },
    "daily_spending": {
      "amount": 300,
      "percentage": 0.3,
      "purpose": "日常零花"
    },
    "golden_goose": {
      "amount": 200,
      "percentage": 0.2,
      "purpose": "投资本金（养鹅生蛋）"
    }
  }
}
```

### 4. 梦想进度追踪

追踪每个梦想的储蓄进度。

**输出：**
```markdown
# 梦想进度报告

## 🚴 变速自行车
目标：2000 元 | 已存：500 元 | 进度：25%
每月需存：500 元 | 剩余：3 个月
状态：🟢 按计划进行

## ✈️ 美国交换学生
目标：50000 元 | 已存：5000 元 | 进度：10%
每月需存：3000 元 | 剩余：15 个月
状态：🟡 需要加速
```

## 🧪 使用示例

### 示例 1：12 岁小学生

```
@ant 帮我设置梦想储蓄罐

信息：
- 年龄：12 岁
- 月收入：500 元（零花钱）
- 梦想：
  1. 变速自行车（2000 元）
  2. 全套漫画（800 元）
  3. 夏令营（3000 元）
```

**预期输出：**
- 梦想清单排序（按优先级）
- 每月储蓄分配：250 元梦想 +150 元零花 +100 元金鹅
- 每个梦想的达成时间

### 示例 2：25 岁职场新人

```
@ant 我想存钱买笔记本电脑和去旅行

信息：
- 年龄：25 岁
- 月收入：8000 元
- 梦想：
  1. 笔记本电脑（10000 元）
  2. 欧洲旅行（30000 元）
  3. 应急基金（50000 元）
```

**预期输出：**
- 梦想优先级建议
- 月度储蓄计划
- 达成时间预测

## ⚠️ 错误处理

| 错误码 | 含义 | 处理建议 |
|--------|------|----------|
| 400 | 参数缺失 | 检查必填字段（年龄、梦想列表） |
| 400 | 梦想过多 | 建议不超过 10 个梦想 |
| 400 | 预算不合理 | 梦想预算应为正数 |
| 500 | 内部错误 | 重试或联系开发者 |
