# 财务思维训练营 💰

**基于《从总账到总监》核心理念**

---

## 📋 功能描述

帮助用户建立财务思维，从财务视角看业务、职场和人生决策。

**适用场景：**
- 财务人士职业进阶
- 业务人士学财务
- 人生财务决策
- 投资理财规划

---

## 🎯 核心功能

### 1. 职场进阶三要素

**升得快靠三样：**
- 能力 (Capabilities)
- 关系 (Networking)
- 可视度 (Visibility)

### 2. 关注圈与影响圈

**能量聚焦：**
- 关注圈：你关心的事
- 影响圈：你能控制的事

### 3. 价值创造矩阵

**四象限：**
- 身边的金子（价值大 + 难度小）
- 地上的果子（价值小 + 难度小）
- 该打的硬仗（价值大 + 难度大）
- 关键场合的关键支持

### 4. 管理者三大杠杆

- 营运杠杆（机器代替人）
- 资本杠杆（善用外部资本）
- 资源杠杆（带团队完成更大绩效）
