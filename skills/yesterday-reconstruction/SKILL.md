# 昨日重现练习器 📅

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

引导用户回顾过去一天的时间安排，了解真实情绪量比，计算积极率。

**适用场景：**
- 回顾昨天的情绪
- 了解真实情绪量比
- 计算每日积极率
- 情绪自我觉察

**边界条件：**
- 如实回顾，不评判
- 分成 10-30 个片段
- 每个片段 10 分钟 -2 小时

---

## 🎯 核心功能

### 1. 方法步骤

**步骤 1：回顾一天**
```
仔细回想昨天一天中的所有片段
描绘你在一天中的情绪细节
```

**步骤 2：分割片段**
```
将一天分成一系列片段
每个片段 10 分钟到 2 小时不等
通常有 10 到 30 个片段
```

**步骤 3：描述片段**
```
给每个片段标上数字
附上简短的描述性标签
如"与孩子们吃早餐"、"上班"等
```

**步骤 4：记录时间和活动**
```
写下每个片段开始和结束的大致时间
不要有时间上的空隙或重叠
```

**步骤 5：情绪评分**
```
针对每个片段完成积极情绪自测量表
对积极情绪和消极情绪进行评分
0=一点都没有 1=有一点 2=中等 3=很多 4=非常多
```

**步骤 6：计算积极率**
```
积极情绪事件赋值 2 分及以上的数量
比
消极情绪事件赋值 1 分及以上的数量
```

---

## ⚠️ 常见错误

**错误 1：片段分得太粗**
```
问题：
• 只分几个大片段
• 忽略细节

解决：
✓ 分成 10-30 个片段
✓ 每个片段 10 分钟 -2 小时
✓ 不要有空隙或重叠
```

**错误 2：评判自己的情绪**
```
问题：
• "我不应该有这种情绪"
• 评判消极情绪

解决：
✓ 如实回顾，不评判
✓ 所有情绪都是正常的
✓ 关键是了解真实量比
```
