# 涛动周期论 Skill 套件 🌊

**基于周金涛《涛动周期论》的经济周期与财富规划技能**

---

## 📖 简介

这套技能将周金涛的康波周期理论转化为实用的 AI 工具，帮助用户理解经济周期，把握人生财富机会，实现财富阶层跃迁。

**核心理念：**
- 🌊 **康波周期** —— 60 年经济周期决定人生财富命运
- 🏠 **房地产周期** —— 20 年轮回，人生两次买房
- 📦 **库存周期** —— 3 年短期波动，18 次交易机会
- 💰 **人生 3 次机会** —— 抓住 1 次成中产，抓住 3 次成富人
- 📊 **大类资产配置** —— 周期决定配置，配置决定财富

## 📚 周金涛周期理论核心

### 康德拉季耶夫周期（康波）

**60 年一轮**，分为四个阶段：

| 阶段 | 时长 | 特征 | 最优资产 |
|------|------|------|----------|
| 回升期 | 15 年 | 经济复苏 | 股票 |
| 繁荣期 | 15 年 | 经济繁荣 | 大宗商品 |
| 衰退期 | 15 年 | 经济衰退 | 现金 |
| 萧条期 | 15 年 | 经济萧条 | 现金 + 黄金 |

**当前定位：**
- 第五次康波：1982-2042
- 当前阶段：萧条期（2018-2030）
- 下一阶段：回升期（2030-2045）

### 房地产周期

**20 年轮回**，人生可碰到 2 次：

- 首次置业：27 岁左右（结婚）
- 改善置业：42 岁左右（二次置业）
- 消费高峰：46 岁
- 46 岁后：消费下降，转向医疗养老

### 库存周期（基钦周期）

**约 3 年（36 个月）**，人生可碰到 18 次：

- 补库存期（12-18 个月）：需求上升，生产扩张
- 去库存期（12-18 个月）：需求下降，生产收缩

### 三周期嵌套

```
人生 = 1 次康波（60 年）
     = 3 次房地产周期（20 年×3）
     = 6 次固定资产投资周期（10 年×6）
     = 18 次库存周期（3 年×18）
```

## ⚠️ 重要提示

**周金涛核心观点：**

1. **人生财富不是靠工资，而是靠资产价格投资**
2. **人生只有 3 次财富机会，抓住 1 次即可成为中产**
3. **2018-2030 年是康波萧条期，以防守为主**
4. **萧条期最佳配置：现金 + 黄金**
5. **任何机会都是赌博，没有完全确定的机会**
