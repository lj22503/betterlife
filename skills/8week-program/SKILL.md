# 8 周正念计划器 📅

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

为用户提供 8 周正念禅修系统训练计划，每周包含禅修和习惯破除练习。

**适用场景：**
- 系统学习正念
- 8 周训练计划
- 正念课程跟随
- 深度正念练习

**边界条件：**
- 每周训练 6 天
- 每天至少 2 次
- 循序渐进

---

## 🎯 核心功能

### 8 周训练计划

**第 1 周：入门**
```
禅修练习：
• 葡萄干禅修
• 对一项日常活动的正念意识（刷牙/淋浴等）
• 每天 2 次身体和呼吸正念禅修（8 分钟）

习惯破除：
• 选择一项日常活动
```

**第 2 周：身体扫描**
```
禅修练习：
• 身体扫描练习（每天至少 2 次，每周 6 天）
• 实施另一项日常活动的正念练习
• 3 分钟呼吸空间（需要时）

习惯破除：
• 本周至少散步一次（至少 15 分钟）
```

**第 3 周：正念运动**
```
禅修练习：
• 8 分钟正念运动练习
• 8 分钟呼吸与身体禅修
• 3 分钟呼吸空间（每天 2 次）

习惯破除：
• "电视评估"
```

**第 4 周：声音和思想**
```
禅修练习：
• 8 分钟呼吸与身体禅修（每天 2 次）
• 8 分钟声音和思想禅修（每天 2 次）
• 3 分钟呼吸空间（每天 2 次）

习惯破除：
• 选择一项习惯破除练习
```

**第 5 周：困境探索**
```
禅修练习：
• 8 分钟呼吸与身体禅修
• 8 分钟声音与思想禅修
• 10 分钟困境探索禅修
• 呼吸空间禅修

习惯破除：
• 参见具体描述
```

**第 6 周：友善禅修**
```
禅修练习：
• 10 分钟友善禅修（6 天）
• 3 分钟呼吸空间（每天 2 次）

习惯破除：
• 选择一项习惯破除练习
```

**第 7 周：自主选择**
```
禅修练习：
• 选择两项训练（20-30 分钟）
• 选择理由：有益滋养或未能完全掌握
• 3 分钟呼吸空间（每天 2 次）

习惯破除：
• 选择一项习惯破除练习
```

**第 8 周：整合**
```
禅修练习：
• 整合前 7 周的练习
• 制定个人练习计划
• 3 分钟呼吸空间（每天 2 次）

习惯破除：
• 持续日常活动正念
```

## ⚠️ 常见错误

**错误 1：急于求成**
```
问题：
• "我想马上完成 8 周"
• 跳过前面的练习

解决：
✓ 循序渐进
✓ 每周完成当周练习
✓ 不要跳过基础
```

**错误 2：追求完美**
```
问题：
• "我必须每天完成"
• 错过一天就放弃

解决：
✓ 每周训练 6 天即可
✓ 错过一天没关系
✓ 继续练习
```
