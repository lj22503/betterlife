# 习惯破除训练器 🔨

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行习惯破除练习，识别自动模式，培养正念意识。

**适用场景：**
- 破除自动习惯
- 提升正念意识
- 识别行动模式
- 减少自动化反应

**边界条件：**
- 每周至少 1 次
- 选择日常活动
- 不受打扰

---

## 🎯 核心功能

### 1. 日常活动正念

**可选择的日常活动：**
```
• 刷牙
• 淋浴
• 散步（至少 15 分钟）
• 喝茶/咖啡
• 洗碗
• 叠衣服
```

**练习要点：**
```
1. 选择一项日常活动
2. 用正念意识进行
3. 注意每个动作
4. 注意自动模式
```

### 2. 电视评估

**步骤：**
```
1. 打开电视前停顿
2. 询问自己：我真的想看吗？
3. 还是只是习惯？
4. 做出有意识的选择
```

### 3. 排队禅修

**步骤：**
```
1. 排队时检查内心状况
2. 询问自己：
   • 我的脑海中正在发生什么？
   • 我的身体中有什么感觉？
   • 我注意到了什么情绪反应？
3. 识别"行动"模式
4. 区分基本痛苦和次要痛苦
5. 承认沮丧，不试图赶走
6. 站直、呼吸、包容、活在当下
```

### 4. 自动模式识别

**行动模式特征：**
```
• "抓紧时间"的欲望
• 因进展缓慢而沮丧
• 不断思考是否应该换队
• 处于自动状态
```

**存在模式特征：**
```
• 承认不快
• 不喜欢它没有关系
• 接受沮丧
• 活在当下
```

---

## ⚠️ 常见错误

**错误 1：评判自动模式**
```
问题：
• "我不应该有这种反应"
• 责备自己

解决：
✓ 这不是一个错误
✓ 意识已经做了最大的努力
✓ 正念让你认清本来就会有不快的经历
```

**错误 2：试图赶走沮丧**
```
问题：
• "我要摆脱沮丧"
• 想尽办法赶走

解决：
✓ 接受排队时的沮丧
✓ 承认它给你带来不快
✓ 继续恶化并失去控制的可能性已大大降低
```
