# 大类资产轮动器 📊

**基于《回归投资常识》（齐俊杰）核心理念**

---

## 📋 功能描述

帮助用户理解大类资产轮动规律（债券→股票→大宗商品→现金），基于经济周期进行资产配置。

**适用场景：**
- 经济周期判断
- 大类资产配置
- 美林投资时钟应用
- 周期六阶段定位

---

## 🎯 核心功能

### 1. 美林投资时钟改良

**三指标六阶段：**
- 先行指标（政策）：PMI、社融
- 同步指标（经济）：工业企业利润、工业增加值
- 滞后指标（通胀）：PPI

**六阶段：**
1. 政策紧 + 经济下 + 通胀高→放缓初期
2. 政策宽 + 经济下 + 通胀下→放缓后期（债券牛市）
3. 政策宽 + 经济上 + 通胀下→复苏初期（股票牛市）
4. 政策宽 + 经济上 + 通胀上→复苏后期（股票最佳）
5. 政策紧 + 经济上 + 通胀上→过热期（大宗商品）
6. 政策紧 + 经济下 + 通胀高→滞胀期（现金/债券）

### 2. 资产轮动规律

**轮动顺序：**
债券（放缓期）→股票（复苏期）→大宗商品（过热期）→现金（滞胀期）

### 3. 当前周期定位

基于三指标拐点定位当前周期阶段。
