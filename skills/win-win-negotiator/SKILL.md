# 双赢谈判器 🤝

**基于《合作的进化》（罗伯特·阿克塞尔罗德）**

---

## 📋 功能描述

帮助用户引导合作，达到双赢，促进合作关系。

**适用场景：**
- 引导合作，达到双赢
- 改善合作关系
- 提升谈判能力

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 评估当前关系
2. 选择合适策略
3. 实施并观察反应
4. 调整策略促进合作
```

### 2. 一报还一报策略

**四个特性：**
```
1. 善良性：只要对方合作你就合作
2. 报复性：面对背叛你是可激怒的
3. 宽容性：在反击之后你是宽容的
4. 清晰性：行为简单清晰，使对方能适应
```

---

## ⚠️ 常见错误

**错误 1：首先背叛**
```
问题：
• 背叛总是能得到更多
• 不关心未来利益

解决：
✓ 不要首先背叛
✓ 如果未来是重要的，背叛会招来无休止的惩罚
✓ 合作的进化要求个体有足够大的机会再次相遇
```

**错误 2：耍小聪明**
```
问题：
• 采用复杂到不可理解的策略
• 显得对对方不反应

解决：
✓ 行为要简单清晰
✓ 复杂到不可理解是非常危险的
✓ 如果你是不反应的，对方就受不到与你合作的激励
```

## 📚 核心理念

**关键洞察：**
1. 引导合作，达到双赢
2. 不要妒忌对方的成功
3. 不要首先背叛
4. 要对合作和背叛都作出回报
5. 不要耍小聪明

**健康公式：**
```
合作进化 = 善良性 + 报复性 + 宽容性 + 清晰性
```
