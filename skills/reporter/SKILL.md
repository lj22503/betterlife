# 汇报器 📊

**基于《脱不花：沟通的方法》**

---

## 📋 功能描述

帮助用户让方案被人重视，实现高效能沟通。

**适用场景：**
- 让方案被人重视
- 改善沟通效果
- 提升职场竞争力

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 倾听理解（结构化倾听）
2. 识别风格（四种动物）
3. 积极回应（换口径/时间/场合/角色）
4. 导向行动（建设性）
```

---

## ⚠️ 常见错误

**错误 1：只听表面**
```
问题：
• 没有分拣情绪/事实/期待
• 被情绪带偏

解决：
✓ 在脑中画三个框：情绪/事实/期待
✓ 听到情绪要隔离并安抚
✓ 从事实中判断客观信息
✓ 明确对方的核心期待
```

**错误 2：傻瓜式选择**
```
问题：
• 以为必须在"说真话"和"维持友情"之间二选一

解决：
✓ 拒绝傻瓜式选择
✓ 既能百分之百坦诚，又能百分之百尊重对方
✓ 寻找兼顾两者的方法
```

## 📚 核心理念

**关键洞察：**
1. 让方案被人重视
2. 结构化倾听：情绪/事实/期待三框
3. 四种沟通风格：老虎/孔雀/猫头鹰/考拉
4. 沟通三大原则：开放性/目标感/建设性

**健康公式：**
```
高效沟通 = 结构化倾听 + 沟通风格识别 + 积极回应 + 三大原则
```
