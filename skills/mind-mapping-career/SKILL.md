# 职业思维导图 🗺️

**基于《人生设计课》**

---

## 📋 功能描述

帮助用户梳理职业发展路径，实现职业成功。

**适用场景：**
- 梳理职业发展路径
- 职业规划
- 职业转型

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 评估当前状态
2. 探索多种可能
3. 设计原型体验
4. 选择并行动
```

---

## ⚠️ 常见错误

**错误 1：只有一个计划**
```
问题：
• 只设计一个人生计划
• 没有备选方案

解决：
✓ 设计 3 个版本的人生计划
✓ 探索多种可能性
✓ 保持灵活性
```

**错误 2：害怕失败**
```
问题：
• 不敢尝试新事物
• 害怕失败

解决：
✓ 设计原型体验
✓ 小步试错
✓ 从失败中学习
```
