# 感激情绪培养器 🙏

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户培养感激情绪，滋养人际关系，使其更坚固、更紧密。

**适用场景：**
- 想学会感恩
- 想表达感激
- 滋养人际关系
- 提升幸福感

**边界条件：**
- 感激需要有人为你做好事
- 可以用语言、好意、礼物表达
- 滋养自己也滋养关系

---

## 🎯 核心功能

### 1. 感激的触发条件

**引发感激的条件：**
```
• 你刚刚意识到
• 有人煞费苦心地为你做了一些好事
• 你感到感激或感谢
• 深深地为某人或某事赞赏
```

### 2. 表达方式

**表达感激的形式：**
```
• 用语言表达
• 用好意表达
• 用礼物形式表达
```

### 3. 感激的好处

**对关系的影响：**
```
• 滋养我们拥有的关系
• 帮助关系变得更坚固
• 帮助关系变得更紧密
• 提高催产素水平（健康的神经肽）
```

### 4. 培养方法

**微习惯：**
```
• 每天记录 3 件感激的事
• 向帮助过你的人表达感谢
• 将看似平凡的事件当成福气
• 珍惜最珍贵的礼物
```

---

## ⚠️ 常见错误

**错误 1：忽视小善意**
```
问题：
• "这没什么好感激的"
• 忽视小的好事

解决：
✓ 善意在人际关系中很普遍
✓ 有时甚至褪色成背景
✓ 将看似平凡的事件当成福气
```
