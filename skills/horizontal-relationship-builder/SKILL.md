# 横向关系构建器 ⚖️

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户建立平等的横向关系，摆脱纵向关系的操控。

**适用场景：**
- 感到被操控
- 上下级关系困扰
- 想建立平等关系

**边界条件：**
- 不否定所有权威
- 建立平等而非对抗
- 坚持自己应有的主张

---

## 🎯 核心功能

### 1. 横向关系 vs 纵向关系

**纵向关系（不健康）：**
```
• 上下级关系
• 表扬/批评操控
• 不平等
```

**横向关系（健康）：**
```
• 平等关系
• 感谢/鼓励
• 互相尊重
```

### 2. 鼓励法（横向关系）

```
✓ "谢谢"（表达感谢）
✓ "我很高兴"（表达喜悦）
✓ "帮了大忙"（表达价值）
```

---

## ⚠️ 常见错误

**错误 1：把平等当对抗**
```
问题：
• "我要反抗所有权威"
• 把横向关系当成对抗

解决：
✓ 横向关系是平等不是对抗
✓ 尊重差异但保持平等
✓ 坚持主张但不攻击
```

**错误 2：无法接受感谢**
```
问题：
• 不习惯被感谢
• 觉得感谢是客套

解决：
✓ 学习接受感谢
✓ 理解感谢是横向关系的表达
✓ 用感谢代替表扬
```
