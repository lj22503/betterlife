# 横向关系构建器 ⚖️

**基于《被讨厌的勇气》（阿德勒心理学）**

---

## 📋 功能描述

帮助用户建立平等的横向关系，摆脱纵向关系的操控。

**适用场景：**
- 感到被操控
- 上下级关系困扰
- 想建立平等关系

---

## 🎯 核心功能

### 1. 横向关系 vs 纵向关系

**纵向关系（不健康）：**
```
• 上下级关系
• 表扬/批评操控
• 不平等
```

**横向关系（健康）：**
```
• 平等关系
• 感谢/鼓励
• 互相尊重
```

### 2. 鼓励法（横向关系）

```
✓ "谢谢"（表达感谢）
✓ "我很高兴"（表达喜悦）
✓ "帮了大忙"（表达价值）
```

---

## 🧪 使用示例

**输入：**
```
@ant 我老板总用表扬操控我
```

**输出：**
```
⚖️ 关系分析

【纵向关系特征】
• 表扬/批评 = 操控
• 上下级不平等

【横向关系建议】
✓ 建立平等关系
✓ 用感谢代替表扬
✓ 坚持自己应有的主张

【关键洞察】
表扬或者批评他人只有"用糖还是用鞭子"的区别，
其背后的目的都是操纵。
```

---

## 📚 核心理念

**关键洞察：**
1. 表扬背后是操控
2. 建立平等的横向关系
3. 用感谢代替表扬

---

*建立平等的横向关系，
摆脱纵向关系的操控。* ⚖️
