# 康波周期定位器 🌊

**基于周金涛《涛动周期论》核心理念**

---

## 📋 功能描述

帮助用户识别当前康德拉季耶夫周期（康波）位置，规划人生财富机会，把握 60 年经济周期中的财富轨迹。

**适用场景：**
- 人生财富规划
- 大类资产配置
- 经济周期判断
- 长期投资决策

**边界条件：**
- 不提供具体投资建议
- 基于历史规律，非精确预测
- 适合 18 岁以上人群

---

## 🎯 核心功能

### 1. 康波周期定位

识别当前康波位置（回升/繁荣/衰退/萧条）。

**康波周期：** 60 年一轮（回升 15 年 + 繁荣 15 年 + 衰退 15 年 + 萧条 15 年）

**当前定位：**
- 第五次康波：1982-2042
- 当前阶段：萧条期（2018-2030）

**输出：**
```json
{
  "current_kondratieff": 5,
  "kondratieff_period": "1982-2042",
  "current_phase": "萧条期",
  "phase_period": "2018-2030",
  "next_phase": "回升期",
  "next_phase_start": 2030,
  "insight": "你正处于第五次康波萧条期，未来 10 年以防守为主"
}
```

### 2. 人生财富机会识别

基于康波理论识别人生财富机会。

**核心观点：**
- 人生财富机会只有 3 次
- 每次机会间隔约 10 年
- 抓住 1 次即可成为中产阶级

### 3. 康波阶段资产配置

基于康波阶段推荐资产配置策略。

**各阶段策略：**
- 回升期：股票 > 房地产 > 大宗商品 > 债券
- 繁荣期：大宗商品 > 股票 > 房地产 > 现金
- 衰退期：现金 > 黄金 > 债券 > 股票
- 萧条期：现金 + 黄金 > 债券 > 股票 > 大宗商品

### 4. 康波价格波动规律

识别不同资产在康波各阶段的价格表现。

**规律：**
- 繁荣期：价格平稳（100% 概率）
- 回升期：价格平稳（67% 概率）
- 衰退期：价格剧烈波动（100% 概率）
- 萧条期：价格冲高回落（100% 概率）

**输出：**
```json
{

    "cash": "保值（实际利率）"
  },
  "key_insight": "萧条期最佳配置：现金 + 黄金"
}
```

## 🧪 使用示例

### 示例 1：1985 年后出生

```
@ant 我 1990 年出生，人生财富机会在哪？
```

**预期输出：**
- 第一次机会：2019 年（29 岁）- 已过去
- 第二次机会：2029 年（39 岁）- 准备中
- 第三次机会：2039 年（49 岁）- 未来
- 当前策略：学习积累，等待 2029 年机会

### 示例 2：当前资产配置

```
@ant 现在是 2026 年，如何配置资产？
```

**预期输出：**
- 康波位置：第五次康波萧条期（2018-2030）
- 最佳配置：40% 现金 +30% 黄金 +20% 债券 +10% 风险资产
- 避免：大宗商品、高杠杆、非核心房产

### 示例 3：房地产周期

```
@ant 现在适合买房吗？
```

**预期输出：**
- 房地产周期：20 年一轮
- 下一轮启动：2020-2025 年（萧条期启动，相对较弱）
- 建议：核心区域可考虑，非核心区域观望

## ⚠️ 错误处理

| 错误码 | 含义 | 处理建议 |
|--------|------|----------|
| 400 | 参数缺失 | 检查必填字段（出生年份） |
| 400 | 年龄无效 | 年龄应在 18-100 之间 |
| 400 | 风险承受无效 | 应为低/中/高 |
| 500 | 内部错误 | 重试或联系开发者 |

## 📚 政策依据

**核心理论：**
- 《涛动周期论》- 周金涛
- 康德拉季耶夫周期（Kondratieff Wave）
- 三周期嵌套理论（康波 + 房地产 + 库存）

**关键概念：**
- 康波周期：60 年一轮（回升/繁荣/衰退/萧条）
- 人生财富机会：3 次/生
- 房地产周期：20 年轮回
- 库存周期：3 年轮回
- 萧条期最佳配置：现金 + 黄金
