# 正念运动引导器 🚶

**基于《5-8 周正念禅修练习》+《十分钟冥想》**

---

## 📋 功能描述

引导用户进行正念运动练习，包括正念伸展、行走冥想、跑步冥想，培养移动中的觉察。

**适用场景：**
- 正念入门练习
- 身体伸展
- 提升身体觉察
- 久坐后活动
- 行走/跑步时的正念

**边界条件：**
- 动作缓慢专注
- 不追求幅度
- 注意身体感受
- 对周围保持觉醒

---

## 🎯 核心功能

### 1. 正念伸展

**准备姿势：**
```
• 光脚或穿袜子站立
• 双脚分开与胯等宽
• 膝盖不要闭锁
• 双腿微微弯曲
• 双脚相互平行
```

**动作序列：**
```
1. 抬举双臂（吸气抬起，呼气放下）
2. 摘水果（伸展手臂，对侧脚后跟离地）
3. 侧弯（形成新月形状）
4. 转肩（根据呼吸频率转动）
```

**关键要点：**
```
• 动作缓慢专注
• 注意肌肉动作
• 弯曲程度不重要
• 重要的是注意力
```

### 2. 行走冥想（来自《十分钟冥想》）

**准备：**
```
在开始行走之前，留意身体的感觉
你是感到沉重还是轻盈，僵硬还是放松？
不要急着回答，先花上几秒时间留意自己的姿势
```

**练习步骤：**
```
1. 留意周围发生的事情（30 秒）
   - 路人、商店橱窗、汽车、广告
   - 田野、树木、动物（郊区）
   - 颜色、形状、动态或静态

2. 留意声音（30 秒）
   - 脚踩到人行道上的声音
   - 路过的车辆的声音
   - 树上的鸟叫声

3. 留意气味（30 秒）
   - 花香、须后水
   - 汽车尾气、汽油味
   - 食物和饮料的气味

4. 留意躯体感觉或情感（30 秒）
   - 温暖的阳光、凉爽的雨
   - 脚跟触到地面时的感觉
   - 胳膊在身体两侧摆动时的感觉
```

**行走觉察：**
```
• 留意重量如何节奏平稳地从右侧转到左侧
• 避免人为的速度调整
• 观察自己已经习惯了的走路方式和节奏
• 把脚跟触到地面时的触感当作觉醒起点
• 意识到心灵游离时，温和地将注意力带回
```

### 3. 跑步冥想（来自《十分钟冥想》）

**跑前准备：**
```
1. 了解自己的努力
   - 心里在想什么？
   - 感到焦虑、自信，还是完全无感？

2. 留意躯体感觉
   - 腿是否沉重
   - 肩膀是否紧张
   - 通体是否有轻盈感

3. 深呼吸集中注意力
   - 用鼻子吸气，用嘴巴呼气
   - 至少做 4-5 次
```

**跑步中觉察：**
```
1. 对周围保持强烈觉醒
   - 其他跑步者、汽车、公园、田野、建筑

2. 留意身体活动时的感受
   - 肌肉如何回应动作
   - 呼吸的快速变化

3. 留意心灵的回应
   - 是否有愉悦感
   - 是否有焦虑感
   - 心灵是非常繁忙还是非常安定

4. 留意稳定下来的节奏
   - 节奏是否舒服
   - 身体是否协调
   - 两条腿用力是否均匀
```

**疼痛处理：**
```
• 不要试图"摆脱"身体上的不适
• 看一看当你把注意力放在这种感受上时会发生什么
• 把你和疼痛视为一体
• 少想"我和我的疼痛"
• 多想"疼痛"带来的简单而直接的体验
• 靠近不适，充分感受它，鼓励它
```

**分程技巧：**
```
• 将跑步里程拆分成几个部分
• 按每步进行分程
• 按每条街、每千米进行分程
• 每 10 步、每 20 步、或每 100 步
• 有点像数自己的呼吸，有助于阻止心灵游离
```

## ⚠️ 常见错误

**错误 1：追求幅度**
```
问题：
• "我要弯得更低"
• 追求动作标准

解决：
✓ 弯曲程度并不重要
✓ 重要的是对移动的注意力
✓ 原地不动都没关系
```

**错误 2：动作过快**
```
问题：
• 快速完成动作
• 不注意感受

解决：
✓ 动作要缓慢
✓ 专心专注
✓ 跟踪身体感受
```

**错误 3：试图摆脱疼痛**
```
问题：
• "我要摆脱这个疼痛"
• 抗拒不适

解决：
✓ 不要试图摆脱身体上的不适
✓ 靠近不适，充分感受它
✓ 把你和疼痛视为一体
```

```
正念运动 = 准备姿势 + 缓慢动作 + 专注觉察 + 分程技巧
```
