# 正念运动引导器 🧘

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行正念运动练习，包括抬臂、摘水果、侧弯、转肩等动作。

**适用场景：**
- 正念入门练习
- 身体伸展
- 提升身体觉察
- 久坐后活动

**边界条件：**
- 动作缓慢专注
- 不追求幅度
- 注意身体感受

---

## 🎯 核心功能

### 1. 准备姿势

**站立姿势：**
```
• 光脚或穿袜子
• 双脚分开与胯等宽
• 膝盖不要闭锁
• 双腿微微弯曲
• 双脚相互平行
```

### 2. 抬举双臂

**步骤：**
```
1. 吸气时缓慢抬起双臂
2. 与地板保持平行
3. 继续吸气，双手超过头顶
4. 注意肌肉动作
5. 指尖轻轻推向天空
6. 双脚稳稳立于地板
7. 感受伸展感觉
8. 呼气时慢慢放下双臂
```

### 3. 摘水果

**步骤：**
```
1. 伸展每只胳膊和手掌
2. 像从树上摘高处的水果
3. 抬头向手指以外看去
4. 对侧脚后跟离开地面
5. 注意从手指到脚趾的感觉
6. 脚后跟回到地面
7. 放下手掌
8. 换另一侧重复
```

### 4. 侧弯

**步骤：**
```
1. 双手放在胯部
2. 身体向左侧倾斜
3. 胯部向右侧移动
4. 形成新月形状
5. 确保身体在一个平面内
6. 吸气恢复站立
7. 呼气向相反方向弯曲
```

### 5. 转肩

**步骤：**
```
1. 双臂被动下垂
2. 肩膀向耳朵方向抬起
3. 向后拉肩胛骨
4. 恢复自然状态
5. 向前挤压双肩
6. 根据呼吸频率决定速度
7. 沿一个方向然后反方向
```

---

## ⚠️ 常见错误

**错误 1：追求幅度**
```
问题：
• "我要弯得更低"
• 追求动作标准

解决：
✓ 弯曲程度并不重要
✓ 重要的是对移动的注意力
✓ 原地不动都没关系
```

**错误 2：动作过快**
```
问题：
• 快速完成动作
• 不注意感受

解决：
✓ 动作要缓慢
✓ 专心专注
✓ 跟踪身体感受
```
