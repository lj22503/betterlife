# 炎症评估 🔥

**基于池谷敏郎《抗炎生活》**

---

## 📋 功能描述

帮助用户评估体内慢性炎症程度，解读超敏 CRP 指标。

**适用场景：**
- 体检报告解读
- 慢性病风险评估
- 健康状态监测
- 抗炎效果追踪

**边界条件：**
- 不替代医疗诊断
- 需结合其他指标
- 个体差异需考虑

---

## 🎯 核心功能

### 1. 慢性炎症基础知识

**急性炎症 vs 慢性炎症：**
```
急性炎症：
• 突然显现
• 红、肿、热、痛
• 快速消退
• 身体防御反应

慢性炎症：
• 不易察觉
• 无明显症状
• 持续蔓延
• 加速老化、导致疾病
```

**为什么慢性炎症危险：**
```
• 无症状，容易被忽视
• 在体内"闷烧"
• 损伤 DNA
• 促进细胞异常分裂
• 最终导致重大疾病
```

### 2. 超敏 CRP 检测

**什么是 CRP：**
```
C 反应蛋白（C-reactive protein）
• 发炎时肝脏制造的蛋白质
• 随血液运送到全身
• 反映体内炎症程度
```

**评估标准：**
```
| CRP 值 (mg/dL) | 评估 | 建议 |
|---------------|------|------|
| ≤0.30 | 标准范围 | 保持 |
| 0.31-0.99 | 需注意 | 生活方式调整 |
| ≥1.00 | 异常 | 就医排查 |
```

**检测频率：**
```
• 健康人群：每年 1 次
• 需注意人群：每 6 个月 1 次
• 异常人群：遵医嘱
```

### 3. 慢性炎症自测

**生活炎症检查表：**

口腔健康：
- [ ] 牙龈出血
- [ ] 口臭
- [ ] 牙周病

生活习惯：
- [ ] 吸烟
- [ ] 饮酒过量
- [ ] 睡眠不足
- [ ] 压力过大
- [ ] 运动不足

饮食：
- [ ] 常吃加工食品
- [ ] 常吃油炸食品
- [ ] 常吃甜食
- [ ] 蔬菜摄入不足
- [ ] 鱼类摄入不足

身体状态：
- [ ] 容易疲劳
- [ ] 体重超标
- [ ] 腹部肥胖
- [ ] 皮肤问题
- [ ] 关节疼痛

**风险评估：**
```
• 0-5 项：低风险
• 6-10 项：中风险
• 11+ 项：高风险
```

### 4. 慢性炎症来源

**主要来源：**
```
1. 肥胖（内脏脂肪）
   • 脂肪组织分泌炎症物质
   • 最大炎症来源

2. 口腔问题
   • 牙周病
   • 炎症扩散到全身

3. 肠道问题
   • 肠漏
   • 菌群失衡
   • 肠道是最易发炎症器官

4. 老化细胞
   • 分泌促炎物质
   • 随年龄增加

5. 压力
   • 慢性压力
   • 皮质醇失衡
```

### 5. 炎症与疾病关联

**心血管疾病：**
```
动脉硬化 → 心肌梗死、脑中风
炎症损伤血管内壁 → 胆固醇沉积
```

**癌症：**
```
慢性炎症 → DNA 损伤 → 细胞异常分裂
例：幽门螺杆菌→胃癌，肝炎病毒→肝癌
```

**代谢疾病：**
```
炎症 → 胰岛素阻抗 → 糖尿病
炎症 → 脂肪代谢异常 → 高血脂
```

**神经疾病：**
```
炎症 → 血清素减少 → 抑郁症
炎症 → β淀粉样蛋白沉积 → 痴呆
```

**皮肤问题：**
```
炎症 → 特应性皮炎
炎症 → 加速皮肤老化
```


- 行动优先级：减重、治牙、改善睡眠

## 📚 核心理念

**关键洞察：**
1. 慢性炎症无症状但危害大
2. 超敏 CRP 是最佳检测指标
3. 肥胖是最大炎症来源
4. 口腔炎症影响全身
5. 炎症与所有慢性病相关

**健康公式：**
```
长寿健康 = 低炎症 + 抗氧化 + 健康生活方式
```
