# 内脏脂肪减少 🎯

**基于池谷敏郎《抗炎生活》**

---

## 📋 功能描述

指导用户减少内脏脂肪，消除慢性炎症的最大来源。

**适用场景：**
- 腹部肥胖
- 代谢综合征
- 脂肪肝
- 慢性病预防

**边界条件：**
- 不替代医疗建议
- 需配合饮食和运动
- 循序渐进减重

---

## 🎯 核心功能

### 1. 内脏脂肪的危害

**什么是内脏脂肪：**
```
• 围绕内脏器官的脂肪
• 不同于皮下脂肪
• 代谢活跃
• 分泌炎症物质
```

**为什么危险：**
```
内脏脂肪持续分泌：
• 促炎细胞因子（TNF-α、IL-6）
• 游离脂肪酸
• 抵抗素
• 减少脂联素（抗炎）

结果：
✓ 全身慢性炎症
✓ 胰岛素阻抗
✓ 动脉硬化
```

### 2. 内脏脂肪自测

**腰围测量：**
```
男性：
• <85cm：正常
• 85-90cm：需注意
• ≥90cm：内脏脂肪型肥胖

女性：
• <80cm：正常
• 80-85cm：需注意
• ≥85cm：内脏脂肪型肥胖
```

**腰臀比：**
```
男性：
• <0.9：正常
• ≥0.9：风险

女性：
• <0.85：正常
• ≥0.85：风险
```

**代谢综合征判断：**
```
腰围超标 + 以下任意 2 项：
• 高血糖（空腹≥100 mg/dL）
• 高血压（≥130/85 mmHg）
• 高血脂（三酸甘油≥150 或 HDL<40/50）
```

### 3. 内脏脂肪减少策略

**饮食调整：**
```
✓ 减少精制碳水化合物
  • 白米饭→糙米
  • 白面包→全麦
  • 戒除含糖饮料

✓ 增加蛋白质
  • 每餐适量优质蛋白
  • 增加饱腹感
  • 保护肌肉

✓ 好脂肪
  • ω-3 脂肪酸
  • 单不饱和脂肪
  • 避免反式脂肪

✓ 充足纤维
  • 可溶性纤维
  • 蔬菜为主
  • 适量水果
```

**间歇性断食：**
```
16-8 断食：
• 空腹时间增加
• 胰岛素敏感性提高
• 脂肪燃烧增加

效果：
✓ 内脏脂肪优先减少
✓ 炎症指标下降
✓ 代谢改善
```

### 4. 运动方案

**有氧运动：**
```
推荐：
• 快走（每日 30 分钟）
• 慢跑
• 游泳
• 骑自行车

频率：
• 每周 150 分钟中等强度
• 或 75 分钟高强度

效果：
✓ 直接燃烧脂肪
✓ 改善心肺功能
✓ 降低炎症
```

**力量训练：**
```
推荐动作：
• 深蹲（大腿、臀部）
• 俯卧撑（胸部、手臂）
• 平板支撑（核心）
• 划船（背部）

频率：
• 每周 2-3 次
• 主要肌群

效果：
✓ 增加肌肉量
✓ 提高基础代谢
✓ 改善胰岛素敏感性
```

**高强度间歇训练（HIIT）：**
```
方案：
• 30 秒冲刺 + 60 秒放松
• 重复 6-10 次
• 每周 2-3 次

效果：
✓ 时间短、效率高
✓ 后燃效应
✓ 内脏脂肪减少明显
```

### 5. 生活方式支持

**睡眠优化：**
```
• 每晚 7-8 小时
• 固定作息时间
• 睡前放松

睡眠不足后果：
× 饥饿素增加
× 瘦素减少
× 渴望高热量食物
× 内脏脂肪增加
```

**压力管理：**
```
• 慢性压力→皮质醇升高
• 皮质醇→腹部脂肪堆积
• 压力→情绪性进食

管理方法：
✓ 冥想
✓ 深呼吸
✓ 运动
✓ 社交
✓ 爱好
```

**酒精控制：**
```
• 酒精→肝脏优先代谢
• 脂肪代谢暂停
• 多余热量→脂肪储存

建议：
✓ 男性≤2 杯/天
✓ 女性≤1 杯/天
✓ 每周至少 2 天无酒日
```

### 6. 进展追踪

**测量指标：**
```
每周：
• 体重（固定时间）
• 腰围（肚脐水平）

每月：
• 腰臀比
• 体脂率（如有设备）

每 3 个月：
• 血液检查
  - 空腹血糖
  - 血脂
  - 肝功能
  - 超敏 CRP
```

**目标设定：**
```
健康减重：
• 每周 0.5-1kg
• 每月 2-4kg
• 3-6 个月减重 5-10%

过快减重风险：
× 肌肉流失
× 代谢下降
× 反弹
```

## 📚 核心理念

**关键洞察：**
1. 内脏脂肪是炎症温床
2. 腰围是最佳筛查指标
3. 饮食 + 运动 + 睡眠综合干预
4. 间歇性断食特别有效
5. 减重 5-10% 显著改善健康

**健康公式：**
```
无炎体质 = 正常腰围 + 充足肌肉 + 健康生活方式
```
