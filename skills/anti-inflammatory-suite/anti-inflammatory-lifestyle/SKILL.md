# 抗炎生活方式 🧘

**基于池谷敏郎《抗炎生活》**

---

## 📋 功能描述

指导用户通过全方位生活方式调整，抑制慢性炎症。

**适用场景：**
- 慢性疲劳
- 亚健康状态
- 慢性病预防
- 健康优化

**边界条件：**
- 不替代医疗治疗
- 需长期坚持
- 循序渐进改变

---

## 🎯 核心功能

### 1. 抗炎生活支柱

**六大支柱：**
```
1. 抗炎饮食
   • ω-3/ω-6 平衡
   • 抗氧化食物
   • 避免促炎食物

2. 适度运动
   • 释放抗炎肌细胞因子
   • 改善血液循环
   • 减少内脏脂肪

3. 压力管理
   • 慢性压力促炎
   • 放松技巧
   • 充足睡眠

4. 口腔健康
   • 牙周病是炎症源
   • 定期洁牙
   • 正确刷牙

5. 肠道健康
   • 肠漏促炎
   • 菌群平衡
   • 充足纤维

6. 环境毒素
   • 减少暴露
   • 选择天然产品
   • 戒烟限酒
```

### 2. 三分钟抗炎体操

**僵尸体操（释放压力）：**
```
上半身：
• 双臂放松下垂
• 前后自然摆动
• 放松肩膀

下半身：
• 原地慢跑或踏步
• 轻松节奏
• 持续 3 分钟

效果：
✓ 释放压力
✓ 促进血液循环
✓ 释放一氧化氮（NO）
✓ 抑制血管壁发炎
```

**执行时间：**
```
• 早晨起床后
• 工作间隙
• 睡前
• 每日 2-3 次
```

### 3. 运动抗炎机制

**肌细胞因子：**
```
肌肉收缩时释放：
• IL-6（抗炎作用）
• IL-10（强效抗炎）
• IL-1ra（抑制促炎）

效果：
✓ 抑制全身炎症
✓ 改善胰岛素敏感性
✓ 促进脂肪燃烧
```

**运动建议：**
```
有氧运动：
• 快走、慢跑、游泳
• 每周 150 分钟中等强度
• 或 75 分钟高强度

力量训练：
• 每周 2-3 次
• 主要肌群
• 保持肌肉量

日常活动：
• 避免久坐
• 每小时活动 5 分钟
• 多走楼梯
```

### 4. 压力管理

**压力与炎症：**
```
慢性压力 → 皮质醇失衡 → 炎症增加
       → 血清素减少 → 抑郁风险
       → 睡眠障碍 → 炎症加重
```

**放松技巧：**
```
腹式呼吸：
• 吸气 4 秒
• 屏息 4 秒
• 呼气 6-8 秒
• 重复 5-10 次

渐进式肌肉放松：
• 依次紧张 - 放松各肌群
• 从脚到头
• 每晚睡前

正念冥想：
• 每日 10-20 分钟
• 专注呼吸
• 接纳不评判
```

### 5. 睡眠抗炎

**睡眠与炎症：**
```
睡眠不足 → 炎症指标上升
         → 皮质醇升高
         → 食欲激素失衡
         → 体重增加
```

**睡眠优化：**
```
睡前例行：
✓ 固定时间上床
✓ 睡前 1 小时不用电子屏幕
✓ 温水泡澡
✓ 阅读或冥想
✓ 卧室黑暗、安静、凉爽

避免：
× 睡前咖啡因
× 睡前大量进食
× 睡前剧烈运动
× 睡前工作/争论
```

### 6. 口腔抗炎

**牙周病与全身炎症：**
```
牙周病 → 口腔细菌 → 进入血液
      → 全身炎症
      → 心血管疾病风险
      → 糖尿病风险
      → 痴呆风险
```

**口腔护理：**
```
每日：
✓ 刷牙 2 次（2-3 分钟）
✓ 使用牙线
✓ 清洁舌苔

定期：
✓ 每 6 个月洗牙
✓ 定期检查
✓ 及时治疗
```

## 📚 核心理念

**关键洞察：**
1. 慢性炎症需要全方位应对
2. 运动释放抗炎肌细胞因子
3. 压力管理至关重要
4. 口腔健康影响全身
5. 睡眠是天然抗炎药

**健康公式：**
```
抗炎生活 = 抗炎饮食 + 适度运动 + 压力管理 + 充足睡眠 + 口腔健康
```
