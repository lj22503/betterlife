# ω-3/ω-6 平衡 ⚖️

**基于池谷敏郎《抗炎生活》**

---

## 📋 功能描述

指导用户平衡ω-3 和ω-6 脂肪酸摄入，抑制慢性炎症。

**适用场景：**
- 慢性发炎
- 过敏体质
- 心血管疾病风险
- 优化用油选择

**边界条件：**
- 不替代医疗建议
- 需长期坚持
- 注意油脂氧化

---

## 🎯 核心功能

### 1. 脂肪酸基础知识

**三类脂肪酸：**
```
ω-3 脂肪酸：
• 抑制炎症
• 转化为 EPA、DHA
• 来源：鱼油、亚麻籽油

ω-6 脂肪酸：
• 过量促进炎症
• 转化为花生四烯酸 (AA)
• 来源：色拉油、玉米油、大豆油

ω-9 脂肪酸：
• 与炎症无关
• 不竞争代谢路径
• 来源：橄榄油、茶籽油
```

### 2. 现代人脂肪酸失衡

**问题现状：**
```
理想比例：ω-6 : ω-3 = 1 : 1 ~ 4 : 1
现代比例：ω-6 : ω-3 = 10 : 1 ~ 50 : 1

原因：
• 色拉油广泛使用
• 加工食品过多
• 鱼类摄入不足
```

**失衡后果：**
```
• 慢性炎症增加
• 过敏体质
• 心血管疾病风险
• 抑郁症风险
```

### 3. ω-3 脂肪酸来源

**最佳来源（直接含 EPA、DHA）：**
```
🥇 小型野生鱼：
   • 沙丁鱼、鲭鱼、秋刀鱼
   • 每周 2-3 次

🥈 养殖鱼（饲料影响）：
   • 三文鱼
   • 选择可靠来源

🥉 鱼油补充剂：
   • 选择高纯度
   • 注意氧化问题
```

**植物来源（需转化）：**
```
α-亚麻酸→EPA/DHA（转化率低）：
• 亚麻籽油
• 紫苏油
• 绿色坚果油
• 核桃

每日用量：1-2 小茶匙
```

### 4. 用油选择指南

**推荐用油：**
```
✓ 特级初榨橄榄油（ω-9）
  用途：凉拌、低温烹饪
  
✓ 亚麻籽油（ω-3）
  用途：凉拌、果汁添加
  注意：不耐高温
  
✓ 紫苏油（ω-3）
  用途：凉拌
  注意：不耐高温
  
✓ 椰子油（饱和脂肪）
  用途：中温烹饪
  注意：适量使用
```

**避免用油：**
```
× 色拉油（ω-6 过高）
× 玉米油（ω-6 过高）
× 大豆油（ω-6 过高）
× 葵花籽油（ω-6 过高）
× 人造奶油（反式脂肪）
× 起酥油（反式脂肪）
```

### 5. 鱼类摄入方案

**推荐鱼类：**
```
每周 2-3 次，每次 100-150g：
✓ 沙丁鱼（EPA+DHA 丰富）
✓ 鲭鱼（青花鱼）
✓ 秋刀鱼
✓ 三文鱼
✓ 鳕鱼
```

**烹饪方法：**
```
✓ 最佳：煎、煮、包铝箔纸烘烤
  （保留鱼油）

× 避免：烤网烤鱼
  （鱼油流失）
```

**注意事项：**
```
• 选择小型鱼（重金属少）
• 避免深海大型鱼
• 新鲜最佳
• 冷冻保存注意氧化
```

### 6. ω-3 补充方案

**饮食优先：**
```
方案 A（鱼类）：
• 每周 2-3 次鱼
• 每次 100-150g
• 多样化选择

方案 B（植物油）：
• 每日亚麻籽油 1-2 茶匙
• 加入果汁或料理最后步骤
• 不加热使用
```

**补充剂考虑：**
```
适合人群：
• 不吃鱼者
• 素食者
• 炎症指标高者

剂量：
• EPA+DHA 合计 1-2g/天
• 选择高纯度
• 注意保质期
```

## 📚 核心理念

**关键洞察：**
1. ω-3 抑制炎症，ω-6 过量促炎
2. 现代人ω-6 摄入过量 10-50 倍
3. 鱼类是最佳ω-3 来源
4. 亚麻籽油需冷用
5. 用油选择决定炎症水平

**平衡公式：**
```
抗炎体质 = ω-3 充足 + ω-6 适量 + 无反式脂肪
```
