## 📖 简介

抗炎生活套件包含 5 个技能，帮助用户抑制慢性炎症，预防现代疾病，延缓老化。

**核心理念：**
- 🔥 慢性炎症是万病之源
- 🩸 超敏 CRP 反映炎症程度
- ⚖️ ω-3/ω-6 平衡是关键
- 🥬 抗氧化就是抗炎
- 🎯 内脏脂肪是炎症温床

## 🔑 关键概念

### 慢性炎症
- 无症状但危害大
- 加速老化、导致疾病
- 超敏 CRP 是最佳检测指标
- 正常值：≤0.30 mg/dL

### 脂肪酸平衡
- ω-3 抑制炎症
- ω-6 过量促炎
- 理想比例：1:1 ~ 4:1
- 现代比例：10:1 ~ 50:1

### 氧化 - 发炎循环
- 活性氧过量→细胞损伤→炎症
- 炎症→更多活性氧
- 抗氧化就是抗炎
- 彩虹饮食最全面

### 内脏脂肪
- 围绕内脏的脂肪
- 持续分泌炎症物质
- 腰围是最佳筛查指标
- 减重 5-10% 显著改善
