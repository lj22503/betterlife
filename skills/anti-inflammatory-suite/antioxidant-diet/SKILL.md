# 抗氧化饮食 🥬

**基于池谷敏郎《抗炎生活》**

---

## 📋 功能描述

指导用户通过抗氧化饮食对抗活性氧，抑制氧化 - 发炎循环。

**适用场景：**
- 抗衰老
- 慢性病预防
- 皮肤保养
- 癌症预防

**边界条件：**
- 不替代医疗治疗
- 需配合整体健康生活方式
- 个体差异需考虑

---

## 🎯 核心功能

### 1. 氧化 - 发炎循环

**活性氧的双面性：**
```
正面作用：
• 免疫细胞武器
• 杀灭入侵细菌病毒
• 必需的健康功能

负面作用（过量时）：
• 伤害正常细胞
• 损伤 DNA
• 促进炎症
• 加速老化
```

**恶性循环：**
```
活性氧增加 → 细胞损伤 → 炎症
     ↑                    ↓
     └──── 炎症 → 更多活性氧 ────┘

结果：加速老化、疾病风险增加
```

### 2. 抗氧化物质分类

**维生素类：**
```
维生素 C：
• 水溶性抗氧化剂
• 来源：柑橘类、草莓、青椒
• 功能：再生维生素 E

维生素 E：
• 脂溶性抗氧化剂
• 来源：坚果、种子、植物油
• 功能：保护细胞膜

β-胡萝卜素：
• 维生素 A 前体
• 来源：胡萝卜、南瓜、深绿蔬菜
• 功能：保护皮肤、视力
```

**植化素类：**
```
花青素：
• 来源：蓝莓、葡萄、紫色蔬菜
• 功能：改善视力、抗发炎

番茄红素：
• 来源：西红柿、西瓜
• 功能：保护前列腺、心血管

儿茶素：
• 来源：绿茶
• 功能：抗癌、降血脂

大豆异黄酮：
• 来源：大豆制品
• 功能：调节激素、护骨
```

**其他抗氧化剂：**
```
谷胱甘肽：
• 体内合成
• 来源：西兰花、牛油果

辅酶 Q10：
• 体内合成（随年龄减少）
• 来源：肉类、鱼类

硫辛酸：
• 来源：菠菜、西兰花、动物内脏
```

### 3. 抗氧化食物排行榜

**最佳抗氧化食物：**
```
🥇 香料类：
   • 丁香、肉桂、姜黄
   • 抗氧化能力最强

🥈 浆果类：
   • 蓝莓、草莓、树莓
   • 花青素丰富

🥉 坚果类：
   • 核桃、杏仁
   • 维生素 E 丰富

其他：
   • 深绿蔬菜（菠菜、羽衣甘蓝）
   • 十字花科（西兰花、花椰菜）
   • 绿茶
   • 黑巧克力（70%+ 可可）
```

### 4. 彩虹饮食法

**原则：**
```
每日摄入 5 种以上颜色的蔬果
每种颜色代表不同抗氧化剂
```

**颜色与抗氧化剂：**
```
红色：番茄红素、花青素
  来源：西红柿、西瓜、草莓

橙色：β-胡萝卜素
  来源：胡萝卜、南瓜、芒果

黄色：叶黄素、玉米黄素
  来源：玉米、蛋黄

绿色：叶绿素、异硫氰酸盐
  来源：菠菜、西兰花、羽衣甘蓝

蓝紫色：花青素
  来源：蓝莓、葡萄、紫甘蓝

白色：大蒜素、槲皮素
  来源：大蒜、洋葱、苹果
```

### 5. 抗氧化饮食实践

**每日目标：**
```
蔬菜：5-7 份（1 份=生蔬菜 1 杯/熟蔬菜 1/2 杯）
水果：2-3 份（1 份=中等大小水果 1 个）
坚果：1 小把（约 30g）
绿茶：2-3 杯
香料：适量使用
```

**进食顺序：**
```
1. 蔬菜（抗氧化剂 + 纤维）
2. 蛋白质
3. 碳水化合物

效果：
• 预防血糖骤升
• 增加抗氧化剂吸收
• 控制食量
```

**烹饪技巧：**
```
✓ 生吃（保留维生素 C）
✓ 快炒（减少营养流失）
✓ 蒸煮（保留水溶性营养）
✓ 连皮吃（抗氧化剂多在皮）

× 长时间高温
× 反复油炸
× 过度加工
```

### 6. 抗氧化补充剂

**饮食优先原则：**
```
• 天然食物含多种抗氧化剂
• 协同作用优于单一补充
• 食物基质帮助吸收
• 补充剂仅作补充
```

**考虑补充的情况：**
```
• 饮食不均衡
• 特殊需求（运动员）
• 吸收问题
• 医生建议
```

**常见补充剂：**
```
• 维生素 C：500-1000mg/天
• 维生素 E：100-200IU/天
• 辅酶 Q10：100-200mg/天（40 岁+）
• 葡萄籽提取物：100-200mg/天
```

## 📚 核心理念

**关键洞察：**
1. 氧化和发炎同时发生
2. 抗氧化就是抗炎
3. 彩虹饮食最全面
4. 天然食物优于补充剂
5. 先吃蔬菜效果最佳

**健康公式：**
```
抗氧化 = 多样化蔬果 + 彩虹饮食 + 正确烹饪
```
