# 激励情绪寻找器 ✨

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户寻找激励情绪，邂逅真正的卓越，被美好所激励和振奋。

**适用场景：**
- 想被激励
- 想感到振奋
- 寻找榜样
- 提升动力

**边界条件：**
- 激励来自真正的卓越
- 目睹人性最好的一面
- 被美好所激励

---

## 🎯 核心功能

### 1. 激励的特征

**引发激励的条件：**
```
• 你无意中发现了真正的卓越
• 目睹人性最好的一面
• 能够启发和振奋你
• 遇到真正的成就或美德
• 看到某个人表现得比你预期的还要好
```

### 2. 激励的感受

**身体感受：**
```
• 被美好所激励、振奋或鼓舞
• 被吸引着去见证在你眼前呈现的成就
• 看见人类的最佳状态
• 下巴差点掉下来
• 感受到尽最大努力以达到更高的水平的冲动
```

### 3. 培养方法

**微习惯：**
```
• 寻找榜样
• 阅读成功人士的故事
• 观看激励人心的演讲
• 见证他人的成就
• 尽最大努力达到更高水平
```

---

## ⚠️ 常见错误

**错误 1：等待被激励**
```
问题：
• "等我被激励"
• 被动等待

解决：
✓ 主动寻找榜样
✓ 阅读成功人士的故事
✓ 见证他人的成就
```
