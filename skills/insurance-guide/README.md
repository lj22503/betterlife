# 你的保险指南 Skill 套件 🛡️

**基于《你的保险指南》的家庭保障管理技能**

---

## 📖 简介

这套技能将保险规划知识转化为实用的 AI 工具，帮助家庭从保障管理角度规划保险，避免保障错配，确保家庭财务安全。

**核心理念：**
- 🏠 **家庭风险评估** —— 先保大人，后保小孩
- 📋 **保险配置五步走** —— 体检报告→险种了解→产品对比→渠道选择→保单整理
- 🏥 **健康告知** —— 有限告知，问什么答什么
- 📁 **保单整理** —— 避免保单沉睡
- 🔍 **产品对比** —— 多维度对比，选择最适合

### 使用示例

#### 1. 家庭风险评估

```
@ant 帮我评估家庭风险

信息：
- 丈夫：35 岁，年收入 50 万
- 妻子：33 岁，年收入 20 万
- 孩子：5 岁
- 房贷：800 万
- 储蓄：50 万
```

**预期输出：**
- 最大风险点：丈夫（70% 收入贡献）
- 保障优先级：丈夫>妻子>孩子
- 寿险缺口：800 万房贷 + 孩子教育

#### 2. 保险配置方案

```
@ant 帮我们配置保险方案
```

**预期输出：**
- 丈夫：重疾 250 万 + 医疗 300 万 + 意外 500 万 + 寿险 1000 万
- 妻子：重疾 100 万 + 医疗 300 万 + 意外 200 万 + 寿险 500 万
- 孩子：重疾 50 万 + 医疗 300 万 + 意外 50 万
- 总保费：约 12-15 万/年（占收入 18%）

#### 3. 健康告知

```
@ant 我有甲状腺结节，如何投保？
```

**预期输出：**
- 有限告知：问什么答什么
- 核保预测：可能加费或除外
- 建议：选择核保宽松的产品

#### 4. 保单整理

```
@ant 帮我整理家庭保单
```

**预期输出：**
- 按成员分类整理
- 缴费提醒设置
- 受益人告知指导

#### 5. 产品对比

```
@ant 对比这 3 款重疾险
```

**预期输出：**
- 保障责任对比
- 高发轻症核对
- 优缺点分析
- 推荐建议

## 📚 保险配置 3 大原则

### 原则 1：先考虑大风险，再看小风险

**家庭最大风险点 = 主要收入来源者**

- 先保经济支柱，后保其他人
- 先保大人，后保小孩
- 避免"孩子保险很多，大人裸奔"

### 原则 2：先看体检报告，再看保险产品

**健康告知规则：**
- 有限告知：问什么答什么，不问不答
- 不要为了投保而去体检
- 已有异常如实告知

### 原则 3：先自省内心担忧，再看产品类型

**8 个自省问题：**
1. 有没有什么问题是让你夜不能寐的？
2. 家里的哪位家庭成员一旦生病，现金流就会被切断？
3. 如果赚钱的家庭成员生重病或者去世，你会不会需要卖房子来还房贷？
4. 假设家庭成员有人生病，特别是疑难杂症，希望享受到什么样的医疗服务？
5. 有没有哪一类疾病让你特别担心且恐惧？
6. 你的家庭储蓄目前有多少？能不能一次性无压力地拿出 20 万元？
7. 父母的医疗保障状况如何？你是否会担心他们将来产生的护理和医疗费用？
8. 有没有某一种极端情况发生，会给家庭财务带来毁灭性的打击？

## 📋 保险配置五步走

```
第 1 步：翻开体检报告，汇总异常指标
    ↓
第 2 步：了解各个险种的作用，挑选适合产品
    ↓
第 3 步：找出匹配的产品，进行产品对比
    ↓
第 4 步：选择合适的渠道投保
    ↓
第 5 步：整理家庭保单
```

---

## 🛡️ 四大基础险种

| 险种 | 作用 | 推荐保额 | 优先级 |
|------|------|----------|--------|
| 重疾险 | 收入损失补偿 | 年收入×3-5 倍 | 高（经济支柱） |
| 医疗险 | 住院费用报销 | 200-400 万 | 高（全员） |
| 意外险 | 意外身故/伤残 | 年收入×10 倍 | 高（全员） |
| 寿险 | 身故理赔（覆盖负债） | 房贷 + 孩子教育 + 父母赡养 | 高（有负债者） |

## 🏠 不同家庭类型配置方案

### 单身贵族/丁克

**配置：** 重疾险 + 医疗险 + 意外险 +（如有房贷）定期寿险 + 充足养老金

**核心：** 唯一能终身照顾自己的人就是年轻时的自己

### 新手爸妈

**配置：**
- 丈夫：重疾 + 医疗 + 意外 + 定期寿险
- 妻子：重疾 + 医疗 + 意外 + 定期寿险
- 孩子：重疾 + 医疗 + 意外

**核心：** 先保大人，后保小孩

### 全职主妇

**配置：** 基础保额重疾险 + 医疗险 + 养老金

**核心：** 给自己一份安全感

### 企业主

**配置：** 家企分离 + 打底资产 + 子女教育现金流 + 终身寿险

**核心：** 抵抗企业经营风险和债务风险

## ⚠️ 重要提示

**保险配置核心原则：**

1. **先保大人，后保小孩** —— 经济支柱优先
2. **先保大风险，后保小风险** —— 重疾/身故优先
3. **先看体检报告，再看保险产品** —— 有限告知
4. **保额优先于期限** —— 先做高保额，再考虑保障期限
5. **整理家庭保单** —— 避免保单沉睡
