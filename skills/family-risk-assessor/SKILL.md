# 家庭风险评估器 🏠

**基于《你的保险指南》**

---

## 📋 功能描述

帮助用户评估家庭风险，建立家庭保障。

**适用场景：**
- 评估家庭风险
- 家庭保障规划
- 保险配置

---

## 🎯 核心功能

### 1. 核心方法

**步骤：**
```
1. 评估家庭风险
2. 确定保险需求
3. 配置保险产品
4. 定期整理保单
```

---

## ⚠️ 常见错误

**错误 1：先保小孩后保大人**
```
问题：
• 给小孩买很多保险
• 大人没有保障

解决：
✓ 先保大人，后保小孩
✓ 经济支柱优先
✓ 先保大风险，后保小风险
```

**错误 2：不看体检报告就买保险**
```
问题：
• 不了解自己的健康状况
• 健康告知不准确

解决：
✓ 先看体检报告
✓ 有限告知原则
✓ 如实告知
```

## 📚 核心理念

**关键洞察：**
1. 评估家庭风险
2. 先保大人，后保小孩
3. 先保大风险，后保小风险
4. 先看体检报告，再看保险产品

**健康公式：**
```
家庭保障 = 重疾险 + 医疗险 + 意外险 + 寿险
```

---

*保险配置原则：
先保大人，后保小孩；
先保大风险，后保小风险；
先看体检报告，再看保险产品。
保险是家庭的护城河，
