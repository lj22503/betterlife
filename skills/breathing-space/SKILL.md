# 3 分钟呼吸空间器 ⏱️

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

引导用户进行 3 分钟呼吸空间练习，快速回到当下，稳定精神状态。

**适用场景：**
- 快速回到当下
- 工作压力时
- 情绪波动时
- 每天固定练习（2 次）

**边界条件：**
- 每天固定时间 2 次
- 需要时随时可进行
- 可与其他禅修配合

---

## 🎯 核心功能

### 1. 第 1 步：集中意念（1 分钟）

**步骤：**
```
1. 选择直立端正的姿势（坐或立）
2. 闭上双眼
3. 询问自己：我现在的感觉是什么？

【想法】
• 脑海中现在有哪些想法？
• 承认想法只是大脑的活动

【感受】
• 我有什么感受？
• 主动面对不适和不快感受
• 承认它们的客观存在

【身体感知】
• 现在有什么身体感知？
• 快速扫描身体
• 搜索紧张或压抑感
```

### 2. 第 2 步：聚集并集中精力（1 分钟）

**步骤：**
```
1. 将注意力转向腹部
2. 关注气息感知区域
3. 气息进入时扩大聚光灯
4. 气息排出时收回聚光灯
5. 不间断跟踪吸气和呼气
6. 将每一次呼吸当成安顿现在的机会
7. 意识游离时温和送回
```

### 3. 第 3 步：扩大注意范围（1 分钟）

**步骤：**
```
1. 将意识范围扩大
2. 将整个身体包含进去
3. 感受姿态和面部表情
4. 好像整个身体都在呼吸
5. 如有不适，想象气息进入
6. 将注意力焦点引入知觉强化区
7. 善待它们，不试图改变
```

---

## ⚠️ 常见错误

**错误 1：跳过第 1 步**
```
问题：
• 直接开始呼吸
• 不承认当前感受

解决：
✓ 第 1 步很重要
✓ 承认当前的想法、感受、身体感知
✓ 不要试图改变它们
```

**错误 2：时间不够**
```
问题：
• "我没有 3 分钟"
• 跳过练习

解决：
✓ 3 分钟很短
✓ 随时可进行
✓ 每天固定 2 次
```
