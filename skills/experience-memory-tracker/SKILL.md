# 体验记忆追踪器 📸

**基于《最优解人生》（Die With Zero）核心理念**

---

## 📋 功能描述

记录用户的人生体验，追踪回忆红利，生成体验报告，帮助用户最大化体验价值。

**适用场景：**
- 记录已完成体验
- 追踪回忆红利积累
- 生成年度体验报告
- 识别体验模式和问题

---

## 🎯 核心功能

### 1. 体验记录

**记录字段：**
- 体验名称
- 日期/时长
- 地点
- 同伴
- 花费
- 愉悦度评分（1-10）
- 照片/视频链接
- 文字记录

### 2. 回忆红利追踪

**公式：** 回忆红利 = 愉悦度 × 已过年数 × 分享次数

### 3. 年度体验报告

**报告内容：**
- 本年度体验数量
- 总花费
- 平均愉悦度
- 最佳体验 TOP3
- 体验类别分布

### 4. 体验模式分析

**分析维度：**
- 体验频率
- 体验多样性
- 支出合理性
- 愉悦度趋势

## 💡 使用建议

**完整体验记忆追踪流程：**

1. **体验记录** - 名称/日期/地点/同伴/花费/愉悦度
2. **回忆红利追踪** - 愉悦度×已过年数×分享次数
3. **年度体验报告** - 数量/花费/愉悦度/TOP3
4. **体验模式分析** - 频率/多样性/支出/趋势

**关键原则：**
- 回忆红利 = 愉悦度 × 已过年数 × 分享次数
- 记录体验增强记忆固化
- 定期回顾提升幸福感
