# 肠道健康方案 🦠

**基于杨定一《真原医：21 世纪完整的预防科学》第肆章**

---

## 📋 功能描述

指导用户通过饮食和生活方式改善结肠健康，影响整体健康。

**适用场景：**
- 便秘/腹泻
- 胀气
- 慢性发炎
- 免疫力低下
- 皮肤问题

**边界条件：**
- 不替代医疗诊断
- 严重症状需就医
- 需长期坚持

---

## 🎯 核心功能

### 1. 结肠三大功能

```
1. 蠕动使废物排出
   • 膳食纤维促进蠕动
   • 水分软化粪便

2. 吸收养分
   • 水分
   • 电解质
   • 维生素（肠道菌群合成）

3. 维持良好的发酵环境和黏液生成
   • 益生菌发酵
   • 黏液保护肠壁
```

### 2. 肠道健康 6 步骤

**步骤 1：清除肠道累积的陈年宿便**
```
• 高膳食纤维饮食
• 充足水分（2-3L/天）
• 蔬菜汁（大麦苗汁最佳）
```

**步骤 2：改变饮食**
```
从产生毒素的饮食 → 平衡、帮助排泄和清洁的饮食
• 减少：肉类、加工食品、精制糖
• 增加：蔬菜、水果、全谷物
```

**步骤 3：适度断食**
```
• 给肠道休息时间
• 16-8 间歇性断食
• 每月 1-2 次 24 小时断食
```

**步骤 4：清洁结肠**
```
• 高纤维食物
• 充足水分
• 适度运动
```

**步骤 5：使菌落生态恢复正常**
```
• 益生菌：发酵食品
• 益生元：膳食纤维
• 避免：抗生素滥用
```

**步骤 6：清除心灵的旧习惯**
```
• 压力管理
• 静坐
• 正面思考
```

### 3. 结肠友好饮食

**推荐食物：**
```
✓ 高膳食纤维：
  • 蔬菜（尤其是绿叶蔬菜）
  • 水果（连皮吃）
  • 全谷物（糙米、燕麦）
  • 豆类

✓ 益生菌：
  • 纳豆、味噌（不加热）
  • 酸菜、泡菜
  • 无糖酸奶

✓ 充足水分：
  • 好水 2-3L/天
  • 蔬菜汁
  • 草本茶
```

**避免食物：**
```
× 加工食品
× 精制糖
× 过量肉类
× 油炸食品
× 酒精
```

### 4. 肠道健康指标

**健康信号：**
```
✓ 每日规律排便（1-2 次）
✓ 粪便成形、不硬不软
✓ 排便轻松、无需费力
✓ 无胀气、无疼痛
✓ 卫生纸干净（无过多黏液）
```

**不健康信号：**
```
× 便秘或腹泻
× 排便费力
× 粪便恶臭
× 胀气、腹痛
× 卫生纸很脏（肠漏信号）
```

### 5. 肠道修复计划

**第 1-2 周：清理期**
```
• 高纤维饮食
• 蔬菜汁每日 500ml
• 断除加工食品
• 开始发酵食品
```

**第 3-4 周：修复期**
```
• 继续高纤维
• 增加益生菌
• 开始间歇性断食
• 适度运动
```

**第 5-8 周：巩固期**
```
• 维持健康饮食
• 规律断食
• 压力管理
• 静坐练习
```

## 📚 核心理念

**关键洞察：**
1. 结肠影响整体健康
2. 高膳食纤维是关键
3. 益生菌 + 益生元并重
4. 适度断食给肠道休息
5. 心灵习惯也影响肠道

**健康公式：**
```
肠道健康 = 高纤维 + 益生菌 + 充足水分 + 适度断食 + 压力管理
```
