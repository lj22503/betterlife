# 生机饮食指南 🥗

**基于杨定一《真原医：21 世纪完整的预防科学》第叁章**

---

## 📋 功能描述

指导用户采用生机饮食（Raw Food），摄取活食材，恢复身体自愈能力。

**适用场景：**
- 改善消化问题
- 提升能量水平
- 慢性病预防
- 体重管理

**边界条件：**
- 不替代医疗建议
- 需渐进过渡
- 个体差异需调整

---

## 🎯 核心功能

### 1. 生机饮食原则

**核心定义：**
```
生机饮食 = 未经烹调或未以 48℃以上加热处理的食物
```

**为什么重要：**
- 食物酶仅存于生食中
- 酵素帮助消化，节省身体能量
- 人体胃部设计原本就是接收含酶食物
- 熟食失去生命力，造成健康能量活力减低

**建议比例：**
```
• 最低：每餐 50% 生食
• 理想：80% 以上生食
• 来源：蔬菜、水果
```

### 2. 活食材选择

**最佳蛋白质来源（植物性）：**
```
✓ 胡萝卜、抱子甘蓝、芽菜
✓ 甘蓝菜、白色菜花、玉米
✓ 小黄瓜、茄子、羽衣甘蓝
✓ 秋葵、豆类、马铃薯
✓ 节瓜、地瓜、西红柿、香蕉
```

**为什么植物蛋白优于动物蛋白：**
- 烹调过的动物蛋白质失去正确能量波动
- 植物蛋白含有完整必需氨基酸
- 更易消化吸收
- 不产生代谢负担

### 3. 蔬菜汁疗法

**为什么蔬菜汁营养最高：**
```
• 去除难以消化的粗膳食纤维
• 保留最丰富的营养
• 具有最高的营养价值
• 立即被身体吸收
```

**最佳选择：**
- 大麦苗汁（最丰富的矿物质、酶、维生素、叶绿素）
- 绿菜花汁
- 菠菜汁
- 综合蔬菜汁

**饮用原则：**
```
• 蔬菜要打汁
• 水果要生吃
• 空腹饮用最佳
• 慢慢喝，充分混合唾液
```

### 4. 进食七原则

```
1. 给自己足够的时间
2. 选用可引起食欲的均衡饮食
3. 以自在悠闲的心情，慢慢进食
4. 小口小口慢慢吃
5. 细嚼慢咽，让每一口都得到唾液的充分浸润
6. 仔细品味、欣赏并感恩每一口的食物
7. 专心吃饭
```

### 5. 植化素彩虹饮食

| 颜色 | 植化素 | 功效 | 食物来源 |
|------|--------|------|----------|
| 绿色 | 叶绿素 | 抗发炎、抗氧化、预防癌症 | 绿菜花、菠菜、青椒、大麦苗 |
| 蓝紫色 | 花青素 | 改善视力、抗发炎、抗菌 | 蓝莓、葡萄、紫色卷心菜、茄子 |
| 白色 | 硫化物 | 提高免疫力、降低胆固醇 | 大蒜、大葱、菜花、卷心菜 |
| 黄色 | β-胡萝卜素 | 降低冠状动脉疾病与癌症 | 胡萝卜、南瓜、玉米 |
| 红色 | 番茄红素 | 消除自由基、保护心血管 | 西红柿、西瓜、樱桃、辣椒 |

## 📚 核心理念

**关键洞察：**
1. 活着不代表健康
2. 食物酶仅存于生食中
3. 人体胃部设计是接收含酶食物
4. 蔬菜汁营养高于固体蔬菜
5. 细嚼慢咽减轻消化负担

**诺曼·沃克博士案例：**
- 50 多岁患重病
- 发现蔬菜汁好处并身体力行
- 健康活到 119 岁
