# 食物酶营养学 🧬

**基于杨定一《真原医：21 世纪完整的预防科学》第叁章**

---

## 📋 功能描述

指导用户通过食物酶优化消化，节省身体能量，激活自愈能力。

**适用场景：**
- 消化问题（胀气、便秘）
- 慢性疲劳
- 能量不足
- 自体免疫问题

**边界条件：**
- 不替代医疗建议
- 需配合整体饮食调整
- 个体差异需观察

---

## 🎯 核心功能

### 1. 食物酶基础知识

**什么是食物酶：**
```
• 仅存于生食中（未经 48℃以上加热）
• 帮助消化食物
• 节省身体制造消化酶的能量
• 让代谢酶更有效率提供给其他系统
```

**为什么重要：**
```
人体能量分配：
× 熟食饮食：大量能量用于制造消化酶
✓ 生机饮食：节省能量用于建构健康、自我治疗
```

### 2. 食物酶来源

**最佳来源：**
```
🥇 亚洲发酵大豆制品（不加热）：
   • 纳豆
   • 味噌

🥈 生蔬菜水果：
   • 木瓜（木瓜酶）
   • 菠萝（菠萝酶）
   • 芒果
   • 芽菜类

🥉 其他：
   • 蜂蜜（生）
   • 牛油果
   • 发酵食品（酸菜、泡菜）
```

**酶活性温度：**
```
• 最佳活性：37-47℃
• 开始失活：48℃以上
• 完全失活：70℃以上
```

### 3. 酶缺乏症状

**消化系统：**
```
• 饭后胀气
• 消化不良
• 便秘或腹泻
• 胃酸逆流
• 食物不耐受
```

**全身症状：**
```
• 慢性疲劳
• 能量不足
• 免疫力下降
• 发炎增加
• 愈合缓慢
```

### 4. 酶补充方案

**饮食调整：**
```
每餐原则：
✓ 50% 以上生食
✓ 先吃生食沙拉
✓ 再吃熟食
✓ 发酵食品不加热
```

**酶补充时机：**
```
• 饭前：生食沙拉（激活消化）
• 饭后：发酵食品（帮助消化）
• 两餐间：水果（单独吃）
```

**注意事项：**
```
× 水果不要饭后立即吃
  （容易在肠胃中滞留发酵）
✓ 水果单独吃或空腹吃
✓ 与正餐间隔 1-2 小时
```

### 5. 发酵食品制作

**简易纳豆：**
```
材料：黄豆、纳豆菌
步骤：
1. 黄豆浸泡 12 小时
2. 蒸熟（保留部分酶）
3. 接种纳豆菌
4. 发酵 24 小时（40℃）
5. 冷藏保存
```

**简易泡菜：**
```
材料：卷心菜、盐、水
步骤：
1. 卷心菜切丝加盐
2. 装入消毒容器
3. 室温发酵 3-7 天
4. 冷藏保存
```

## 📚 核心理念

**关键洞察：**
1. 食物酶仅存于生食中
2. 48℃以上加热酶开始失活
3. 节省消化酶能量用于自愈
4. 发酵食品是酶的最佳来源
5. 水果不要饭后立即吃

**能量分配原理：**
```
熟食饮食 → 大量能量用于消化 → 自愈能量不足
生机饮食 → 消化能量节省 → 自愈能量充足
```
