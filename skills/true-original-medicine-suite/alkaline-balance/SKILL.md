# 酸碱平衡评估 ⚖️

**基于杨定一《真原医：21 世纪完整的预防科学》第肆章**

---

## 📋 功能描述

帮助用户评估和维持身体酸碱平衡，创造健康生化反应环境。

**适用场景：**
- 慢性疲劳
- 骨质流失风险
- 慢性发炎
- 饮食优化

**边界条件：**
- 不替代医疗检测
- 血液 pH 有自我调节机制
- 重点在饮食平衡

---

## 🎯 核心功能

### 1. 酸碱平衡基础

**健康状态：**
```
• 人体健康状态：弱碱性
• 正常血液 pH 值：7.4 左右
• 此时体内各种生化作用才能正常运作
```

**为什么重要：**
```
• 矿物质参与人体各项酶活动
• 缺少矿物质，维生素吃再多也不会有作用
• 矿物质被视为五大营养素之首
• 酸碱平衡影响矿物质吸收
```

### 2. 食物酸碱性

**碱性食物（80% 饮食）：**
```
✓ 绝大多数蔬菜
✓ 水果（包括柠檬）
✓ 豆类
✓ 芽菜
✓ 海藻类

注意：
• 柠檬虽酸，代谢后产生阳离子
• 是极佳的碱性食物
```

**酸性食物（20% 饮食）：**
```
× 肉类
× 鱼类
× 蛋类
× 谷类（尤其是精制）
× 坚果（适量）
```

**理想比例：**
```
酸性食物：碱性食物 = 20% : 80%
```

### 3. 酸碱失衡症状

**轻度失衡：**
```
• 慢性疲劳
• 免疫力下降
• 容易感冒
• 皮肤问题
```

**中度失衡：**
```
• 骨质流失
• 肌肉流失
• 慢性发炎
• 消化问题
```

**重度失衡：**
```
• 骨质疏松
• 肾结石
• 慢性疼痛
• 器官功能下降
```

### 4. 酸碱平衡方案

**饮食调整：**
```
每餐原则：
✓ 80% 蔬菜（大量绿叶蔬菜）
✓ 20% 蛋白质 + 全谷物
✓ 水果作为加餐
✓ 柠檬水（早晨空腹）
```

**每日计划：**
```
早餐：蔬菜汁 + 少量水果
午餐：大量沙拉 + 适量蛋白质
晚餐：蔬菜汤 + 少量全谷物
加餐：水果、坚果（少量）
```

**生活方式：**
```
✓ 深呼吸（排出酸性代谢物）
✓ 适度运动
✓ 充足水分（2-3L/天）
✓ 减少压力
✓ 充足睡眠
```

### 5. 酸碱测试方法

**尿液 pH 测试：**
```
• 使用 pH 试纸
• 早晨第一次尿液
• 理想范围：6.5-7.5
• 连续测试 1 周取平均
```

**唾液 pH 测试：**
```
• 饭前测试
• 理想范围：7.0-7.5
• 反映消化状态
```

**注意：**
- 血液 pH 由身体严格调控（7.35-7.45）
- 尿液/唾液 pH 反映饮食影响
- 重点是长期饮食模式

## 📚 核心理念

**关键洞察：**
1. 健康身体呈弱碱性（pH 7.4）
2. 绝大多数蔬菜水果是碱性
3. 柠檬虽酸但是碱性食物
4. 理想比例：20% 酸性：80% 碱性
5. 矿物质是五大营养素之首

**酸碱公式：**
```
健康 = 80% 碱性食物 + 20% 酸性食物
     = 大量蔬菜 + 适量水果 + 少量蛋白质/谷类
```
