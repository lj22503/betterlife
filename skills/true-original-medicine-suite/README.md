## 📖 简介

真原医基础套件包含 5 个技能，帮助用户采用生机饮食，维持酸碱平衡，打造健康基础。

**核心理念：**
- 🥗 生机饮食保留食物酶
- 🌿 叶绿素是天然解毒剂
- 🧬 食物酶节省身体能量
- ⚖️ 酸碱平衡 20:80
- 🦠 结肠影响整体健康
