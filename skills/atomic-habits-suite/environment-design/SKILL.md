# 环境设计 🏠

**基于《原子习惯》**

---

## 📋 功能描述

指导用户通过环境设计培养好习惯，让好习惯显而易见、简便易行，让坏习惯难以执行。

**适用场景：**
- 习惯养成困难
- 意志力不足
- 环境干扰多
- 行为设计需求

**边界条件：**
- 不替代自我管理
- 需配合内在动机
- 个体差异需调整

---

## 🎯 核心功能

### 1. 环境设计原理

**核心原理：**
```
原动力被高估，环境往往更重要

问题：
× 依赖意志力
× 自我控制有限
× 容易耗尽

解决：
✓ 设计环境
✓ 减少阻力
✓ 让好习惯自动发生
```

**环境影响力：**
```
• 视觉提示触发行为
• 环境线索塑造习惯
• 空间设计影响选择
• 物品摆放决定使用频率
```

### 2. 空间设计原则

**一个空间，一种用途：**
```
原则：
• 每个空间有明确用途
• 避免混合不同习惯情境
• 减少习惯混淆

实践：
✓ 工作区：只工作
✓ 学习区：只学习
✓ 锻炼区：只锻炼
✓ 娱乐区：只娱乐
✓ 睡眠区：只睡眠
```

**避免情境混合：**
```
问题：
× 在床上工作
× 在沙发上玩手机
× 在餐桌前看电视

后果：
• 习惯混淆
• 简单习惯占上风
• 坏习惯更容易

解决：
✓ 分离不同活动空间
✓ 明确空间用途
✓ 建立空间 - 行为连接
```

### 3. 让好习惯显而易见

**视觉提示设计：**
```
策略：
✓ 放在显眼位置
✓ 增加可见性
✓ 减少隐藏

例子：
• 想多喝水：水瓶放在各处显眼位置
• 想多读书：书放在床边、沙发
• 想多运动：运动装备放在门口
• 想吃健康食物：水果放在台面
```

**提示叠加：**
```
方法：
• 新习惯提示 + 现有习惯提示
• 利用现有行为线索
• 建立提示链

例子：
• 牙刷旁放牙线
• 咖啡机旁放维生素
• 门口放运动包
```

**环境重置：**
```
每日重置：
✓ 睡前准备第二天
✓ 物品归位
✓ 减少次日阻力

每周重置：
✓ 整理空间
✓ 补充消耗品
✓ 检查提示有效性
```

### 4. 让坏习惯难以执行

**增加阻力策略：**
```
物理阻力：
✓ 藏起来（看不见）
✓ 锁起来（拿不到）
✓ 放远点（不方便）

步骤阻力：
✓ 增加执行步骤
✓ 添加障碍
✓ 要求额外努力

例子：
• 手机放另一个房间
• 不健康零食放高处/藏起来
• 卸载社交媒体 APP
• 信用卡放难拿的地方
```

**减少提示：**
```
视觉提示：
✓ 移除触发物
✓ 遮盖提示
✓ 改变布局

数字提示：
✓ 关闭通知
✓ 隐藏 APP
✓ 减少屏幕时间

例子：
• 电视遥控器藏起来
• 关闭手机推送
• 取消邮件订阅
```

### 5. 环境设计实践

**工作空间设计：**
```
优化：
✓ 桌面整洁
✓ 重要任务物品显眼
✓ 减少干扰物
✓ 专用工作区域

避免：
× 手机在桌面
× 社交媒体标签页打开
× 杂物堆积
```

**厨房/饮食设计：**
```
健康饮食：
✓ 水果放在台面
✓ 健康零食显眼
✓ 不健康食物藏起来
✓ 大碗换小碗

喝水：
✓ 水瓶装满放各处
✓ 桌上放水杯
✓ 买喜欢的水瓶
```

**卧室/睡眠设计：**
```
优化睡眠：
✓ 完全黑暗
✓ 凉爽温度
✓ 移除电子设备
✓ 床只用于睡眠

避免：
× 手机在床边
× 电视在卧室
× 工作在床上
```

**运动空间设计：**
```
优化：
✓ 运动装备显眼
✓ 前一天晚上准备好
✓ 减少准备步骤
✓ 专用运动区域

避免：
× 装备藏起来
× 准备步骤多
× 空间杂乱
```

### 6. 环境设计检查清单

**提示可见性：**
```
好习惯提示：
□ 是否看得见？
□ 是否显眼？
□ 是否具体？
□ 是否可执行？

坏习惯提示：
□ 是否移除？
□ 是否隐藏？
□ 是否减少暴露？
```

**执行阻力：**
```
好习惯：
□ 步骤是否最少？
□ 工具是否就位？
□ 阻力是否最小？

坏习惯：
□ 是否增加步骤？
□ 是否添加障碍？
□ 阻力是否足够？
```

**空间用途：**
```
□ 每个空间是否有明确用途？
□ 是否避免混合情境？
□ 空间 - 行为连接是否清晰？
```

## 📚 核心理念

**关键洞察：**
1. 原动力被高估，环境往往更重要
2. 一个空间，一种用途
3. 视觉提示触发行为
4. 减少阻力让好习惯自动发生
5. 增加阻力让坏习惯难以执行

**健康公式：**
```
好习惯 = 显眼提示 + 最小阻力 + 专用空间
坏习惯戒除 = 移除提示 + 增加阻力 + 避免情境
```
