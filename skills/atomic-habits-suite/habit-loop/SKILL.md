# 习惯循环四步法 🔄

**基于《原子习惯》**

---

## 📋 功能描述

帮助用户理解习惯形成的四步模型，设计好习惯循环，戒除坏习惯。

**适用场景：**
- 习惯养成
- 习惯戒除
- 行为设计
- 自我观察

**边界条件：**
- 不替代专业治疗
- 需配合具体行动
- 个体差异需调整

---

## 🎯 核心功能

### 1. 习惯循环四步模型

**步骤 1：提示（Cue）**
```
定义：触发习惯的线索
作用：让大脑进入自动模式
类型：
• 时间提示（早上 7 点）
• 地点提示（厨房）
• 情绪提示（压力大）
• 人物提示（和朋友一起）
• 前置动作提示（喝完咖啡后）

关键：
✓ 提示必须显而易见
✓ 提示必须具体
✓ 提示必须可执行
```

**步骤 2：渴求（Craving）**
```
定义：对奖励的期待和欲望
作用：提供行动动力
来源：
• 对愉悦的期待
• 对痛苦的逃避
• 对身份的认同
• 对归属感的需求

关键：
✓ 渴求必须足够强
✓ 与身份认同连接
✓ 与内在需求对齐
```

**步骤 3：反应（Response）**
```
定义：实际执行的习惯行为
作用：产生结果
影响因素：
• 行为难度
• 环境阻力
• 能力水平
• 能量状态

关键：
✓ 行为必须简便易行
✓ 降低执行阻力
✓ 从两分钟开始
```

**步骤 4：奖励（Reward）**
```
定义：行为带来的满足感
作用：强化习惯循环
类型：
• 即时奖励（愉悦感）
• 延迟奖励（健康、财富）
• 内在奖励（成就感）
• 外在奖励（认可、金钱）

关键：
✓ 奖励必须令人愉悦
✓ 即时奖励最有效
✓ 与身份认同连接
```

### 2. 习惯循环反馈图

**完整循环：**
```
提示 → 渴求 → 反应 → 奖励
 ↑                        ↓
 └────── 强化循环 ────────┘

特点：
• 无止境的循环
• 只要活着就运行
• 不断扫描环境
• 预测下一步
• 从结果中学习
```

**问题阶段 vs 解决阶段：**
```
问题阶段：
• 提示（意识到需要改变）
• 渴求（想要改变）

解决阶段：
• 反应（采取行动）
• 奖励（实现改变）
```

### 3. 培养好习惯四定律

**第一定律：让它显而易见（提示）**
```
策略：
✓ 执行意图
  "我将于［时间］在［地点］做［行为］"
✓ 习惯叠加
  "继［当前习惯］之后，我将［新习惯］"
✓ 环境设计
  让提示随处可见

例子：
• 我将于早上 7 点在厨房冥想 5 分钟
• 喝完咖啡后，我将写一页
• 运动鞋放在门口显眼处
```

**第二定律：让它有吸引力（渴求）**
```
策略：
✓ 诱惑捆绑
  把需要的和想要的绑定
✓ 社交强化
  加入期望行为正常的群体
✓ 身份认同
  与理想身份连接

例子：
• 听完播客（需要）后可以读新闻（想要）
• 加入跑步俱乐部
• "我是健康的人"
```

**第三定律：让它简便易行（反应）**
```
策略：
✓ 两分钟规则
  缩减为两分钟版本
✓ 环境优化
  减少执行阻力
✓ 自动化
  使用技术锁定好习惯

例子：
• "跑 3 英里"→"系好跑鞋鞋带"
• 健身包前一晚准备好
• 自动转账储蓄
```

**第四定律：让它令人愉悦（奖励）**
```
策略：
✓ 即时奖励
  行为后立即奖励
✓ 习惯追踪
  可视化进步
✓ 绝不连续错过两次
  快速恢复

例子：
• 完成后打勾✓
• 习惯追踪日历
• 错过一天后立刻恢复
```

### 4. 戒除坏习惯四定律（反转）

**第一定律反转：让它无从显现（提示）**
```
策略：
✓ 移除提示
  让触发物看不见
✓ 环境改变
  改变触发情境
✓ 减少暴露
  避免触发环境

例子：
• 手机放另一个房间
• 不健康零食藏起来
• 避开触发路径
```

**第二定律反转：让它缺乏吸引力（渴求）**
```
策略：
✓ 重构认知
  改变对行为的看法
✓ 强调负面
  关注坏习惯的代价
✓ 身份分离
  "我不是那种人"

例子：
• "锻炼是增强体能"而非"消耗能量"
• "吸烟浪费钱且损害健康"
• "我不是吸烟者"
```

**第三定律反转：让它难以施行（反应）**
```
策略：
✓ 增加阻力
  让行为更难执行
✓ 增加步骤
  添加执行障碍
✓ 移除便利
  让工具不易获得

例子：
• 卸载社交媒体 APP
• 把游戏机收起来
• 信用卡放难拿的地方
```

**第四定律反转：让它令人厌烦（奖励）**
```
策略：
✓ 问责伙伴
  让别人监督
✓ 习惯契约
  设定惩罚条款
✓ 公开承诺
  让别人知道你的目标

例子：
• 找问责伙伴
• 失败捐款给不喜欢的组织
• 公开宣布目标
```

### 5. 习惯设计实践

**设计好习惯：**
```
步骤 1：明确提示
• 时间：具体时间点
• 地点：具体位置
• 前置动作：现有习惯

步骤 2：增强渴求
• 与身份连接
• 诱惑捆绑
• 社交强化

步骤 3：简化反应
• 两分钟版本
• 减少阻力
• 准备环境

步骤 4：设计奖励
• 即时满足
• 追踪进步
• 庆祝小赢
```

**戒除坏习惯：**
```
步骤 1：识别提示
• 什么触发这个习惯？
• 时间、地点、情绪？

步骤 2：降低渴求
• 重构认知
• 强调负面后果

步骤 3：增加阻力
• 添加障碍
• 移除便利

步骤 4：移除奖励
• 问责机制
• 惩罚条款
```

## 📚 核心理念

**关键洞察：**
1. 所有习惯形成都经历四个阶段
2. 四定律培养好习惯
3. 四定律反转戒除坏习惯
4. 提示必须显而易见
5. 奖励必须令人愉悦

**健康公式：**
```
好习惯 = 显而易见 + 有吸引力 + 简便易行 + 令人愉悦
坏习惯戒除 = 无从显现 + 缺乏吸引力 + 难以施行 + 令人厌烦
```
