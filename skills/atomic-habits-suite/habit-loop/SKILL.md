# 习惯循环四步法 🔄

**基于《原子习惯》**

---

## 📋 功能描述

帮助用户理解习惯形成的四步模型，设计好习惯循环，戒除坏习惯。

**适用场景：**
- 习惯养成（"想每天冥想"）
- 习惯戒除（"戒不掉刷手机"）
- 行为设计
- 自我观察

**边界条件：**
- 不替代专业治疗
- 需配合具体行动
- 个体差异需调整

---

## 🎯 核心功能

### 1. 习惯循环四步模型

```
提示（Cue）→ 渴求（Craving）→ 反应（Response）→ 奖励（Reward）
 ↑                                                      ↓
 └────────────────── 强化循环 ──────────────────────────┘
```

**四步说明：**

| 步骤 | 作用 | 关键问题 |
|------|------|---------|
| **提示** | 触发习惯的线索 | 什么触发了这个习惯？ |
| **渴求** | 对奖励的期待 | 你想要得到什么？ |
| **反应** | 实际执行的行为 | 你实际做了什么？ |
| **奖励** | 满足感，强化循环 | 你得到了什么？ |

### 2. 培养好习惯四定律

| 定律 | 目标 | 核心策略 |
|------|------|---------|
| **第 1 定律** | 让它显而易见 | 执行意图、习惯叠加、环境设计 |
| **第 2 定律** | 让它有吸引力 | 诱惑捆绑、社交强化、身份认同 |
| **第 3 定律** | 让它简便易行 | 两分钟规则、环境优化、自动化 |
| **第 4 定律** | 让它令人愉悦 | 即时奖励、习惯追踪、绝不连续错过两次 |

### 3. 戒除坏习惯四定律（反转）

| 定律 | 目标 | 核心策略 |
|------|------|---------|
| **第 1 定律** | 让它无从显现 | 移除提示、改变环境 |
| **第 2 定律** | 让它缺乏吸引力 | 重构认知、强调负面 |
| **第 3 定律** | 让它难以施行 | 增加阻力、添加障碍 |
| **第 4 定律** | 让它令人厌烦 | 问责伙伴、习惯契约 |

## 🧪 使用示例

### 示例 1：培养冥想习惯

**输入：**
```
@ant 我想养成每天冥想的习惯

- 习惯类型：培养好习惯
- 习惯描述：每天冥想 10 分钟
- 目前循环：无固定时间、经常忘记
- 动力水平：中等
- 环境控制：中等
```

**输出：**
```
🔄 习惯循环设计

【执行意图】
我将于早上 7 点在卧室冥想 5 分钟

【习惯叠加】
喝完咖啡后，我将冥想

【两分钟版本】
坐下深呼吸 1 分钟

【环境设计】
冥想垫放在显眼处

【追踪方法】
日历打勾✓
```

### 示例 2：戒除刷手机习惯

**输入：**
```
@ant 我晚上控制不住刷手机

- 习惯类型：戒除坏习惯
- 习惯描述：睡前刷手机 1-2 小时
- 目前循环：躺床上→拿手机→刷到睡着
- 动力水平：高
- 环境控制：低
```

**输出：**
```
🔄 坏习惯戒除计划

【移除提示】
手机放客厅充电，卧室放闹钟

【增加阻力】
卸载社交媒体 APP，需要时再下载

【重构认知】
"刷手机让我第二天疲惫"而非"放松"

【问责机制】
告诉伴侣目标，失败罚款 100 元
```

## ⚠️ 常见错误

**错误 1：目标过大**
```
问题：
• "每天冥想 30 分钟"（太难坚持）
• "每天跑 5 公里"（超出能力）

解决：
✓ 从两分钟版本开始
✓ "坐下深呼吸 1 分钟"
✓ "穿上跑鞋出门"
```

**错误 2：依赖意志力**
```
问题：
• 认为"坚持就能成功"
• 忽视环境设计

解决：
✓ 优化环境而非依赖意志力
✓ 让好习惯显而易见，坏习惯无从显现
```

**错误 3：没有即时奖励**
```
问题：
• 只有延迟奖励（如健康）
• 大脑感受不到愉悦

解决：
✓ 设计即时奖励（完成后打勾✓）
✓ 庆祝小赢
✓ 习惯追踪可视化
```

## 📚 核心理念

**关键洞察：**
1. 习惯循环永不停止，只要活着就运行
2. 改变习惯的关键是杠杆点（提示和奖励）
3. 环境设计 > 意志力
4. 两分钟规则：任何习惯都能缩减为两分钟
5. 绝不连续错过两次：快速恢复比完美更重要

**健康公式：**
```
好习惯 = 显而易见 + 有吸引力 + 简便易行 + 令人愉悦
坏习惯戒除 = 无从显现 + 缺乏吸引力 + 难以施行 + 令人厌烦
```
