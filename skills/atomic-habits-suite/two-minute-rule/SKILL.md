# 两分钟规则 ⏱️

**基于《原子习惯》**

---

## 📋 功能描述

指导用户应用两分钟规则克服拖延，将任何习惯缩减为两分钟版本，轻松启动并坚持。

**适用场景：**
- 拖延症
- 习惯启动困难
- 完美主义
- 动力不足

**边界条件：**
- 不替代时间管理
- 需配合长期规划
- 个体差异需调整

---

## 🎯 核心功能

### 1. 两分钟规则原理

**核心定义：**
```
两分钟规则：
"新习惯开始时应该不超过两分钟"

目的：
• 克服启动阻力
• 建立行为模式
• 逐步扩展习惯

理念：
✓ 先开始，再完善
✓ 频率重于时长
✓ 小步前进
```

**为什么有效：**
```
心理学原理：
• 启动阻力最大
• 一旦开始就容易继续
• 小赢建立信心
• 降低完美主义

行为科学：
• 减少决策疲劳
• 增加执行可能性
• 建立身份证据
• 形成自动化
```

### 2. 习惯缩减技术

**缩减方法：**
```
原习惯 → 两分钟版本

"每晚睡前阅读" → "读一页"
"做 30 分钟瑜伽" → "拿出瑜伽垫"
"复习功课" → "打开笔记"
"整理衣物" → "叠一双袜子"
"跑 3 英里" → "系好跑鞋鞋带"
"写 3000 字" → "写一页"
"冥想 30 分钟" → "坐下深呼吸 1 次"
```

**缩减原则：**
```
原则 1：只关注开始
• 不关注完成
• 不关注质量
• 只关注启动

原则 2：保持简单
• 最少步骤
• 最少时间
• 最少阻力

原则 3：允许扩展
• 两分钟后可以继续
• 但不要求继续
• 自然延伸
```

### 3. 克服拖延策略

**拖延根源：**
```
问题：
× 任务太大
× 完美主义
× 害怕失败
× 缺乏动力

解决：
✓ 缩减为两分钟
✓ 关注启动而非完成
✓ 允许不完美
✓ 建立小赢
```

**启动技巧：**
```
倒数启动：
• 5-4-3-2-1，开始行动
• 不给大脑犹豫时间

两分钟承诺：
• "我只做两分钟"
• 降低心理阻力
• 通常会自动继续

环境触发：
• 设置明显提示
• 减少启动步骤
• 让开始变得容易
```

### 4. 习惯扩展路径

**扩展阶段：**
```
阶段 1：启动习惯（1-2 周）
• 只做两分钟版本
• 建立启动模式
• 不计较结果

阶段 2：自然扩展（3-4 周）
• 两分钟后自然继续
• 不强制但允许
• 观察自然时长

阶段 3：优化习惯（5-8 周）
• 基于自然时长设定目标
• 逐步增加难度
• 保持可持续性

阶段 4：精通习惯（9-12 周）
• 习惯已自动化
• 注意力转向精进
• 以新习惯为跳板
```

**扩展示例：**
```
阅读习惯：
周 1-2：读一页
周 3-4：自然阅读（可能 5-10 页）
周 5-8：设定 10 页目标
周 9-12：设定 30 分钟目标

运动习惯：
周 1-2：系好跑鞋鞋带
周 3-4：出门走一圈
周 5-8：跑 1 英里
周 9-12：跑 3 英里
```

### 5. 两分钟规则应用

**健康习惯：**
```
运动：
• "跑 3 英里"→"系好跑鞋鞋带"
• "健身 1 小时"→"换上运动服"
• "做 50 个俯卧撑"→"做 1 个俯卧撑"

饮食：
• "健康饮食"→"吃一份蔬菜"
• "戒糖"→"今天不喝含糖饮料"
• "多喝水"→"喝一杯水"

睡眠：
• "早睡"→"晚上 10 点放下手机"
• "睡 8 小时"→"晚上 10 点上床"
```

**工作习惯：**
```
写作：
• "写 3000 字"→"写一页"
• "写报告"→"打开文档"
• "回复邮件"→"打开邮箱"

学习：
• "学习 2 小时"→"打开书"
• "复习功课"→"打开笔记"
• "背单词"→"打开单词 APP"

整理：
• "整理房间"→"整理一个抽屉"
• "洗衣服"→"把衣服放进洗衣机"
• "洗碗"→"洗一个碗"
```

**个人成长：**
```
冥想：
• "冥想 30 分钟"→"坐下深呼吸 1 次"
• "正念练习"→"觉察一次呼吸"

阅读：
• "读一本书"→"读一页"
• "学习新技能"→"看一个教程视频"

社交：
• "联系朋友"→"发一条信息"
• "建立人脉"→"点赞一条动态"
```

### 6. 常见问题解答

**问题 1：两分钟太短，有用吗？**
```
回答：
✓ 重点是建立启动模式
✓ 频率比时长重要
✓ 小赢积累改变身份
✓ 通常会自然扩展

关键：
• 不要小看两分钟
• 每次都是证据
• 积累改变信念
```

**问题 2：两分钟后必须停止吗？**
```
回答：
✓ 不要求停止
✓ 可以继续如果自然
✓ 但不强制继续
✓ 重点是启动

关键：
• 允许但不要求
• 听身体的
• 保持可持续性
```

**问题 3：什么时候扩展？**
```
回答：
✓ 当两分钟变得容易
✓ 当自然想要继续
✓ 当身份已建立
✓ 通常 2-4 周后

关键：
• 不急于扩展
• 让习惯自然生长
• 保持耐心
```

## ⚠️ 常见错误

**错误 1：轻视两分钟的力量**
```
问题：
• "两分钟太短，没用的"
• 想要直接做完整版
• 不屑于从小事开始

解决：
✓ 重点是建立启动模式
✓ 频率比时长重要
✓ 小赢积累改变身份
```

**错误 2：两分钟后强迫继续**
```
问题：
• 强制自己做满 30 分钟
• 没完成目标就自责
• 违背"允许但不要求"原则

解决：
✓ 两分钟后可以停止
✓ 自然继续才可持续
✓ 听身体的信号
```

**错误 3：急于扩展**
```
问题：
• 第一天就想跑 3 英里
• 一周后就想写 3000 字
• 不耐烦导致放弃

解决：
✓ 等待 2-4 周自然扩展
✓ 让习惯自己生长
✓ 保持耐心
```

**错误 4：忘记身份连接**
```
问题：
• 只做动作不想身份
• "我在跑 2 分钟"而非"我是跑者"
• 缺乏内在动力

解决：
✓ 每次行动都是投票
✓ 连接理想身份
✓ 积累身份证据
```

## 📚 核心理念

**关键洞察：**
1. 新习惯应该不超过两分钟
2. 启动阻力最大
3. 频率比时长重要
4. 小赢积累改变身份
5. 习惯会自然扩展

**健康公式：**
