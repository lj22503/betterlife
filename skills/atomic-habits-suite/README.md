# 原子习惯套件 ⚛️

**基于《掌控习惯》（原子习惯）**

---

## 📖 简介

原子习惯套件包含 5 个技能，帮助用户基于身份认同培养习惯，通过四步模型、环境设计、两分钟规则和习惯追踪，实现持久行为改变。

**核心理念：**
- 🎭 身份认同比结果更重要
- 🔄 习惯循环四步法（提示/渴求/反应/奖励）
- 🏠 环境设计比意志力更可靠
- ⏱️ 两分钟规则克服拖延
- 📊 习惯追踪 + 绝不错过两次

---

## 🎯 技能列表

| 技能 | 功能 | 适用场景 |
|------|------|----------|
| [identity-habits](identity-habits/) | 身份认同习惯 | 内在动力不足、长期改变 |
| [habit-loop](habit-loop/) | 习惯循环四步法 | 习惯设计、行为改变 |
| [environment-design](environment-design/) | 环境设计 | 意志力不足、环境干扰 |
| [two-minute-rule](two-minute-rule/) | 两分钟规则 | 拖延症、启动困难 |
| [habit-tracking](habit-tracking/) | 习惯追踪 | 坚持困难、缺乏反馈 |

## 📊 使用流程

```
1. identity-habits → 确定想成为的人
2. habit-loop → 设计习惯循环
3. environment-design → 优化环境
4. two-minute-rule → 克服启动阻力
5. habit-tracking → 追踪进步、保持动力
```

## 🧪 使用示例

### 示例 1：身份认同

```
@ant 我想养成写作习惯，但总是坚持不了

信息：
- 期望身份：作家
- 目前行为：偶尔写作、经常拖延
```

### 示例 2：习惯设计

```
@ant 我想养成每天冥想的习惯

信息：
- 习惯描述：每天冥想 10 分钟
- 动力水平：中等
```

### 示例 3：环境设计

```
@ant 我想通过环境设计培养好习惯

信息：
- 目标习惯：多喝水、少吃零食
- 意志力水平：低
```

### 示例 4：两分钟规则

```
@ant 我总是拖延写作，如何开始？

信息：
- 目标习惯：每天写作 3000 字
- 拖延程度：高
```

### 示例 5：习惯追踪

```
@ant 我想追踪我的习惯，但总是坚持不了

信息：
- 目前习惯：运动、阅读、写作
- 坚持程度：低
```

## 🔑 关键概念

### 行为改变三层次
- 结果改变（减肥、出版）
- 过程改变（习惯、体系）
- 身份改变（信念、自我形象）

### 习惯循环四步
- 提示（触发线索）
- 渴求（欲望动力）
- 反应（实际行为）
- 奖励（满足感）

### 培养好习惯四定律
1. 让它显而易见
2. 让它有吸引力
3. 让它简便易行
4. 让它令人愉悦

### 戒除坏习惯四定律（反转）
1. 让它无从显现
2. 让它缺乏吸引力
3. 让它难以施行
4. 让它令人厌烦

### 核心策略
- 执行意图：我将于［时间］在［地点］做［行为］
- 习惯叠加：继［当前习惯］之后，我将［新习惯］
- 两分钟规则：缩减为两分钟版本
- 绝不错过两次：快速恢复
