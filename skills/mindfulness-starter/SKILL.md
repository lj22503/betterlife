# 正念入门引导器 🧘

**基于《5-8 周正念禅修练习》**

---

## 📋 功能描述

帮助用户开始正念禅修练习，了解基本姿势、呼吸觉察和思绪处理。

**适用场景：**
- 想学习正念禅修
- 初学者入门
- 减压、放松
- 提升专注力

**边界条件：**
- 不替代专业心理治疗
- 严重心理问题需就医
- 循序渐进练习

---

## 🎯 核心功能

### 1. 准备姿势

**坐姿：**
```
• 坐在椅子上，后背挺直但不僵硬
• 双脚平放地板，双腿不交叉
• 双手放在大腿上
• 可以闭眼或视线放低
```

**卧姿：**
```
• 仰卧在地板或床上
• 双腿不交叉，双脚自然分开
• 双臂沿身体两侧摆放
• 手掌向上
```

### 2. 呼吸觉察

**步骤：**
1. 将注意力放在腹部
2. 感受气息进出时腹部的起伏
3. 不要控制呼吸，让呼吸自然进行
4. 注意力游离时，温和地带回呼吸

### 3. 处理思绪游离

**正确态度：**
```
✓ 思绪游离不是错误，是大脑的正常活动
✓ 发现游离时，恭贺自己足够警醒
✓ 温和地将注意力带回呼吸
✓ 培养耐心，对自己宽容
```

---

## ⚠️ 常见错误

**错误 1：试图控制呼吸**
```
问题：
• "我应该这样呼吸"
• 刻意调节呼吸节奏

解决：
✓ 完全没有必要控制呼吸
✓ 让气息自然呼吸
✓ 只是观察，不改变
```

**错误 2：对思绪游离感到沮丧**
```
问题：
• "我怎么又走神了"
• 责备自己无法专注

解决：
✓ 思绪游离是正常的
✓ 每次发现都是警醒的表现
✓ 温和地带回，不责备
```
