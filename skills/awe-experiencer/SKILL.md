# 敬佩情绪体验器 🌟

**基于《积极情绪的力量》（芭芭拉·弗雷德里克森）**

---

## 📋 功能描述

帮助用户体验敬佩情绪，在大规模邂逅善举时被伟大征服，感觉渺小和谦卑。

**适用场景：**
- 想感到敬佩
- 被伟大征服
- 体验庄严
- 提升谦卑感

**边界条件：**
- 敬佩来自大规模邂逅善举
- 被伟大彻底征服
- 感觉渺小和谦卑

---

## 🎯 核心功能

### 1. 敬佩的特征

**与激励的关系：**
```
• 它与激励的关系密切
• 在你大规模地邂逅善举时产生
• 你被伟大彻底征服了
• 相比之下，你感觉渺小和谦卑
• 敬佩令你停在自己的轨道上
• 你一时间动弹不得
```

### 2. 敬佩的感受

**身体感受：**
```
• 感到强烈的好奇或惊讶
• 真正地对周围环境感到惊叹
• 极大程度地对伟大或美丽感到不知所措
• 为庄严惊呆、止步
• 感觉到自己属于一个远远大于你自己的事物的一部分
```

### 3. 培养方法

**微习惯：**
```
• 参观宏伟的建筑
• 欣赏壮丽的自然景观
• 见证伟大的善举
• 体验庄严的时刻
• 感受自己属于更大的事物
```

---

## ⚠️ 常见错误

**错误 1：追求日常敬佩**
```
问题：
• "我想每天都感到敬佩"
• 追求日常敬佩

解决：
✓ 敬佩来自大规模邂逅善举
✓ 不是日常情绪
✓ 珍惜敬佩的时刻
```
