## 🎯 核心功能

**步骤：**
```
1. 评估当前关系状态

3. 制定维护策略
4. 持续经营关系
```

---

## ⚠️ 常见错误

**错误 1：只认识不维护**
```
问题：
• 认识更多人，但没有深度连接
• 只交换公开信息，不交换半公开信息

解决：
✓ 人脉不是"认识更多人"，而是"与关键的人建立深度连接"
✓ 交换半公开信息才是人与人结交的第一步
✓ 人脉需要精心设计、用心维护
```

**错误 2：信息不对称**
```
问题：
• 你知道对方很多信息，对方不知道你的
• 信息共享不对等

解决：
✓ 彼此情况互通程度是决定人脉关系程度的重要指标
✓ 主动分享，增进双向沟通
✓ 平衡关系投入
```
