# 断糖实践指南 🚫🍬

**基于《疗愈的饮食与断食》第 12、16-18 章**

---

## 📋 功能描述

指导用户进行 30 天断糖实验，戒除精制糖和添加糖，重建代谢健康。

**适用场景：**
- 想戒除糖瘾
- 改善代谢健康
- 减重需求
- 皮肤问题改善

**边界条件：**
- 糖尿病患者需医师指导
- 孕妇/哺乳期不建议严格断糖
- 初期可能有"戒断反应"

---

## 🎯 核心功能

### 1. 糖的 50 个名字识别

**隐藏糖成分清单：**
```
× 糖类：蔗糖、红糖、冰糖、蜂蜜、枫糖浆
× 糖浆：高果糖玉米糖浆、麦芽糖浆、糙米糖浆
× 结尾带"糖"：葡萄糖、果糖、乳糖、麦芽糖
× 结尾带"浆"：玉米浆、麦芽浆
× 其他：浓缩果汁、蒸发甘蔗汁
```

### 2. 30 天断糖计划

**第 1 周：清理期**
- 清理厨房：移除含糖食品
- 阅读标签：识别所有添加糖
- 替代方案：用完整水果满足甜味需求

**第 2 周：适应期**
- 身体开始适应低糖状态
- 可能有头痛、疲倦、渴望（酮流感）
- 多喝水、补充电解质

**第 3-4 周：稳定期**
- 味觉恢复，能品尝天然甜味
- 精力稳定，不再大起大落
- 皮肤改善，发炎减少

### 3. 外食/便利商店策略

**便利商店选择：**
- 生菜色拉（不加酱）
- 水煮鸡肉/茶叶蛋
- 地瓜（适量）
- 自带橄榄油/牛油果油

**外食原则：**
- 叶菜类为主
- 适量蛋白质
- 不用现成酱料
- 略过甜点和含糖饮料

### 4. 好脂肪补充

**断糖期间必需补充：**
```
✓ 果实油：橄榄油、椰子油、牛油果油
✓ 坚果：核桃、胡桃、榛果
✓ 动物脂肪：草饲牛、放养鸡
✓ 其他：印度酥油 (ghee)、牛油果
```

**避免的油：**
```
× 种籽油：玉米油、大豆油、棉籽油、葵花籽油、芥花油
× 原因：omega-6 过高，促发炎
```

## ⚠️ 注意事项

**常见戒断反应（1-2 周）：**
- 头痛、疲倦
- 情绪波动
- 强烈渴望甜食
- 失眠

**应对策略：**
- 多喝水（2-3L/天）
- 补充电解质（海盐）
- 吃好脂肪增加饱足感
- 早点睡觉

**何时停止：**
- 严重不适
- 低血糖症状
- 怀孕/哺乳期
- 医师建议停止

---

## 📚 核心理念

**关键洞察：**
1. 糖有 50 个名字，隐藏在加工食品中
2. 断糖需要好脂肪支持
3. 戒断反应 1-2 周会消失
4. 味觉会在 3-4 周恢复
5. 断糖是代谢修复的第一步

**成功关键：**
- 提前规划
- 清理环境
- 补充好脂肪
- 找到支持系统
