# 肠道修复方案 🦠

**基于《疗愈的饮食与断食》第 25-27 章**

---

## 📋 功能描述

帮助用户识别肠漏症状，提供肠道修复的饮食和生活方式方案。

**适用场景：**
- 慢性发炎
- 过敏/自体免疫问题
- 消化问题（胀气、便秘、腹泻）
- 皮肤问题（痘痘、湿疹）
- 脑雾、情绪问题

**边界条件：**
- 不替代医疗诊断
- 严重症状需就医
- 修复需要时间（3-6 个月）

---

## 🎯 核心功能

### 1. 肠漏自测

**肠漏信号：**
```
✓ 上完厕所要用很多卫生纸才能擦干净（肠道黏液）
✓ 慢性胀气、消化不良
✓ 食物过敏/敏感增多
✓ 皮肤问题（痘痘、湿疹、玫瑰痤疮）
✓ 慢性疲劳、脑雾
✓ 关节疼痛
✓ 自体免疫疾病
✓ 情绪问题（焦虑、抑郁）
```

**风险评估：**
- 0-2 个信号：肠道健康
- 3-5 个信号：轻度肠漏
- 6+ 个信号：严重肠漏

### 2. 肠道修复 4R 方案

**Remove（移除）：**
- 精制糖、精制淀粉
- 种籽油、加工食品
- 个人过敏原（麸质、豆类、茄科等）
- 酒精、人工甜味剂

**Replace（替代）：**
- 消化酶（苹果醋、柠檬汁）
- 胃酸支持（盐、锌）
- 胆汁支持（好脂肪）

**Reinoculate（重新接种）：**
- 益生菌：发酵食品（酸菜、泡菜、纳豆）
- 益生元：膳食纤维（蔬菜、菇菌类）
- 多糖体：菇菌类、魔芋

**Repair（修复）：**
- 骨汤（胶原蛋白、甘氨酸）
- L-谷氨酰胺（肉类、蔬菜）
- 锌、维他命 A、D
- 好脂肪（椰子油、橄榄油）

### 3. 肠道友好饮食

**推荐食物：**
```
✓ 大量蔬菜（生吃最佳）
✓ 发酵食品（酸菜、泡菜、纳豆）
✓ 骨汤/肉汤
✓ 好脂肪（椰子油、橄榄油、牛油果）
✓ 菇菌类（多糖体）
✓ 十字花科蔬菜（花椰菜、绿花椰菜）
✓ 绿色叶菜
```

**避免食物：**
```
× 精制糖、加工食品
× 种籽油
× 个人过敏原
× 过量酒精
× 人工甜味剂、乳化剂
```

### 4. 断食辅助修复

**间歇性断食：**
- 16-8 断食：每天 16 小时不进食
- 18-6 断食：进阶版
- 给肠道休息时间

**效果：**
- 减少肠道负担
- 促进自噬作用
- 改善肠道菌群
- 降低肠道通透性

## 📚 核心理念

**关键洞察：**
1. 肠 - 脑轴线：肠道健康影响大脑
2. 肠漏是慢性发炎根源
3. 肠道通透性可修复
4. 发酵食品是关键
5. 断食给肠道休息时间

**修复时间线：**
- 2-4 周：症状开始改善
- 3 个月：肠道屏障功能恢复
- 6 个月：免疫系统平衡
