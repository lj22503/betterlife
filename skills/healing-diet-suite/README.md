# 疗愈饮食套件 🥗

**基于杨定一《疗愈的饮食与断食：新时代的个人营养学》**

---

## 📖 简介

疗愈饮食套件包含 5 个技能，帮助用户从现代饮食错误中恢复，重建代谢健康。

**核心理念：**
- 💡 低脂饮食推广是营养学最大错误
- 🍬 精制糖和种籽油是健康杀手
- 🥑 饱和脂肪对健康有保护力
- 🦠 肠道健康是全身健康基础
- 🌾 凝集素可能是过敏根源

## 📊 使用流程

```
1. insulin-resistance → 评估代谢风险
2. sugar-detox → 开始断糖（第 1-4 周）
3. healthy-fat-guide → 替换用油（同步进行）
4. gut-healing → 修复肠道（第 5-12 周）
5. lectin-avoidance → 排查过敏（如仍有症状）
```

## 🔑 关键概念

### 胰岛素阻抗
- 现代慢性病根源
- 由长期营养过剩引起
- 改善方法：断糖、好脂肪、断食

### 果实油 vs 种籽油
- 果实油：橄榄油、椰子油、牛油果油（✅）
- 种籽油：玉米油、大豆油、葵花籽油（❌）
- omega-6/omega-3 比例关键

### 肠漏
- 肠道通透性增加
- 导致慢性发炎和过敏
- 修复方法：4R 方案

### 凝集素
- 植物天然防御物质
- 可能导致肠漏和过敏
- 高凝集素食物：谷类、豆类、茄科
