# 健康脂肪选择指南 🥑

**基于《疗愈的饮食与断食》第 19-20 章**

---

## 📋 功能描述

帮助用户识别和选择健康脂肪，理解果实油与种籽油的本质差异。

**适用场景：**
- 购买食用油
- 改善饮食脂肪质量
- 降低发炎体质
- 调整 omega-6/omega-3 比例

**边界条件：**
- 不替代医疗建议
- 个体差异需自我观察
- 适量为原则

---

## 🎯 核心功能

### 1. 脂肪分类系统

**✅ 推荐脂肪（果实油/草饲动物）：**
```
• 椰子油：高饱和脂肪，稳定耐高温
• 橄榄油：单不饱和脂肪，抗氧化
• 牛油果油：单不饱和脂肪，高烟点
• 印度酥油 (ghee)：去除酪蛋白，耐高温
• 草饲牛脂肪：omega-3 比例较高
• 放养鸡脂肪：优于谷饲鸡
• 小型野生鱼：沙丁鱼、鲭鱼（omega-3）
```

**❌ 避免脂肪（种籽油）：**
```
• 玉米油：omega-6 过高
• 大豆油：omega-6 过高，易氧化
• 棉籽油：工业提炼，残留溶剂
• 葵花籽油：omega-6 过高
• 芥花油：高度加工
• 花生油：omega-6 过高，易有黄曲霉素
```

### 2. omega-6/omega-3 比例评估

**理想比例：** 1:1 ~ 3:1

**现代西式饮食：** 15:1（严重失衡）

**改善策略：**
```
增加 omega-3：
✓ 亚麻籽油、藻油、鱼油
✓ 草饲牛肉
✓ 小型野生鱼（沙丁鱼、鲭鱼）

减少 omega-6：
× 种籽油
× 谷饲牛肉
× 加工食品
```

### 3. 脂肪使用指南

**烹饪方式匹配：**
| 油类 | 烟点 | 适合烹饪 |
|------|------|----------|
| 椰子油 | 177°C | 中火炒、烘焙 |
| 橄榄油（特级初榨） | 160°C | 凉拌、低温 |
| 橄榄油（精炼） | 240°C | 中高温 |
| 牛油果油 | 270°C | 高温煎炸 |
| 印度酥油 | 250°C | 印度料理、高温 |
| 猪油（草饲） | 190°C | 中式炒菜 |

**每日脂肪摄入建议：**
```
• 总脂肪：热量 50-60%
• 饱和脂肪：20-25%
• 单不饱和脂肪：25-30%
• 多不饱和脂肪：5-10%
```

### 4. 脂肪质量判断

**优质脂肪特征：**
- 冷压/初榨
- 有机/草饲
- 深色玻璃瓶包装
- 无异味、无油耗味

**劣质脂肪特征：**
- 精炼/脱臭
- 透明塑料瓶
- 价格便宜
- 有油耗味

## 📚 核心理念

**关键洞察：**
1. 不是所有脂肪都是平等的
2. 果实油优于种籽油
3. omega-6/omega-3 比例关键
4. 饱和脂肪对健康有保护力
5. 草饲优于谷饲

**脂肪真相：**
- 种籽油是 20 世纪新成分（之前用于制肥皂、蜡烛）
- 工业提炼过程：高温高压 + 有机溶剂
- 氢化产生反式脂肪
- 多元不饱和脂肪易氧化
