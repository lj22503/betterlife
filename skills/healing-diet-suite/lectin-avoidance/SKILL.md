# 凝集素回避清单 🌾

**基于《疗愈的饮食与断食》第 28-31 章**

---

## 📋 功能描述

帮助用户识别高凝集素食物，提供回避和处理的实用方案。

**适用场景：**
- 慢性发炎
- 自体免疫疾病
- 不明原因过敏
- 肠道问题
- 尝试纯肉饮食/低敏饮食

**边界条件：**
- 不是所有人都对凝集素敏感
- 需个人观察和测试
- 长期回避需营养均衡

---

## 🎯 核心功能

### 1. 凝集素识别

**什么是凝集素：**
- 植物的天然防御物质
- 会结合肠道细胞
- 可能导致肠漏和发炎
- 麸质是其中一种

**高凝集素食物清单：**

**❌ 谷类（最高）：**
```
• 玉米、小麦、稻米
• 藜麦、奇亚籽、荞麦
• 所有麸质食品
```

**❌ 豆类：**
```
• 花生、黄豆、腰豆
• 鹰嘴豆、扁豆、红豆、绿豆
• 所有豆类都含凝集素
```

**❌ 瓜科和茄科：**
```
• 南瓜、栉瓜
• 番茄、茄子、彩椒、青椒、辣椒
• 马铃薯（茄科）
• 枸杞
```

**❌ 其他：**
```
• 未成熟水果（催熟的）
• 谷类/豆类喂养的动物肉
• 坚果（杏仁、腰果等）
```

### 2. 安全食物清单

**✅ 低凝集素蔬菜：**
```
• 绿色叶菜：茴香、莴苣、芽球菊苣
• 十字花科：花椰菜、绿花椰菜、白菜、卷心菜
• 其他：蘑菇、芦笋、芹菜、洋葱
• 根茎类：地瓜、芋头（去皮）
```

**✅ 安全脂肪：**
```
• 牛油果
• 橄榄油
• 椰子油
• 草饲动物脂肪
```

**✅ 安全蛋白质：**
```
• 草饲牛肉、放养鸡
• 小型野生鱼（沙丁鱼、鲭鱼）
• 蛋（耐受者）
```

### 3. 凝集素处理方法

**浸泡：**
- 豆类浸泡 12-24 小时
- 换水多次
- 减少凝集素 50%+

**压力锅煮：**
- 高温高压破坏凝集素
- 豆类必须煮透
- 比一般烹饪更有效

**去皮去籽：**
- 番茄去皮去籽
- 黄瓜去皮去籽
- 凝集素集中在皮和籽

**发酵：**
- 发酵过程分解凝集素
- 酸菜、泡菜、纳豆
- 提高消化性

**发芽：**
- 发芽降低凝集素
- 芽菜比种子安全
- 但仍需观察耐受性

### 4. 减敏饮食实验

**30 天纯肉/低敏实验：**
```
周 1-2：严格回避所有高凝集素食物
周 3-4：逐一重新引入，观察反应
```

**重新引入顺序：**
1. 低凝集素蔬菜（十字花科）
2. 根茎类（地瓜、芋头）
3. 坚果（去皮）
4. 豆类（浸泡 + 压力锅）
5. 谷类（最后，可选）

**观察指标：**
- 消化症状（胀气、便秘、腹泻）
- 皮肤反应（痘痘、湿疹）
- 关节疼痛
- 疲劳程度
- 脑雾

## 📚 核心理念

**关键洞察：**
1. 麸质只是众多凝集素之一
2. 豆类凝集素常被忽视
3. 茄科植物对某些人是过敏原
4. 处理方法可降低凝集素
5. 个人差异很大，需自我观察

**处理原则：**
- 能去皮去籽就去皮去籽
- 豆类必须浸泡 + 压力锅
- 发酵是好朋友
- 简化饮食，逐一测试
