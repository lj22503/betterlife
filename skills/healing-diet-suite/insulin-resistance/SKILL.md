# 胰岛素阻抗自测与改善 💉

**基于《疗愈的饮食与断食》第 5-6 章**

---

## 📋 功能描述

帮助用户评估胰岛素阻抗风险，提供改善代谢的 actionable 建议。

**适用场景：**
- 评估代谢健康状况
- 理解慢性病根源
- 制定饮食调整计划

**边界条件：**
- 不替代医疗诊断
- 不调整处方药物
- 建议配合医生指导

---

## 🎯 核心功能

### 1. 胰岛素阻抗风险评估

**风险信号：**
- 腰围超标（男≥90cm，女≥80cm）
- 饭后容易疲倦
- 渴望甜食/精制淀粉
- 体重难以下降
- 血压偏高
- 三酸甘油酯偏高
- HDL 胆固醇偏低
- 空腹血糖偏高（100-125 mg/dL 为糖尿病前期）

**评估逻辑：**
```
风险等级 = 符合信号数量
- 0-2 个：低风险
- 3-4 个：中风险
- 5+ 个：高风险
```

### 2. 代谢改善方案

**第一阶段（1-4 周）：断精制糖**
- 戒除含糖饮料、甜点、加工食品
- 阅读食品标签，识别 50 种糖的名字
- 用完整水果替代果汁

**第二阶段（5-8 周）：断精制淀粉**
- 白米饭→糙米/地瓜/芋头
- 白面包/面条→全谷物
- 糕饼/面皮→根茎类

**第三阶段（9 周+）：低糖高脂生机饮食**
- 大量叶菜类（生吃最佳）
- 好脂肪：橄榄油、椰子油、牛油果、坚果
- 适量蛋白质：豆类、鱼、蛋、草饲肉

### 3. 饮食比例调整

**理想比例：**
```
| 营养素 | 热量占比 | 食物来源 |
|--------|----------|----------|
| 脂肪 | 50-60% | 果实油、草饲动物脂肪 |
| 蛋白质 | 20-25% | 鱼、蛋、豆类、肉类 |
| 碳水化合物 | 20-30% | 蔬菜、少量全谷物 |
```

**饮食顺序：**
1. 脂肪（橄榄油、牛油果）
2. 蛋白质（肉、鱼、蛋）
3. 蔬菜（大量叶菜）
4. 淀粉类（最后，少量）

## 📚 核心理念

**关键洞察：**
1. 胰岛素阻抗是代谢失衡的核心
2. 低脂饮食推广是错误方向
3. 饱和脂肪对健康有保护力
4. 果糖代谢负担≈酒精
5. 恢复代谢灵活性是目标

**关键公式：**
```
代谢灵活性 = 能在糖代谢和脂肪代谢间自由切换
胰岛素阻抗 = 长期营养过剩的保护机制
```
