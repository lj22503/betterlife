# 胰岛素阻抗自测清单

## 风险信号评估

请勾选符合的项目：

### 身体指标
- [ ] 腰围超标（男≥90cm，女≥80cm）
- [ ] BMI≥24
- [ ] 血压偏高（≥130/85 mmHg）

### 症状
- [ ] 饭后容易疲倦
- [ ] 渴望甜食/精制淀粉
- [ ] 体重难以下降
- [ ] 难以减掉的腹部脂肪

### 体检指标（如有）
- [ ] 空腹血糖偏高（100-125 mg/dL）
- [ ] 三酸甘油酯偏高（≥150 mg/dL）
- [ ] HDL 胆固醇偏低（男<40，女<50 mg/dL）
- [ ] 尿酸偏高

## 行动建议

### 第一阶段（1-4 周）：断精制糖
- [ ] 戒除含糖饮料
- [ ] 戒除甜点、加工食品
- [ ] 阅读食品标签

### 第二阶段（5-8 周）：断精制淀粉
- [ ] 白米饭→糙米/地瓜
- [ ] 白面包→全谷物
- [ ] 减少面食

### 第三阶段（9 周+）：低糖高脂
- [ ] 大量蔬菜
- [ ] 好脂肪（橄榄油、椰子油）
- [ ] 适量蛋白质
