# 职业生态系统构建器 🌐

**基于《远见：如何规划职业生涯 3 大阶段》核心理念**

---

## 📋 功能描述

帮助用户建立和维护职业生态系统，找到关键同事、导师和支持者。

**适用场景：**
- 职业发展
- 人脉建设
- 职业转型
- 回归正轨

---

## 🎯 核心功能

### 1. 职业生态系统地图

**核心圈：**
- 导师（3-5 人）
- 关键同事（5-10 人）
- 专家团

**外围圈：**
- 联系人
- 行业同行
- 校友网络

### 2. 关键同事识别

**定义：** 在目前公司里对你的发展拥有决定性影响力的人

**排序：**
1. 直接上司（影响职业成功和幸福的头号人物）
2. 重要同事
3. 关键下属
4. 跨部门合作者

### 3. 导师寻找策略

**标准：**
- 真正能称为导师的人
- 在你成长的道路上提供指导
- 愿意为你投入时间

**数量：** 3-5 个

### 4. 回归正轨之法

1. **重新组织经验** - 从新角度看待过往经历
2. **重新包装技能** - 用新语言表达老技能
3. **重新连接职业生态系统** - 激活旧关系
4. **重新建立自信** - 从小胜利开始

### 5. 总部 vs 小型市场

**总部→小型市场：** 学会凭借较少资源完成任务
**小型市场→总部：** 学会在更大背景下处理复杂问题
