# 愤怒表达引导器 😠

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户学习正确表达愤怒，理解愤怒是基本情感，避免怄气和抑郁。

**适用场景：**
- 不会表达愤怒
- 生闷气
- 愤怒转向自己
- 感到抑郁

---

## 🎯 核心功能

### 1. 愤怒表达方式

**不健康的愤怒：**
```
• 怄气（用时间稀释愤怒）
• 愤怒转向自己（抑郁）
• 投射到别人身上（攻击）
```

**健康的愤怒：**
```
• 直接表达感受
• 不攻击别人
• 不转向自己
```

### 2. 愤怒与抑郁

```
愤怒能够非常有效地抵抗抑郁。
愤怒的人从不觉得内疚，因为他们总在怪罪别人。
```

---

## ⚠️ 常见错误

**错误 1：认为愤怒是坏的**
```
问题：
• "生气是不好的"
• "我不应该生气"

解决：
✓ 愤怒是基本的情感反应
✓ 需要学会正确看待和表达
✓ 不转向自己（抑郁）
```
