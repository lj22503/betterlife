# 情商大师 🎓

**基于《蛤蟆先生去看心理医生》（罗伯特·戴博德）**

---

## 📋 功能描述

帮助用户提升情商，理解并掌控内心情感世界，从悲伤中重新振作。

**适用场景：**
- 想提升情商
- 不会管理情绪
- 容易冲动
- 情绪困扰

**边界条件：**
- 情商和智商完全不是一回事
- 高情商能管理情绪
- 能从悲伤和不幸中重新振作

---

## 🎯 核心功能

### 1. 情商定义

**情商真正的意思：**
```
• 理解你内心的情感世界
• 掌控自己的情绪
• 从悲伤中重新振作
• 控制冲动
• 懂得延迟满足
```

### 2. 高情商特征

**高情商的人：**
- [ ] 有强大的自我意识
- [ ] 了解自己的情感
- [ ] 能管理情绪
- [ ] 能从悲伤中重新振作
- [ ] 能控制冲动
- [ ] 懂得延迟满足
- [ ] 避免轻率的决定和不妥的行为

### 3. 情商提升路径

**步骤 1：自我意识**
```
了解自己的情绪
```

**步骤 2：情绪管理**
```
管理而非压抑情绪
```

**步骤 3：自我激励**
```
从悲伤中重新振作
```

**步骤 4：冲动控制**
```
延迟满足，避免轻率决定
```

## ⚠️ 常见错误

**错误 1：以为情商就是会说话**
```
问题：
• "情商高就是会来事"
• 忽略内心情感世界

解决：
✓ 情商是理解内心情感世界
✓ 掌控自己的情绪
✓ 不是表面的会说话
```

**错误 2：压抑情绪**
```
问题：
• "高情商就是不发脾气"
• 压抑负面情绪

解决：
✓ 高情商是管理情绪，不是压抑
✓ 允许自己感受所有情绪
✓ 从悲伤中重新振作
```
