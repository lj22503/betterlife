# 创造模式启动器 🎨

**基于《认知驱动：做成一件对他人很有用的事》核心理念**

---

## 📋 功能描述

帮助用户启动创造模式，做成一件对他人很有用的事，体验顶级的生活——创造。

**适用场景：**
- 想创造价值但不知道如何开始
- 想成为创作者
- 想找到人生目标
- 想体验创造的快乐

---

## 🎯 核心功能

### 1. 创造者思维

**顶级生活的定义：**
- 不是奢华，而是创造
- 创造有价值的内容/产品/服务
- 对他人很有用的事

### 2. 长期主义者模式

**人生最好模式：**
- 长期乐观：看清机制、寻找意义
- 短期悲观：在舒适区边缘持续拓展
- 当下愉悦：学会转换视角、换取反馈

### 3. 成事三觉醒

**愿望觉醒：**
- 从不知道要变好到想要变好
- 从浑浑噩噩到向往美好生活

**方法觉醒：**
- 从不知道怎么变好到知道怎么变好
- 从盲目毅力支撑到科学认知驱动

**目标觉醒：**
- 寻找自己的人生目标
- 做成一件或多件对自己和他人有用的事
- 让自己成为一个很有价值的人

### 4. 苍蝇模式 vs 蜜蜂模式

**苍蝇模式（探索期）：**
- 尽情试错
- 寻找出口
- 刚开始进入新领域时使用

**蜜蜂模式（执行期）：**
- 全力奔向目标
- 找到出口后使用
- 参考要素：价值、利他、复制、跨界、刚需、触动
