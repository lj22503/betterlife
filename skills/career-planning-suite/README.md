## 📚 核心理论

### 职业生涯长度

**公式：** 62 - 目前年龄 = 剩余职业生涯年数

**示例：** 35 岁 → 62-35 = 27 年

### 三种职场燃料

| 燃料类型 | 内容 |
|----------|------|
| 可迁移技能 | 学位/证书/语言/情商 |
| 有意义的经验 | 旅行/管理/创业/志愿者 |
| 持久的关系 | 联系人/专家团/关键同事/导师 |

### 三大职业生涯阶段

| 阶段 | 时长 | 主导策略 |
|------|------|----------|
| 第一阶段 | 0-15 年 | 加添燃料，强势开局 |
| 第二阶段 | 15-30 年 | 聚焦长板，达到高点 |
| 第三阶段 | 30-45 年 | 优化长尾，持续发挥影响力 |

### 5 个关键数字

1. **职业生涯长度** = 62 - 目前年龄
2. **精通技能时间** = 10000 小时
3. **40 岁后财富占比** = 85%-90%
4. **社交货币** = 不是越多越好
5. **导师人数** = 3-5 个

### 4 大黄金问题

1. 我是否正在学习和成长？
2. 我是否正在对某些人、公司、社会拥有影响力？
3. 我体验到乐趣了吗？
4. 我是否得到了适当的奖励，并创造了经济价值？
